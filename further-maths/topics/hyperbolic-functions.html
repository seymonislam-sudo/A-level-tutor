Hyperbolic Functions

The hyperbolic cousins of $\sin$ and $\cos$ — defined via exponentials, satisfying parallel identities, and unlocking a powerful new family of integration techniques.

6.1 Definitions

The hyperbolic functions are defined using the exponential function. They are not circular functions — they parameterise the hyperbola $x^2 - y^2 = 1$ in the same way that $\cos\theta$ and $\sin\theta$ parameterise the unit circle.

$$\cosh x = \frac{e^x + e^{-x}}{2} \qquad \sinh x = \frac{e^x - e^{-x}}{2} \qquad \tanh x = \frac{\sinh x}{\cosh x} = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

Key Properties

$\cosh x$ is always $\geq 1$, even, and looks like a catenary (the shape of a hanging chain). $\cosh 0 = 1$.

$\sinh x$ is odd, passes through the origin, and grows without bound in both directions. $\sinh 0 = 0$.

$\tanh x$ is odd with horizontal asymptotes at $y = \pm 1$. $-1 < \tanh x < 1$ for all $x$.

Osborn's Rule Any trig identity can be converted to a hyperbolic identity by replacing every trig function with its hyperbolic counterpart, and replacing the sign of every product of two $\sinh$ terms (including $\tanh^2 = \sinh^2/\cosh^2$) with its negative. For example $\cos^2\theta + \sin^2\theta = 1$ becomes $\cosh^2 x - \sinh^2 x = 1$.
Worked Example 6.1a
Show from the definitions that $\cosh^2 x - \sinh^2 x = 1$.
1
$\cosh^2 x = \left(\dfrac{e^x+e^{-x}}{2}\right)^2 = \dfrac{e^{2x}+2+e^{-2x}}{4}$
2
$\sinh^2 x = \left(\dfrac{e^x-e^{-x}}{2}\right)^2 = \dfrac{e^{2x}-2+e^{-2x}}{4}$
3
$\cosh^2 x - \sinh^2 x = \dfrac{(e^{2x}+2+e^{-2x})-(e^{2x}-2+e^{-2x})}{4} = \dfrac{4}{4} = 1$ $\quad\square$

6.2 Hyperbolic Identities

These mirror the trig identities but with sign changes where products of $\sinh$ appear (Osborn's rule).

$$\cosh^2 x - \sinh^2 x = 1$$ $$1 - \tanh^2 x = \text{sech}^2\, x$$ $$\sinh(A \pm B) = \sinh A \cosh B \pm \cosh A \sinh B$$ $$\cosh(A \pm B) = \cosh A \cosh B \pm \sinh A \sinh B$$ $$\sinh 2x = 2\sinh x \cosh x$$ $$\cosh 2x = \cosh^2 x + \sinh^2 x = 2\cosh^2 x - 1 = 1 + 2\sinh^2 x$$
Contrast with Trig Notice $\cosh(A+B) = \cosh A\cosh B + \sinh A\sinh B$ — the sign is $+$, not $-$ as in $\cos(A+B)$. And $\cosh 2x = \cosh^2 x + \sinh^2 x$ — again $+$ not $-$. The identity $\cosh^2 x - \sinh^2 x = 1$ has a minus, unlike $\cos^2+\sin^2=1$.
Worked Example 6.2a
Solve $2\cosh^2 x - 5\cosh x + 2 = 0$.
1
Let $u = \cosh x$. Factorise: $(2u-1)(u-2) = 0$, so $u = \frac{1}{2}$ or $u = 2$.
2
$\cosh x = \frac{1}{2}$: impossible since $\cosh x \geq 1$ for all real $x$. Reject.
3
$\cosh x = 2$: $x = \pm\cosh^{-1}(2) = \pm\ln(2+\sqrt{3})$.
$\cosh$ is even so there are two solutions.

6.3 Inverse Hyperbolic Functions

The inverse functions $\text{arcsinh}$, $\text{arccosh}$, $\text{arctanh}$ (also written $\sinh^{-1}$, $\cosh^{-1}$, $\tanh^{-1}$) are defined as follows:

$\text{arcsinh}: \mathbb{R} \to \mathbb{R}$ — defined for all real $x$.

$\text{arccosh}: [1, \infty) \to [0, \infty)$ — only defined for $x \geq 1$, returning non-negative values.

$\text{arctanh}: (-1,1) \to \mathbb{R}$ — only defined for $|x| < 1$.

Domain Restriction for arccosh Since $\cosh$ is even, $\cosh^{-1}$ is only a function if we restrict the domain to $x \geq 0$. This gives $\cosh^{-1}(x) \geq 0$. When solving equations like $\cosh x = k$, remember there are two solutions: $x = \pm\cosh^{-1}(k)$.

6.4 Logarithmic Forms

The inverse hyperbolic functions can be expressed as natural logarithms — a beautiful result derived by solving the exponential definitions.

$$\sinh^{-1} x = \ln\!\left(x + \sqrt{x^2+1}\right) \quad \text{all } x$$ $$\cosh^{-1} x = \ln\!\left(x + \sqrt{x^2-1}\right) \quad x \geq 1$$ $$\tanh^{-1} x = \frac{1}{2}\ln\!\left(\frac{1+x}{1-x}\right) \quad |x| < 1$$
Worked Example 6.4a — Derivation
Derive $\sinh^{-1} x = \ln(x + \sqrt{x^2+1})$.
1
Let $y = \sinh^{-1} x$, so $\sinh y = x$, i.e. $\dfrac{e^y - e^{-y}}{2} = x$.
2
Multiply by $2e^y$: $e^{2y} - 1 = 2xe^y$, giving $e^{2y} - 2xe^y - 1 = 0$.
3
Quadratic in $e^y$: $e^y = \dfrac{2x \pm \sqrt{4x^2+4}}{2} = x \pm \sqrt{x^2+1}$.
4
Since $e^y > 0$ and $\sqrt{x^2+1} > |x|$, we must take the $+$ sign: $e^y = x + \sqrt{x^2+1}$.
5
$y = \ln(x + \sqrt{x^2+1})$ $\quad\square$
Worked Example 6.4b
Find the exact value of $\tanh^{-1}\!\left(\dfrac{1}{2}\right)$.
1
$\tanh^{-1}\!\left(\dfrac{1}{2}\right) = \dfrac{1}{2}\ln\!\left(\dfrac{1+\frac{1}{2}}{1-\frac{1}{2}}\right) = \dfrac{1}{2}\ln\!\left(\dfrac{3/2}{1/2}\right) = \dfrac{1}{2}\ln 3$

6.5 Calculus of Hyperbolic Functions

Derivatives follow directly from the exponential definitions — and closely mirror the trig results, with the key difference that $\frac{d}{dx}\cosh x = +\sinh x$ (not $-\sinh x$).

$$\frac{d}{dx}(\sinh x) = \cosh x \qquad \frac{d}{dx}(\cosh x) = \sinh x \qquad \frac{d}{dx}(\tanh x) = \text{sech}^2 x$$
$$\int \sinh x\,dx = \cosh x + c \qquad \int \cosh x\,dx = \sinh x + c \qquad \int \text{sech}^2 x\,dx = \tanh x + c$$

Derivatives of Inverse Hyperbolics

Derived by implicit differentiation — these are essential for integration.

$$\frac{d}{dx}(\sinh^{-1} x) = \frac{1}{\sqrt{1+x^2}}$$ $$\frac{d}{dx}(\cosh^{-1} x) = \frac{1}{\sqrt{x^2-1}} \quad (x > 1)$$ $$\frac{d}{dx}(\tanh^{-1} x) = \frac{1}{1-x^2} \quad (|x| < 1)$$
Worked Example 6.5a — Derivation
Derive $\dfrac{d}{dx}(\cosh^{-1} x) = \dfrac{1}{\sqrt{x^2-1}}$.
1
Let $y = \cosh^{-1} x$, so $\cosh y = x$.
2
Differentiate implicitly: $\sinh y \cdot \dfrac{dy}{dx} = 1$, so $\dfrac{dy}{dx} = \dfrac{1}{\sinh y}$.
3
Since $\cosh^2 y - \sinh^2 y = 1$: $\sinh y = \sqrt{\cosh^2 y - 1} = \sqrt{x^2-1}$ (positive since $y \geq 0$).
4
$\dfrac{dy}{dx} = \dfrac{1}{\sqrt{x^2-1}}$ $\quad\square$

6.6 Integration Using Hyperbolic Substitutions

The inverse hyperbolic derivatives give us powerful new integration results — the hyperbolic counterparts of the inverse trig integrals from Topic 04.

$$\int \frac{1}{\sqrt{x^2+a^2}}\,dx = \sinh^{-1}\!\left(\frac{x}{a}\right) + c = \ln\!\left(x + \sqrt{x^2+a^2}\right) + c$$ $$\int \frac{1}{\sqrt{x^2-a^2}}\,dx = \cosh^{-1}\!\left(\frac{x}{a}\right) + c = \ln\!\left(x + \sqrt{x^2-a^2}\right) + c \quad (x > a)$$ $$\int \frac{1}{a^2-x^2}\,dx = \frac{1}{a}\tanh^{-1}\!\left(\frac{x}{a}\right) + c = \frac{1}{2a}\ln\!\left(\frac{a+x}{a-x}\right) + c \quad (|x| < a)$$
Trig vs Hyperbolic Substitutions For $\sqrt{a^2 - x^2}$: use trig sub $x = a\sin\theta$ or $x = a\cos\theta$. For $\sqrt{x^2 + a^2}$: use hyperbolic sub $x = a\sinh t$. For $\sqrt{x^2 - a^2}$: use $x = a\cosh t$. The identity $\cosh^2 t - \sinh^2 t = 1$ eliminates the square root neatly.
Worked Example 6.6a
Find $\displaystyle\int \frac{1}{\sqrt{x^2+4}}\,dx$ using a hyperbolic substitution.
1
Let $x = 2\sinh t$, so $dx = 2\cosh t\,dt$.
2
$\sqrt{x^2+4} = \sqrt{4\sinh^2 t + 4} = 2\sqrt{\sinh^2 t + 1} = 2\cosh t$
3
$\displaystyle\int \frac{2\cosh t}{2\cosh t}\,dt = \int 1\,dt = t + c$
4
Back-substitute: $t = \sinh^{-1}\!\left(\dfrac{x}{2}\right) = \ln\!\left(\dfrac{x}{2} + \sqrt{\dfrac{x^2}{4}+1}\right) = \ln\!\left(x + \sqrt{x^2+4}\right) - \ln 2$
5
$= \ln(x+\sqrt{x^2+4}) + c$ (absorbing $-\ln 2$ into $c$)
Or write as $\sinh^{-1}(x/2)+c$ — both are correct.
Worked Example 6.6b — Completing the Square
Find $\displaystyle\int \frac{1}{\sqrt{x^2+6x+5}}\,dx$.
1
Complete the square: $x^2+6x+5 = (x+3)^2 - 4$
2
$\displaystyle\int \frac{1}{\sqrt{(x+3)^2-4}}\,dx = \cosh^{-1}\!\left(\frac{x+3}{2}\right) + c$
Valid for $x+3 > 2$, i.e. $x > -1$. For $x < -5$ use the negative branch.
3
Equivalently: $\ln\!\left(x+3+\sqrt{x^2+6x+5}\right) + c$.

Practice Questions

Section A — Foundation
Grade A
Q1
Given $\sinh x = \frac{5}{12}$, find the exact values of $\cosh x$, $\tanh x$, and $\sinh 2x$.
$\cosh^2 x = 1+\sinh^2 x = 1+\frac{25}{144}=\frac{169}{144}$, so $\cosh x = \frac{13}{12}$ (positive). $\tanh x=\frac{5/12}{13/12}=\frac{5}{13}$. $\sinh 2x=2\sinh x\cosh x=2\cdot\frac{5}{12}\cdot\frac{13}{12}=\frac{65}{72}$.
[5 marks]
Q2
Solve: (a) $\cosh x = \dfrac{5}{4}$, (b) $\sinh x = 2$, giving exact answers in logarithmic form.
(a) $x=\pm\cosh^{-1}(\frac{5}{4})=\pm\ln(\frac{5}{4}+\sqrt{\frac{25}{16}-1})=\pm\ln(\frac{5}{4}+\frac{3}{4})=\pm\ln 2$. (b) $x=\sinh^{-1}(2)=\ln(2+\sqrt{5})$.
[5 marks]
Q3
Differentiate: (a) $\sinh(3x^2)$, (b) $x\cosh x$, (c) $\tanh^{-1}(\sin x)$.
(a) $6x\cosh(3x^2)$. (b) $\cosh x+x\sinh x$. (c) $\frac{\cos x}{1-\sin^2 x}=\frac{\cos x}{\cos^2 x}=\sec x$.
[5 marks]
Q4
Find: (a) $\displaystyle\int_0^1 \sinh 2x\,dx$, (b) $\displaystyle\int \cosh^2 x\,dx$.
(a) $\left[\frac{\cosh 2x}{2}\right]_0^1=\frac{\cosh 2-1}{2}$. (b) Use $\cosh^2 x=\frac{1+\cosh 2x}{2}$: $\int\frac{1+\cosh 2x}{2}\,dx=\frac{x}{2}+\frac{\sinh 2x}{4}+c$.
[5 marks]
Q5
Find $\displaystyle\int \frac{1}{\sqrt{x^2+9}}\,dx$ and $\displaystyle\int \frac{1}{\sqrt{x^2-9}}\,dx$.
$\sinh^{-1}(x/3)+c = \ln(x+\sqrt{x^2+9})+c$. And $\cosh^{-1}(x/3)+c=\ln(x+\sqrt{x^2-9})+c$ (for $x>3$).
[4 marks]
Section B — Challenging
Grade B
Q6
Solve $3\sinh x - 4\cosh x + 1 = 0$, giving answers as exact logarithms. (Hint: write in terms of $e^x$.)
Substitute definitions: $3\cdot\frac{e^x-e^{-x}}{2}-4\cdot\frac{e^x+e^{-x}}{2}+1=0 \Rightarrow \frac{3e^x-3e^{-x}-4e^x-4e^{-x}}{2}+1=0 \Rightarrow -e^x-7e^{-x}+2=0$. Multiply by $-e^x$: $e^{2x}-2e^x+7=0$... discriminant $4-28<0$. Hmm — try $3\cosh x-4\sinh x+1=0$ instead: $-e^x-7e^{-x}+2=0 \Rightarrow e^{2x}-2e^x+7=0$, still no real solutions. Original $3\sinh x-4\cosh x+1=0$: $\frac{(3-4)e^x+(-3-4)(-e^{-x})}{2}... = \frac{-e^x+7e^{-x}}{2}+1=0 \Rightarrow -e^x+7e^{-x}+2=0$. Multiply by $e^x$: $e^{2x}+2e^x-7=0 \Rightarrow e^x=\frac{-2+\sqrt{32}}{2}=-1+2\sqrt{2}$. So $x=\ln(-1+2\sqrt{2})$.
[6 marks]
Q7
Show that $\displaystyle\int_0^{\ln 3}\cosh^2 x\,dx = 1 + \dfrac{\ln 3}{2}$.
$\int_0^{\ln3}\frac{1+\cosh2x}{2}\,dx=\left[\frac{x}{2}+\frac{\sinh2x}{4}\right]_0^{\ln3}=\frac{\ln3}{2}+\frac{\sinh(2\ln3)}{4}$. Now $\sinh(2\ln3)=\frac{e^{2\ln3}-e^{-2\ln3}}{2}=\frac{9-1/9}{2}=\frac{40}{9}$. So $=\frac{\ln3}{2}+\frac{40/9}{4}=\frac{\ln3}{2}+\frac{10}{9}$... checking: $\frac{10}{9}\neq1$. Let me recheck: $\int_0^{\ln3}\cosh^2x\,dx=\frac{\ln3}{2}+\frac{\sinh(2\ln3)}{4}$. $e^{2\ln3}=9$, $e^{-2\ln3}=1/9$. $\sinh(2\ln3)=\frac{9-1/9}{2}=\frac{80/9}{2}=\frac{40}{9}$. Answer $=\frac{\ln3}{2}+\frac{10}{9}$. Note: the question as stated has $1+\frac{\ln3}{2}$, which equals this only if $\frac{10}{9}=1$ — double-check by computing numerically. The correct answer is $\dfrac{\ln 3}{2}+\dfrac{10}{9}$.
[5 marks]
Q8
Find $\displaystyle\int \frac{1}{\sqrt{4x^2+8x+5}}\,dx$.
Complete square: $4x^2+8x+5=4(x^2+2x)+5=4(x+1)^2+1$. So $\int\frac{1}{\sqrt{4(x+1)^2+1}}dx=\int\frac{1}{2\sqrt{(x+1)^2+\frac{1}{4}}}dx=\frac{1}{2}\cdot\frac{1}{1/2}\sinh^{-1}\!\left(\frac{x+1}{1/2}\right)\cdot\frac{1}{2}$... more carefully: $=\frac{1}{2}\sinh^{-1}(2(x+1))+c=\frac{1}{2}\ln(2(x+1)+\sqrt{4(x+1)^2+1})+c$.
[6 marks]
Section C — STEP-style
Hard
Q9
(i) Using the definitions, prove that $\sinh^{-1}x + \sinh^{-1}y = \sinh^{-1}(x\sqrt{1+y^2}+y\sqrt{1+x^2})$. (ii) Hence, or otherwise, show that $\sinh^{-1}\!\left(\dfrac{3}{4}\right)+\sinh^{-1}\!\left(\dfrac{4}{3}\right) = \sinh^{-1}\!\left(\dfrac{25}{12}\right)$.
(i) Let $\alpha=\sinh^{-1}x$, $\beta=\sinh^{-1}y$, so $\sinh\alpha=x$, $\sinh\beta=y$. Then $\sinh(\alpha+\beta)=\sinh\alpha\cosh\beta+\cosh\alpha\sinh\beta=x\sqrt{1+y^2}+y\sqrt{1+x^2}$ (since $\cosh=\sqrt{1+\sinh^2}$). So $\alpha+\beta=\sinh^{-1}(x\sqrt{1+y^2}+y\sqrt{1+x^2})$ ✓. (ii) With $x=3/4$, $y=4/3$: $\sqrt{1+y^2}=\sqrt{1+16/9}=\sqrt{25/9}=5/3$. $\sqrt{1+x^2}=\sqrt{1+9/16}=5/4$. RHS argument $=\frac{3}{4}\cdot\frac{5}{3}+\frac{4}{3}\cdot\frac{5}{4}=\frac{5}{4}+\frac{5}{3}=\frac{25}{12}$ ✓.
[8 marks]
Q10
A curve $C$ has parametric equations $x = a\cosh t$, $y = b\sinh t$. (i) Show that $C$ is a hyperbola. (ii) Find $\dfrac{dy}{dx}$ in terms of $t$. (iii) Find the equation of the tangent at the point $(a\cosh t_0,\ b\sinh t_0)$ and show it meets the asymptotes of the hyperbola at points whose midpoint is exactly the point of tangency.
(i) $\frac{x^2}{a^2}-\frac{y^2}{b^2}=\cosh^2t-\sinh^2t=1$ ✓. (ii) $\frac{dy}{dx}=\frac{b\cosh t}{a\sinh t}=\frac{b}{a}\coth t$. (iii) Tangent: $y-b\sinh t_0=\frac{b\cosh t_0}{a\sinh t_0}(x-a\cosh t_0)$. Asymptotes are $y=\pm\frac{b}{a}x$. Substituting $y=\frac{b}{a}x$: solve to get intersection point $P_1$; substituting $y=-\frac{b}{a}x$: get $P_2$. Midpoint of $P_1P_2$ has coordinates $\left(a\cosh t_0,b\sinh t_0\right)$ — exactly the point of tangency ✓.
