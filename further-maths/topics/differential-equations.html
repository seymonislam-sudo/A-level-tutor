7.1 First-Order: Separable Equations

A first-order ODE is separable if it can be written as $g(y)\,dy = f(x)\,dx$ — variables on opposite sides. Integrate both sides to find the general solution.

$$\frac{dy}{dx} = f(x)g(y) \implies \int \frac{1}{g(y)}\,dy = \int f(x)\,dx$$
Worked Example 7.1a
Solve $\dfrac{dy}{dx} = xy^2$, given $y = 1$ when $x = 0$.
1
Separate: $\dfrac{1}{y^2}\,dy = x\,dx$
2
Integrate: $-\dfrac{1}{y} = \dfrac{x^2}{2} + c$
3
Apply $y=1$, $x=0$: $-1 = 0 + c$, so $c = -1$.
4
Rearrange: $y = \dfrac{1}{1 - x^2/2} = \dfrac{2}{2-x^2}$
Worked Example 7.1b
Solve $\dfrac{dy}{dx} = \dfrac{y\ln y}{x}$.
1
Separate: $\dfrac{1}{y\ln y}\,dy = \dfrac{1}{x}\,dx$
2
Left side: let $u = \ln y$, $du = \frac{1}{y}\,dy$: $\displaystyle\int\frac{du}{u} = \ln|\ln y|$
3
$\ln|\ln y| = \ln|x| + c$, so $\ln y = Ax$ for some constant $A$, giving $y = e^{Ax}$.

7.2 First-Order: Integrating Factor

For first-order linear ODEs of the form $\dfrac{dy}{dx} + P(x)y = Q(x)$, multiply through by the integrating factor $I = e^{\int P\,dx}$. This turns the left side into an exact derivative.

$$\frac{dy}{dx} + P(x)y = Q(x) \implies I = e^{\int P(x)\,dx}$$ $$\frac{d}{dx}(Iy) = IQ(x) \implies Iy = \int IQ(x)\,dx$$
Why it works After multiplying by $I = e^{\int P\,dx}$, the left side becomes $\frac{d}{dx}(Iy)$ by the product rule — since $I' = PI$. So the equation becomes $\frac{d}{dx}(Iy) = IQ$, which integrates directly.
Worked Example 7.2a
Solve $\dfrac{dy}{dx} + \dfrac{2y}{x} = x^2$.
1
$P(x) = \dfrac{2}{x}$. Integrating factor: $I = e^{\int 2/x\,dx} = e^{2\ln x} = x^2$.
2
Multiply through: $x^2\dfrac{dy}{dx} + 2xy = x^4$, i.e. $\dfrac{d}{dx}(x^2 y) = x^4$.
3
Integrate: $x^2 y = \dfrac{x^5}{5} + c$
4
General solution: $y = \dfrac{x^3}{5} + \dfrac{c}{x^2}$

7.3 Second-Order Homogeneous ODEs

The standard form is:

$$a\frac{d^2y}{dx^2} + b\frac{dy}{dx} + cy = 0$$

where $a$, $b$, $c$ are constants. The general solution is built from exponential functions found via the auxiliary equation.

Structure of the Solution A second-order linear ODE always has a general solution with exactly two arbitrary constants. These are determined by initial or boundary conditions. The complementary function (CF) is the full solution to the homogeneous equation.

7.4 The Auxiliary Equation

Try the solution $y = e^{mx}$. Substituting gives $am^2 + bm + c = 0$ — the auxiliary equation. The nature of its roots determines the form of the complementary function.

$$am^2 + bm + c = 0$$

Case 1: Two distinct real roots $m_1 \neq m_2$

$$y = Ae^{m_1 x} + Be^{m_2 x}$$

Case 2: Repeated real root $m_1 = m_2 = m$

$$y = (A + Bx)e^{mx}$$

The extra factor of $x$ is essential — without it you only have one independent solution.

Case 3: Complex conjugate roots $m = p \pm qi$

$$y = e^{px}(A\cos qx + B\sin qx)$$

This arises from $e^{(p+qi)x} = e^{px}(\cos qx + i\sin qx)$ combined with its conjugate.

Worked Example 7.4a
Find the general solution of $\dfrac{d^2y}{dx^2} - 5\dfrac{dy}{dx} + 6y = 0$.
1
Auxiliary equation: $m^2 - 5m + 6 = 0 \Rightarrow (m-2)(m-3) = 0$, so $m = 2$ or $m = 3$.
2
Two distinct real roots → $y = Ae^{2x} + Be^{3x}$
Worked Example 7.4b
Find the general solution of $\dfrac{d^2y}{dx^2} + 4\dfrac{dy}{dx} + 4y = 0$.
1
Auxiliary: $m^2 + 4m + 4 = 0 \Rightarrow (m+2)^2 = 0$, so $m = -2$ (repeated).
2
Repeated root → $y = (A+Bx)e^{-2x}$
Worked Example 7.4c
Find the general solution of $\dfrac{d^2y}{dx^2} - 4\dfrac{dy}{dx} + 13y = 0$.
1
Auxiliary: $m^2 - 4m + 13 = 0$. Discriminant: $16 - 52 < 0$. $m = \dfrac{4 \pm \sqrt{-36}}{2} = 2 \pm 3i$.
2
Complex roots $p=2$, $q=3$ → $y = e^{2x}(A\cos 3x + B\sin 3x)$

7.5 Particular Integrals

For the non-homogeneous equation $a y'' + b y' + cy = f(x)$, the general solution is:

$$y = \underbrace{\text{CF}}_{\text{complementary function}} + \underbrace{\text{PI}}_{\text{particular integral}}$$

The PI is any single solution to the full equation. Guess its form based on $f(x)$:

$$f(x) = ke^{\lambda x} \implies \text{try } y_p = \lambda e^{\lambda x}$$ $$f(x) = k\sin nx \text{ or } k\cos nx \implies \text{try } y_p = p\cos nx + q\sin nx$$ $$f(x) = \text{polynomial degree } n \implies \text{try } y_p = \text{polynomial degree } n$$
⚠ When the trial PI is part of the CF If your trial PI already appears in the complementary function, multiply it by $x$. If that also appears, multiply by $x^2$. For example, if $f(x) = e^{2x}$ but $e^{2x}$ is already in the CF, try $xe^{2x}$ as the PI instead.
Worked Example 7.5a
Find the general solution of $\dfrac{d^2y}{dx^2} - 5\dfrac{dy}{dx} + 6y = 2e^x$.
1
CF (from 7.4a): $y_c = Ae^{2x} + Be^{3x}$
2
Try PI: $y_p = \lambda e^x$. Then $y_p' = \lambda e^x$, $y_p'' = \lambda e^x$.
3
Substitute: $\lambda e^x - 5\lambda e^x + 6\lambda e^x = 2e^x \Rightarrow 2\lambda = 2 \Rightarrow \lambda = 1$. So $y_p = e^x$.
4
General solution: $y = Ae^{2x} + Be^{3x} + e^x$
Worked Example 7.5b
Find a particular integral for $y'' + y = \sin 2x$.
1
Try $y_p = p\cos 2x + q\sin 2x$. Then $y_p'' = -4p\cos 2x - 4q\sin 2x$.
2
Substitute: $(-4p+p)\cos 2x + (-4q+q)\sin 2x = \sin 2x$
3
$-3p = 0$ and $-3q = 1$, so $p = 0$, $q = -\dfrac{1}{3}$. Thus $y_p = -\dfrac{\sin 2x}{3}$.

7.6 General Solution & Boundary Conditions

The general solution $y = \text{CF} + \text{PI}$ contains two arbitrary constants. These are pinned down by initial conditions (values of $y$ and $y'$ at a single point) or boundary conditions (values of $y$ at two different points).

Worked Example 7.6a
Solve $y'' - 5y' + 6y = 2e^x$ given $y = 1$ and $y' = 0$ when $x = 0$.
1
From 7.5a: $y = Ae^{2x} + Be^{3x} + e^x$
2
$y' = 2Ae^{2x} + 3Be^{3x} + e^x$
3
At $x=0$: $y=1 \Rightarrow A + B + 1 = 1 \Rightarrow A + B = 0$
4
At $x=0$: $y'=0 \Rightarrow 2A + 3B + 1 = 0 \Rightarrow 2A + 3B = -1$
5
From $A = -B$: $-2B + 3B = -1 \Rightarrow B = -1$, $A = 1$.
6
Particular solution: $y = e^{2x} - e^{3x} + e^x$

7.7 Substitution Methods

Some ODEs can be reduced to a standard form by a substitution, either given in the question or suggested by the structure of the equation.

Worked Example 7.7a — Bernoulli Equation
Use the substitution $v = y^{1-n}$ to solve the Bernoulli equation $\dfrac{dy}{dx} + P(x)y = Q(x)y^n$. Apply to $\dfrac{dy}{dx} - \dfrac{y}{x} = xy^2$.
1
Here $n=2$, so let $v = y^{-1} = 1/y$. Then $\dfrac{dv}{dx} = -y^{-2}\dfrac{dy}{dx}$.
2
Divide the ODE by $y^2$: $y^{-2}\dfrac{dy}{dx} - \dfrac{1}{xy} = x$
3
Substitute $v = 1/y$, $-dv/dx = y^{-2}dy/dx$: $-\dfrac{dv}{dx} - \dfrac{v}{x} = x$, i.e. $\dfrac{dv}{dx} + \dfrac{v}{x} = -x$
4
Linear in $v$! Integrating factor $I = x$: $\dfrac{d}{dx}(xv) = -x^2 \Rightarrow xv = -\dfrac{x^3}{3} + c$
5
So $v = -\dfrac{x^2}{3} + \dfrac{c}{x}$, i.e. $\dfrac{1}{y} = \dfrac{c}{x} - \dfrac{x^2}{3}$, giving $y = \dfrac{3x}{3c-x^3}$
Worked Example 7.7b — Reduction of Order
Given that $y_1 = e^x$ is a solution of $y'' - 2y' + y = 0$, use the substitution $y = ve^x$ to find the general solution.
1
$y = ve^x$, $y' = v'e^x + ve^x$, $y'' = v''e^x + 2v'e^x + ve^x$
2
Substitute: $e^x(v'' + 2v' + v) - 2e^x(v' + v) + ve^x = 0$
3
Simplify: $e^x(v'' + 2v' + v - 2v' - 2v + v) = e^x\cdot v'' = 0$, so $v'' = 0$
4
$v = A + Bx$, so $y = (A+Bx)e^x$ ✓ — confirms repeated root result.

7.8 Coupled Differential Equations

A system of coupled ODEs involves two (or more) functions of $t$ that depend on each other. The standard approach is to eliminate one variable to get a single second-order ODE.

Worked Example 7.8a
Solve the coupled system: $\dfrac{dx}{dt} = x + y$, $\dfrac{dy}{dt} = x - y$.
1
From the first equation: $y = \dot{x} - x$ (where dot = $d/dt$).
2
Differentiate: $\dot{y} = \ddot{x} - \dot{x}$. Substitute into the second equation: $\ddot{x} - \dot{x} = x - (\dot{x} - x) = 2x - \dot{x}$
3
$\ddot{x} - 2x = 0$. Auxiliary: $m^2 = 2 \Rightarrow m = \pm\sqrt{2}$.
4
$x = Ae^{\sqrt{2}t} + Be^{-\sqrt{2}t}$
5
$y = \dot{x} - x = (\sqrt{2}-1)Ae^{\sqrt{2}t} + (-\sqrt{2}-1)Be^{-\sqrt{2}t}$
Method Summary (1) Express one variable in terms of the other and its derivatives. (2) Differentiate and substitute to get a single second-order ODE. (3) Solve it. (4) Back-substitute to find the second variable — use the original equations, not new arbitrary constants.

Practice Questions

Section A — Foundation
Grade A
Q1
Solve the following separable ODEs: (a) $\dfrac{dy}{dx} = \dfrac{x}{y}$, (b) $\dfrac{dy}{dx} = e^{x+y}$, (c) $\dfrac{dy}{dx} = \dfrac{\sin x}{y^2}$ with $y=1$ when $x=0$.
(a) $y\,dy=x\,dx \Rightarrow y^2=x^2+c$. (b) $e^{-y}dy=e^x dx \Rightarrow -e^{-y}=e^x+c$. (c) $y^2\,dy=\sin x\,dx \Rightarrow \frac{y^3}{3}=-\cos x+c$. At $x=0,y=1$: $\frac{1}{3}=-1+c \Rightarrow c=\frac{4}{3}$. So $y^3=4-3\cos x$.
[7 marks]
Q2
Solve using an integrating factor: $\dfrac{dy}{dx} - 3y = e^{2x}$.
$I=e^{-3x}$. Multiply: $\frac{d}{dx}(e^{-3x}y)=e^{-x}$. Integrate: $e^{-3x}y=-e^{-x}+c$. So $y=-e^{2x}+ce^{3x}$.
[5 marks]
Q3
Find the general solution of each: (a) $y''-3y'-4y=0$, (b) $y''+6y'+9y=0$, (c) $y''+2y'+5y=0$.
(a) $m^2-3m-4=0 \Rightarrow (m-4)(m+1)=0$, so $y=Ae^{4x}+Be^{-x}$. (b) $(m+3)^2=0$, repeated, so $y=(A+Bx)e^{-3x}$. (c) $m=-1\pm2i$, so $y=e^{-x}(A\cos2x+B\sin2x)$.
[7 marks]
Q4
Find the general solution of $y'' - y' - 2y = 6e^{3x}$.
CF: $m^2-m-2=(m-2)(m+1)=0$, so $y_c=Ae^{2x}+Be^{-x}$. PI: try $y_p=\lambda e^{3x}$: $9\lambda-3\lambda-2\lambda=6 \Rightarrow 4\lambda=6 \Rightarrow \lambda=\frac{3}{2}$. General solution: $y=Ae^{2x}+Be^{-x}+\frac{3}{2}e^{3x}$.
[6 marks]
Q5
Solve $y'' + 4y = 8x^2$ with $y(0) = 1$, $y'(0) = 0$.
CF: $m^2+4=0 \Rightarrow m=\pm2i$, so $y_c=A\cos2x+B\sin2x$. PI: try $y_p=ax^2+bx+c$. $y_p''=2a$. Substitute: $2a+4(ax^2+bx+c)=8x^2 \Rightarrow 4a=8,4b=0,2a+4c=0 \Rightarrow a=2,b=0,c=-1$. So $y_p=2x^2-1$. General: $y=A\cos2x+B\sin2x+2x^2-1$. ICs: $y(0)=A-1=1\Rightarrow A=2$. $y'=-2A\sin2x+2B\cos2x+4x$: $y'(0)=2B=0\Rightarrow B=0$. Solution: $y=2\cos2x+2x^2-1$.
[8 marks]
Section B — Challenging
Grade B
Q6
Find the general solution of $y'' - 4y' + 4y = e^{2x}$. (Note: the trial PI clashes with the CF.)
CF: $(m-2)^2=0$, so $y_c=(A+Bx)e^{2x}$. Trial $\lambda e^{2x}$ is in CF; trial $\lambda xe^{2x}$ is also in CF. Try $y_p=\lambda x^2 e^{2x}$. $y_p'=\lambda e^{2x}(2x^2+2x)$, $y_p''=\lambda e^{2x}(4x^2+8x+2)$. Substitute: $\lambda e^{2x}[(4x^2+8x+2)-4(2x^2+2x)+(4x^2)]=\lambda e^{2x}\cdot2=e^{2x}$. So $\lambda=\frac{1}{2}$. General solution: $y=(A+Bx)e^{2x}+\frac{x^2}{2}e^{2x}$.
[7 marks]
Q7
Solve the coupled system $\dot{x} = 3x - 2y$, $\dot{y} = 2x - y$ with $x(0) = 1$, $y(0) = 0$.
From first: $y=\frac{3x-\dot{x}}{2}$. Differentiate: $\dot{y}=\frac{3\dot{x}-\ddot{x}}{2}$. Sub into second: $\frac{3\dot{x}-\ddot{x}}{2}=2x-\frac{3x-\dot{x}}{2} \Rightarrow 3\dot{x}-\ddot{x}=4x-3x+\dot{x}=x+\dot{x} \Rightarrow \ddot{x}-2\dot{x}+x=0$. Aux: $(m-1)^2=0$, so $x=(A+Bt)e^t$. $y=\frac{3x-\dot{x}}{2}$: $\dot{x}=(A+Bt)e^t+Be^t=(A+B+Bt)e^t$. $y=\frac{(3A+3Bt)e^t-(A+B+Bt)e^t}{2}=\frac{(2A-B+2Bt)e^t}{2}$. ICs: $x(0)=A=1$. $y(0)=\frac{2A-B}{2}=0\Rightarrow B=2$. Solution: $x=(1+2t)e^t$, $y=te^t$.
[9 marks]
Q8
By using the substitution $y = vx$, solve the homogeneous equation $\dfrac{dy}{dx} = \dfrac{x^2 + y^2}{xy}$.
Let $y=vx$, $\frac{dy}{dx}=v+x\frac{dv}{dx}$. RHS $=\frac{x^2+v^2x^2}{vx^2}=\frac{1+v^2}{v}$. So $v+x\frac{dv}{dx}=\frac{1+v^2}{v} \Rightarrow x\frac{dv}{dx}=\frac{1+v^2-v^2}{v}=\frac{1}{v}$. Separate: $v\,dv=\frac{dx}{x} \Rightarrow \frac{v^2}{2}=\ln|x|+c$. Back-substitute: $\frac{y^2}{2x^2}=\ln|x|+c$, i.e. $y^2=2x^2(\ln|x|+c)$.
[7 marks]
Section C — STEP-style
Hard
Q9
The equation $x^2y'' + xy' - y = x^2$ is an Euler–Cauchy equation. (i) Use the substitution $x = e^t$ to transform it into a second-order ODE with constant coefficients (in terms of $t$ and $y$). (ii) Solve the transformed equation. (iii) Write the solution in terms of $x$.
(i) With $x=e^t$: $\frac{dy}{dx}=\frac{1}{x}\frac{dy}{dt}$, $\frac{d^2y}{dx^2}=\frac{1}{x^2}\left(\frac{d^2y}{dt^2}-\frac{dy}{dt}\right)$. Substituting: $\frac{d^2y}{dt^2}-\frac{dy}{dt}+\frac{dy}{dt}-y=e^{2t}$, i.e. $\frac{d^2y}{dt^2}-y=e^{2t}$. (ii) CF: $m^2-1=0\Rightarrow y_c=Ae^t+Be^{-t}$. PI: try $\lambda e^{2t}$: $4\lambda-\lambda=1\Rightarrow\lambda=\frac{1}{3}$. General: $y=Ae^t+Be^{-t}+\frac{e^{2t}}{3}$. (iii) Replace $e^t=x$: $y=Ax+\frac{B}{x}+\frac{x^2}{3}$.
[10 marks]
Q10
The simple harmonic oscillator with damping satisfies $\ddot{x} + 2k\dot{x} + \omega^2 x = 0$ where $k, \omega > 0$. (i) Find the general solution in all three cases: overdamped ($k > \omega$), critically damped ($k = \omega$), underdamped ($k < \omega$). (ii) Show that in the underdamped case, the amplitude decays exponentially. (iii) In the underdamped case, find the times $t_n$ at which $\dot{x} = 0$ and show they are equally spaced — find this spacing.
(i) Aux: $m^2+2km+\omega^2=0$, $m=-k\pm\sqrt{k^2-\omega^2}$. Overdamped ($k>\omega$): $x=Ae^{m_1t}+Be^{m_2t}$ (two real roots). Critically damped ($k=\omega$): $x=(A+Bt)e^{-kt}$. Underdamped ($k<\omega$): let $\mu=\sqrt{\omega^2-k^2}$; $x=e^{-kt}(A\cos\mu t+B\sin\mu t)$. (ii) Write $A\cos\mu t+B\sin\mu t=R\cos(\mu t-\phi)$ where $R=\sqrt{A^2+B^2}$. Then $|x|\leq Re^{-kt}\to0$ — amplitude $Re^{-kt}$ decays exponentially ✓. (iii) $\dot{x}=e^{-kt}[(-kA+\mu B)\cos\mu t+(-kB-\mu A)\sin\mu t]=0$. This gives $\tan\mu t=\frac{-kA+\mu B}{kB+\mu A}=\text{const}$, so $\mu t_n = \arctan(\cdots)+n\pi$. Spacing $t_{n+1}-t_n=\frac{\pi}{\mu}=\frac{\pi}{\sqrt{\omega^2-k^2}}$, the half-period of oscillation.
[10 marks]
