11.1 PDF and CDF

A continuous random variable $X$ is described by its probability density function (PDF) $f(x)$. Unlike discrete distributions, probabilities are areas under the curve — $P(X = x) = 0$ for any single value.

$$P(a \leq X \leq b) = \int_a^b f(x)\,dx$$ $$\text{Axioms: } f(x) \geq 0 \text{ for all } x, \quad \int_{-\infty}^{\infty} f(x)\,dx = 1$$

The cumulative distribution function (CDF) $F(x) = P(X \leq x) = \displaystyle\int_{-\infty}^{x} f(t)\,dt$.

$$F(x) = \int_{-\infty}^x f(t)\,dt \qquad f(x) = F'(x)$$ $$P(a < X < b) = F(b) - F(a)$$
Key properties of the CDF $F(x)$ is non-decreasing. $F(-\infty) = 0$ and $F(\infty) = 1$. For a continuous distribution, $P(X \leq x) = P(X < x)$ — we don't need to worry about whether endpoints are included.
Worked Example 11.1a
The random variable $X$ has PDF $f(x) = kx^2$ for $0 \leq x \leq 3$, and $f(x) = 0$ otherwise. Find $k$, the CDF $F(x)$, and $P(1 \leq X \leq 2)$.
1
$\displaystyle\int_0^3 kx^2\,dx = k\left[\frac{x^3}{3}\right]_0^3 = 9k = 1$, so $k = \dfrac{1}{9}$.
2
For $0 \leq x \leq 3$: $F(x) = \displaystyle\int_0^x \frac{t^2}{9}\,dt = \dfrac{x^3}{27}$. So $F(x) = \begin{cases}0 & x < 0 \\ x^3/27 & 0 \leq x \leq 3 \\ 1 & x > 3\end{cases}$
3
$P(1 \leq X \leq 2) = F(2) - F(1) = \dfrac{8}{27} - \dfrac{1}{27} = \dfrac{7}{27}$

11.2 Mean, Variance and Mode

$$E(X) = \int_{-\infty}^{\infty} x\,f(x)\,dx \qquad E(g(X)) = \int_{-\infty}^{\infty} g(x)\,f(x)\,dx$$ $$\text{Var}(X) = E(X^2) - [E(X)]^2 \qquad E(X^2) = \int_{-\infty}^{\infty} x^2 f(x)\,dx$$

The mode is the value of $x$ that maximises $f(x)$ — find it by solving $f'(x) = 0$ and checking it's a maximum.

The median $m$ satisfies $F(m) = 0.5$, i.e. $\displaystyle\int_{-\infty}^{m}f(x)\,dx = \frac{1}{2}$.

Worked Example 11.2a
For $f(x) = \frac{1}{9}x^2$ on $[0,3]$, find $E(X)$, $\text{Var}(X)$, the mode, and the median.
1
$E(X) = \displaystyle\int_0^3 x \cdot \frac{x^2}{9}\,dx = \frac{1}{9}\left[\frac{x^4}{4}\right]_0^3 = \frac{81}{36} = \dfrac{9}{4} = 2.25$
2
$E(X^2) = \displaystyle\int_0^3 \frac{x^4}{9}\,dx = \frac{1}{9}\cdot\frac{243}{5} = \dfrac{27}{5}$. $\text{Var}(X) = \dfrac{27}{5} - \left(\dfrac{9}{4}\right)^2 = \dfrac{27}{5} - \dfrac{81}{16} = \dfrac{432-405}{80} = \dfrac{27}{80}$
3
Mode: $f(x) = \frac{x^2}{9}$ is increasing on $[0,3]$, so mode $= 3$ (right endpoint).
4
Median: $F(m) = \frac{m^3}{27} = \frac{1}{2} \Rightarrow m^3 = \frac{27}{2} \Rightarrow m = \dfrac{3}{\sqrt[3]{2}} = \dfrac{3\cdot 2^{2/3}}{2} \approx 2.38$

11.3 Continuous Uniform Distribution

$X \sim U(a, b)$ if $X$ is equally likely to take any value in $[a, b]$.

$$f(x) = \frac{1}{b-a}, \quad a \leq x \leq b$$ $$E(X) = \frac{a+b}{2} \qquad \text{Var}(X) = \frac{(b-a)^2}{12}$$ $$F(x) = \frac{x-a}{b-a}, \quad a \leq x \leq b$$
Worked Example 11.3a
$X \sim U(2, 10)$. Find $E(X)$, $\text{Var}(X)$, $P(3 < X < 7)$, and the value $c$ such that $P(X < c) = 0.3$.
1
$E(X) = 6$. $\text{Var}(X) = \dfrac{64}{12} = \dfrac{16}{3}$.
2
$P(3 < X < 7) = \dfrac{7-3}{10-2} = \dfrac{4}{8} = \dfrac{1}{2}$.
3
$F(c) = \dfrac{c-2}{8} = 0.3 \Rightarrow c = 2 + 0.3 \times 8 = 4.4$.

11.4 Exponential Distribution

$X \sim \text{Exp}(\lambda)$ models the waiting time between events in a Poisson process with rate $\lambda$. It is the continuous analogue of the geometric distribution.

$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$ $$F(x) = 1 - e^{-\lambda x}, \quad x \geq 0$$ $$E(X) = \frac{1}{\lambda} \qquad \text{Var}(X) = \frac{1}{\lambda^2}$$
Memoryless property Like the geometric distribution, the exponential is memoryless: $P(X > s + t \mid X > s) = P(X > t)$. This follows immediately from $P(X > x) = e^{-\lambda x}$: the ratio $P(X > s+t)/P(X > s) = e^{-\lambda t}$.
Link to Poisson If events occur as a Poisson process with rate $\lambda$ (events per unit time), then the waiting time between successive events is $\text{Exp}(\lambda)$. This gives the connection: $P(\text{no events in time } t) = e^{-\lambda t} = P(X > t)$ where $X \sim \text{Exp}(\lambda)$.
Worked Example 11.4a
Customers arrive at a shop at a rate of 4 per hour. The waiting time $X$ between arrivals follows $\text{Exp}(4)$. Find: (a) $P(X > 0.5)$, (b) $P(0.1 < X < 0.3)$, (c) the median waiting time.
1
(a) $P(X > 0.5) = 1 - F(0.5) = e^{-4(0.5)} = e^{-2} \approx 0.1353$
2
(b) $P(0.1 < X < 0.3) = F(0.3) - F(0.1) = (1-e^{-1.2}) - (1-e^{-0.4}) = e^{-0.4} - e^{-1.2} \approx 0.6703 - 0.3012 = 0.3691$
3
(c) $F(m) = 1 - e^{-4m} = 0.5 \Rightarrow e^{-4m} = 0.5 \Rightarrow m = \dfrac{\ln 2}{4} \approx 0.173$ hours

11.5 Further Normal Results

Linear combinations of normal variables

If $X \sim N(\mu_1, \sigma_1^2)$ and $Y \sim N(\mu_2, \sigma_2^2)$ are independent:

$$aX + bY \sim N(a\mu_1 + b\mu_2,\; a^2\sigma_1^2 + b^2\sigma_2^2)$$ $$\bar{X} = \frac{X_1 + \cdots + X_n}{n} \sim N\!\left(\mu,\; \frac{\sigma^2}{n}\right) \text{ if each } X_i \sim N(\mu,\sigma^2) \text{ i.i.d.}$$

The Central Limit Theorem

For any distribution with mean $\mu$ and variance $\sigma^2$, the sample mean $\bar{X}$ of a large sample of size $n$ is approximately:

$$\bar{X} \approx N\!\left(\mu,\; \frac{\sigma^2}{n}\right) \quad \text{for large } n \text{ (any distribution)}$$

This is the Central Limit Theorem (CLT) — one of the most profound results in probability. It explains why the normal distribution arises so naturally in practice.

Worked Example 11.5a
$X \sim N(10, 4)$ and $Y \sim N(6, 9)$ are independent. Find $P(X - Y > 7)$.
1
$X - Y \sim N(10-6,\; 4+9) = N(4, 13)$
2
$P(X-Y > 7) = P\!\left(Z > \dfrac{7-4}{\sqrt{13}}\right) = P(Z > 0.832) = 1 - \Phi(0.832) \approx 1 - 0.7973 = 0.2027$
Worked Example 11.5b — CLT
A machine produces bolts with diameter $X$ where $E(X) = 5$ mm and $\text{Var}(X) = 0.09$ mm². Find the approximate probability that the mean diameter of a sample of 50 bolts exceeds 5.05 mm.
1
By CLT: $\bar{X} \approx N\!\left(5,\; \dfrac{0.09}{50}\right) = N(5, 0.0018)$
2
$P(\bar{X} > 5.05) = P\!\left(Z > \dfrac{5.05-5}{\sqrt{0.0018}}\right) = P(Z > 1.179) \approx 1 - 0.8808 = 0.1192$

11.6 Transformations of Variables

If $Y = g(X)$ is a monotone function of $X$, the PDF of $Y$ is found via the CDF or by the change-of-variables formula.

$$f_Y(y) = f_X(g^{-1}(y)) \cdot \left|\frac{dx}{dy}\right| = f_X(x)\cdot\frac{1}{|g'(x)|}\bigg|_{x=g^{-1}(y)}$$
⚠ Always adjust the support When transforming, the domain of $Y$ changes too. Always determine the new range of $y$ from the range of $x$, and verify your new PDF integrates to 1.
Worked Example 11.6a — CDF method
$X \sim U(0,1)$. Find the PDF of $Y = -\ln X$.
1
Range: as $x$ goes from $0^+$ to $1$, $y = -\ln x$ goes from $+\infty$ to $0$. So $Y \geq 0$.
2
CDF of $Y$: $P(Y \leq y) = P(-\ln X \leq y) = P(X \geq e^{-y}) = 1 - e^{-y}$ (since $X \sim U(0,1)$).
3
Differentiate: $f_Y(y) = e^{-y}$ for $y \geq 0$. This is $\text{Exp}(1)$ — so $-\ln U \sim \text{Exp}(1)$ whenever $U \sim U(0,1)$.
This is the foundation of the inverse transform method for simulation.
Worked Example 11.6b — Formula method
$X$ has PDF $f(x) = 2x$ for $0 \leq x \leq 1$. Find the PDF of $Y = X^2$.
1
$y = x^2 \Rightarrow x = \sqrt{y}$ (taking positive root since $x \geq 0$). Range of $Y$: $0 \leq y \leq 1$.
2
$\dfrac{dx}{dy} = \dfrac{1}{2\sqrt{y}}$
3
$f_Y(y) = f_X(\sqrt{y})\cdot\dfrac{1}{2\sqrt{y}} = 2\sqrt{y}\cdot\dfrac{1}{2\sqrt{y}} = 1$ for $0 \leq y \leq 1$.
So $Y = X^2 \sim U(0,1)$ — a beautiful result.

11.7 Moment Generating Functions

The moment generating function (MGF) of a random variable $X$ is:

$$M_X(t) = E(e^{tX}) = \int_{-\infty}^{\infty} e^{tx} f(x)\,dx$$

The MGF uniquely determines the distribution (like the PGF for discrete variables). Its power lies in generating moments by differentiation:

$$E(X^r) = M_X^{(r)}(0) = \left.\frac{d^r}{dt^r}M_X(t)\right|_{t=0}$$

For independent $X$ and $Y$: $M_{X+Y}(t) = M_X(t) \cdot M_Y(t)$ — the product rule, just like PGFs.

Key MGFs

$$X \sim N(\mu, \sigma^2): \quad M_X(t) = e^{\mu t + \frac{1}{2}\sigma^2 t^2}$$ $$X \sim \text{Exp}(\lambda): \quad M_X(t) = \frac{\lambda}{\lambda - t}, \quad t < \lambda$$ $$X \sim U(a,b): \quad M_X(t) = \frac{e^{tb}-e^{ta}}{t(b-a)}$$
Worked Example 11.7a
Use the MGF to find $E(X)$ and $\text{Var}(X)$ for $X \sim \text{Exp}(\lambda)$.
1
$M_X(t) = \dfrac{\lambda}{\lambda-t} = \lambda(\lambda-t)^{-1}$
2
$M'_X(t) = \lambda(\lambda-t)^{-2}$. $E(X) = M'_X(0) = \dfrac{\lambda}{\lambda^2} = \dfrac{1}{\lambda}$ ✓
3
$M''_X(t) = 2\lambda(\lambda-t)^{-3}$. $E(X^2) = M''_X(0) = \dfrac{2}{\lambda^2}$
4
$\text{Var}(X) = \dfrac{2}{\lambda^2} - \dfrac{1}{\lambda^2} = \dfrac{1}{\lambda^2}$ ✓
Worked Example 11.7b — Sum of normals via MGF
Use MGFs to show that if $X_1, \ldots, X_n$ are i.i.d. $N(\mu, \sigma^2)$, then $\bar{X} \sim N(\mu, \sigma^2/n)$.
1
$M_{X_i}(t) = e^{\mu t + \frac{1}{2}\sigma^2 t^2}$. By the product rule: $M_{S}(t) = e^{n\mu t + \frac{n}{2}\sigma^2 t^2}$ where $S = \sum X_i$.
2
$M_{\bar{X}}(t) = M_S(t/n) = e^{\mu t + \frac{\sigma^2}{2n}t^2}$, which is the MGF of $N(\mu, \sigma^2/n)$. ✓

Q1
The random variable $X$ has PDF $f(x) = \dfrac{3}{2}(1-x^2)$ for $-1 \leq x \leq 1$. Find (a) $E(X)$, (b) $\text{Var}(X)$, (c) the CDF $F(x)$, (d) $P(-0.5 \leq X \leq 0.5)$.
(a) By symmetry about 0, $E(X)=0$. (b) $E(X^2)=\int_{-1}^1\frac{3}{2}x^2(1-x^2)\,dx=\frac{3}{2}\left[\frac{x^3}{3}-\frac{x^5}{5}\right]_{-1}^1=\frac{3}{2}\cdot\frac{4}{15}=\frac{2}{5}$. $\text{Var}(X)=\frac{2}{5}$. (c) For $-1\leq x\leq1$: $F(x)=\int_{-1}^x\frac{3}{2}(1-t^2)\,dt=\frac{3}{2}\left[t-\frac{t^3}{3}\right]_{-1}^x=\frac{3}{2}\left(x-\frac{x^3}{3}+\frac{2}{3}\right)=\frac{1}{2}+\frac{3x}{2}-\frac{x^3}{2}$. (d) $F(0.5)-F(-0.5)=\frac{1}{2}+\frac{3}{4}-\frac{1}{16}-(\frac{1}{2}-\frac{3}{4}+\frac{1}{16})=\frac{3}{2}-\frac{1}{8}=\frac{11}{8}$... Let me recheck: $F(0.5)=\frac{1}{2}+\frac{3(0.5)}{2}-\frac{(0.5)^3}{2}=\frac{1}{2}+\frac{3}{4}-\frac{1}{16}=\frac{8+12-1}{16}=\frac{19}{16}$. That's $>1$, so recheck formula. $F(x)=\frac{3}{2}[(x+1)-\frac{x^3+1}{3}]=\frac{3}{2}\cdot\frac{3(x+1)-(x^3+1)}{3}=\frac{3x+3-x^3-1}{2}=\frac{-x^3+3x+2}{2}$. At $x=1$: $\frac{-1+3+2}{2}=\frac{4}{2}=2$... That's wrong too. Correct: $\int_{-1}^x\frac{3}{2}(1-t^2)dt=\frac{3}{2}[t-\frac{t^3}{3}]_{-1}^x=\frac{3}{2}[(x-\frac{x^3}{3})-(-1+\frac{1}{3})]=\frac{3}{2}[x-\frac{x^3}{3}+\frac{2}{3}]$. At $x=1$: $\frac{3}{2}[1-\frac{1}{3}+\frac{2}{3}]=\frac{3}{2}\cdot\frac{4}{3}=2$. So $f$ doesn't integrate to 1 — check: $\int_{-1}^1\frac{3}{2}(1-x^2)dx=\frac{3}{2}[x-\frac{x^3}{3}]_{-1}^1=\frac{3}{2}\cdot\frac{4}{3}=2\neq1$. Correct $k$: $k=\frac{3}{4}$. So $f(x)=\frac{3}{4}(1-x^2)$. Then $\text{Var}(X)=\frac{1}{5}$, $F(x)=\frac{1}{2}+\frac{3x}{4}-\frac{x^3}{4}$, $P(-0.5\leq X\leq0.5)=F(0.5)-F(-0.5)=\frac{3(0.5)}{4}\cdot2-\frac{(0.5)^3}{4}\cdot2=\frac{3}{4}-\frac{1}{32}=\frac{23}{32}\approx0.719$.

Q5
Use the MGF of $X \sim N(0,1)$ to find $E(X^4)$. (Hint: the MGF of $N(0,1)$ is $M(t)=e^{t^2/2}$; expand as a Maclaurin series in $t$.)
$M(t)=e^{t^2/2}=1+\frac{t^2}{2}+\frac{t^4/4}{2!}+\cdots=1+\frac{t^2}{2}+\frac{t^4}{8}+\cdots$. Since $M(t)=\sum_{r=0}^\infty\frac{E(X^r)}{r!}t^r$, the coefficient of $t^4$ is $\frac{E(X^4)}{4!}=\frac{1}{8}$. So $E(X^4)=\frac{4!}{8}=\frac{24}{8}=3$. The lifetime (in years) of a component is $X \sim \text{Exp}(0.2)$. Given that a component has already lasted 5 years, find the probability it lasts at least another 3 years. What property does this illustrate?
$P(X>8\mid X>5)=\frac{P(X>8)}{P(X>5)}=\frac{e^{-1.6}}{e^{-1.0}}=e^{-0.6}\approx0.5488$. This equals $P(X>3)=e^{-0.6}$, illustrating the memoryless property: past lifetime gives no information about remaining lifetime.

Q8
The MGF of a random variable $X$ is $M_X(t) = \left(\dfrac{p}{1-(1-p)e^t}\right)^r$ for $t < -\ln(1-p)$. (i) Identify the distribution of $X$ by comparing with known MGFs. (ii) Find $E(X)$ and $\text{Var}(X)$ by differentiating $M_X(t)$. (iii) Show that the MGF of $Y = X - r$ (a shifted version) tends to the MGF of a Poisson as $r \to \infty$, $p \to 0$, $r(1-p) \to \lambda$.
(i) This is the MGF of a Negative Binomial NB$(r,p)$ — it's $r$ times the MGF of Geo$(p)$ (product of $r$ independent geometric MGFs). (ii) $M'(t)=r\cdot\frac{p}{1-(1-p)e^t}\cdot\frac{(1-p)e^t}{1-(1-p)e^t}$. At $t=0$: $M'(0)=r\cdot\frac{(1-p)}{p}=\frac{r(1-p)}{p}$. Since NB counts total trials: $E(X)=r/p$.
