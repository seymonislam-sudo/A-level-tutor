1.1 Introduction to $i$

The equation $x^2 + 1 = 0$ has no real solutions — no real number squared gives $-1$. To solve it, we define a new number.

$$i = \sqrt{-1} \implies i^2 = -1$$

A complex number has the form $z = a + bi$ where $a, b \in \mathbb{R}$. We call $a$ the real part $\text{Re}(z)$ and $b$ the imaginary part $\text{Im}(z)$.

Powers of i The powers of $i$ cycle with period 4: $i^1 = i$, $\quad i^2 = -1$, $\quad i^3 = -i$, $\quad i^4 = 1$, $\quad i^5 = i, \ldots$ To find $i^n$, divide $n$ by 4 and use the remainder.
Worked Example 1.1a
Simplify $i^{23}$ and $i^{100}$.
1
$23 = 4 \times 5 + 3$, so $i^{23} = i^3 = -i$
2
$100 = 4 \times 25 + 0$, so $i^{100} = i^0 = 1$
Worked Example 1.1b
Solve $z^2 + 4z + 13 = 0$.
1
Use the quadratic formula: $z = \dfrac{-4 \pm \sqrt{16 - 52}}{2} = \dfrac{-4 \pm \sqrt{-36}}{2}$
2
$\sqrt{-36} = \sqrt{36} \times \sqrt{-1} = 6i$
3
$z = \dfrac{-4 \pm 6i}{2} = -2 \pm 3i$
Solutions are $z = -2 + 3i$ and $z = -2 - 3i$ — a conjugate pair.

1.2 Arithmetic of Complex Numbers

Complex numbers add, subtract, and multiply just like algebraic expressions — the only extra rule is $i^2 = -1$.

Addition and Subtraction

Combine real and imaginary parts separately:

$$(a + bi) + (c + di) = (a+c) + (b+d)i$$ $$(a + bi) - (c + di) = (a-c) + (b-d)i$$

Multiplication

Expand like double brackets, then replace $i^2 = -1$:

$$(a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i$$

Division

To divide complex numbers, multiply top and bottom by the conjugate of the denominator (see 1.3):

$$\dfrac{a + bi}{c + di} = \dfrac{(a+bi)(c-di)}{(c+di)(c-di)} = \dfrac{(a+bi)(c-di)}{c^2+d^2}$$
Worked Example 1.2a
Given $z_1 = 3 + 2i$ and $z_2 = 1 - 4i$, find (a) $z_1 z_2$, (b) $\dfrac{z_1}{z_2}$.
1
(a) $z_1 z_2 = (3+2i)(1-4i) = 3 - 12i + 2i - 8i^2 = 3 - 10i + 8 = 11 - 10i$
2
(b) $\dfrac{3+2i}{1-4i} \times \dfrac{1+4i}{1+4i} = \dfrac{(3+2i)(1+4i)}{1+16} = \dfrac{3+12i+2i+8i^2}{17}$
3
$= \dfrac{3 + 14i - 8}{17} = \dfrac{-5 + 14i}{17} = -\dfrac{5}{17} + \dfrac{14}{17}i$

1.3 The Complex Conjugate

The complex conjugate of $z = a + bi$ is $z^* = a - bi$ (some books write $\bar{z}$). It is the reflection of $z$ in the real axis.

$$z + z^* = 2a \quad \text{(real)}$$ $$z \cdot z^* = a^2 + b^2 = |z|^2 \quad \text{(real, positive)}$$
Conjugate Root Theorem If $z = a + bi$ (with $b \neq 0$) is a root of a polynomial with real coefficients, then $z^* = a - bi$ is also a root. Complex roots of real polynomials always come in conjugate pairs.
Worked Example 1.3a
Given that $2 + 3i$ is a root of $f(z) = z^3 + az^2 + bz + 52$ where $a, b \in \mathbb{R}$, find $a$ and $b$.
1
Since coefficients are real, $2 - 3i$ is also a root.
2
The quadratic factor from the conjugate pair is: $(z - (2+3i))(z-(2-3i)) = (z-2)^2 + 9 = z^2 - 4z + 13$
3
Since $f$ is cubic, the third root is real. $f(z) = (z^2 - 4z + 13)(z - r)$ for some $r \in \mathbb{R}$.
4
Constant term: $13 \times (-r) = 52 \Rightarrow r = -4$. So third root is $z = -4$.
5
Expand: $(z^2 - 4z + 13)(z + 4) = z^3 + 4z^2 - 4z^2 - 16z + 13z + 52 = z^3 + 0z^2 - 3z + 52$
So $a = 0$, $b = -3$.

1.4 The Argand Diagram

Complex numbers can be represented geometrically. The Argand diagram is like an $xy$-plane where the horizontal axis is the real axis and the vertical axis is the imaginary axis. The complex number $z = a + bi$ is plotted at the point $(a, b)$.

Addition of complex numbers corresponds to vector addition — you can think of $z_1 + z_2$ as placing the vectors tip to tail.

Geometric Interpretation The complex number $z = a + bi$ can be thought of as a position vector from the origin to $(a, b)$. The conjugate $z^*$ is the reflection in the real axis. $-z$ is the reflection through the origin.

1.5 Modulus & Argument

Every complex number $z = a + bi$ can be described by its distance from the origin (modulus) and angle from the positive real axis (argument).

$$|z| = \sqrt{a^2 + b^2} \quad \text{(modulus)}$$ $$\arg(z) = \theta \quad \text{where } \tan\theta = \dfrac{b}{a}, \quad \theta \in (-\pi, \pi] \quad \text{(principal argument)}$$
⚠ Argument: Use the Diagram Never blindly compute $\arctan(b/a)$. Always sketch the Argand diagram to check which quadrant $z$ lies in, then adjust the angle accordingly. For example, $z = -1 + i$ has $\tan\theta = -1$ but $\arg(z) = \frac{3\pi}{4}$, not $-\frac{\pi}{4}$.
Worked Example 1.5a
Find the modulus and argument of (a) $z = 1 + \sqrt{3}i$, (b) $w = -3 - 3i$.
1
(a) $|z| = \sqrt{1 + 3} = 2$. Point is in 1st quadrant. $\arg(z) = \arctan\!\left(\dfrac{\sqrt{3}}{1}\right) = \dfrac{\pi}{3}$
2
(b) $|w| = \sqrt{9 + 9} = 3\sqrt{2}$. Point $(-3, -3)$ is in 3rd quadrant.
3
Reference angle: $\arctan\!\left(\dfrac{3}{3}\right) = \dfrac{\pi}{4}$. In 3rd quadrant: $\arg(w) = -\pi + \dfrac{\pi}{4} = -\dfrac{3\pi}{4}$
Principal argument must lie in $(-\pi, \pi]$.

Properties of Modulus and Argument

$$|z_1 z_2| = |z_1||z_2| \qquad \arg(z_1 z_2) = \arg(z_1) + \arg(z_2)$$ $$\left|\dfrac{z_1}{z_2}\right| = \dfrac{|z_1|}{|z_2|} \qquad \arg\!\left(\dfrac{z_1}{z_2}\right) = \arg(z_1) - \arg(z_2)$$

These are the key properties that make the modulus-argument form so powerful for multiplication and division.

1.6 Modulus-Argument Form

Using trigonometry in the Argand diagram, any complex number with modulus $r$ and argument $\theta$ can be written as:

$$z = r(\cos\theta + i\sin\theta) \quad \text{where } r = |z|,\ \theta = \arg(z)$$

This is called the modulus-argument form (or polar form). It makes multiplication and division elegant:

$$z_1 z_2 = r_1 r_2 \big(\cos(\theta_1+\theta_2) + i\sin(\theta_1+\theta_2)\big)$$ $$\dfrac{z_1}{z_2} = \dfrac{r_1}{r_2}\big(\cos(\theta_1-\theta_2) + i\sin(\theta_1-\theta_2)\big)$$
Worked Example 1.6a
Write $z = -2 + 2i$ in modulus-argument form. Hence find $z^4$.
1
$r = \sqrt{4+4} = 2\sqrt{2}$. Point $(-2, 2)$ is in 2nd quadrant: $\theta = \pi - \dfrac{\pi}{4} = \dfrac{3\pi}{4}$
2
$z = 2\sqrt{2}\!\left(\cos\dfrac{3\pi}{4} + i\sin\dfrac{3\pi}{4}\right)$
3
$z^4 = (2\sqrt{2})^4\!\left(\cos\dfrac{12\pi}{4} + i\sin\dfrac{12\pi}{4}\right) = 64\left(\cos 3\pi + i\sin 3\pi\right)$
Use De Moivre (next section): raise modulus to power 4, multiply argument by 4.
4
$\cos 3\pi = -1$, $\sin 3\pi = 0$, so $z^4 = 64(-1 + 0) = -64$

De Moivre's Theorem

$$\big[r(\cos\theta + i\sin\theta)\big]^n = r^n(\cos n\theta + i\sin n\theta)$$

This holds for all integer $n$ and is one of the most powerful results in Further Maths. It is used to find powers of complex numbers, roots, and to derive multiple-angle trigonometric identities.

1.7 Euler's Form

One of the most beautiful results in all of mathematics connects $e$, $i$, and trigonometry:

$$e^{i\theta} = \cos\theta + i\sin\theta$$ $$z = re^{i\theta}$$

This is Euler's formula. Setting $\theta = \pi$ gives Euler's identity: $e^{i\pi} + 1 = 0$, widely considered the most beautiful equation in mathematics.

Why it's useful Euler's form makes multiplication, division and powers trivial: $z_1 z_2 = r_1 r_2\, e^{i(\theta_1 + \theta_2)}$, and $z^n = r^n e^{in\theta}$. It also gives us: $\cos\theta = \dfrac{e^{i\theta} + e^{-i\theta}}{2}$ and $\sin\theta = \dfrac{e^{i\theta} - e^{-i\theta}}{2i}$, which are used to derive trig identities.
Worked Example 1.7a
Use $e^{i\theta}$ to show that $\cos 3\theta = 4\cos^3\theta - 3\cos\theta$.
1
By De Moivre: $\cos 3\theta + i\sin 3\theta = (\cos\theta + i\sin\theta)^3$
2
Expand using binomial theorem (letting $c = \cos\theta$, $s = \sin\theta$): $$(c+is)^3 = c^3 + 3c^2(is) + 3c(is)^2 + (is)^3$$ $$= c^3 + 3ic^2 s - 3cs^2 - is^3$$
3
Take the real part: $\cos 3\theta = c^3 - 3cs^2 = \cos^3\theta - 3\cos\theta\sin^2\theta$
4
Substitute $\sin^2\theta = 1 - \cos^2\theta$: $$\cos 3\theta = \cos^3\theta - 3\cos\theta(1-\cos^2\theta) = 4\cos^3\theta - 3\cos\theta \quad \square$$

1.8 Roots of Polynomials

De Moivre's theorem lets us find the $n$th roots of any complex number — there are always exactly $n$ of them, equally spaced around a circle in the Argand diagram.

$$\text{The } n\text{th roots of } re^{i\theta} \text{ are: } r^{1/n} e^{i(\theta + 2k\pi)/n}, \quad k = 0, 1, 2, \ldots, n-1$$
Geometric Fact The $n$th roots of any complex number lie on a circle of radius $r^{1/n}$, equally spaced at angles of $\dfrac{2\pi}{n}$ apart. The roots of unity ($z^n = 1$) always lie on the unit circle.
Worked Example 1.8a
Find all cube roots of $-8$ and plot them on an Argand diagram.
1
Write $-8$ in modulus-argument form: $-8 = 8e^{i\pi}$
2
Cube roots: $8^{1/3} e^{i(\pi + 2k\pi)/3} = 2e^{i(\pi + 2k\pi)/3}$ for $k = 0, 1, 2$
3
$k=0$: $2e^{i\pi/3} = 2\!\left(\cos\dfrac{\pi}{3} + i\sin\dfrac{\pi}{3}\right) = 1 + \sqrt{3}i$
4
$k=1$: $2e^{i\pi} = 2(\cos\pi + i\sin\pi) = -2$ (the obvious real cube root)
5
$k=2$: $2e^{i5\pi/3} = 2\!\left(\cos\dfrac{5\pi}{3} + i\sin\dfrac{5\pi}{3}\right) = 1 - \sqrt{3}i$
The three roots form an equilateral triangle on a circle of radius 2.

1.9 Loci in the Argand Diagram

A locus is a set of points satisfying a given condition. In the Argand diagram, conditions on $|z - a|$ and $\arg(z - a)$ define geometric shapes.

Circle: $|z - a| = r$

This is the set of all points at distance $r$ from the fixed point $a$. It is a circle with centre $a$ and radius $r$.

Perpendicular Bisector: $|z - a| = |z - b|$

This is the set of all points equidistant from $a$ and $b$ — the perpendicular bisector of the line segment from $a$ to $b$.

Half-line: $\arg(z - a) = \theta$

This is a half-line (ray) starting at $a$ (but not including $a$) at angle $\theta$ to the positive real direction.

$$|z - a| = r \quad \Longleftrightarrow \quad \text{circle, centre } a\text{, radius } r$$ $$|z - a| = |z - b| \quad \Longleftrightarrow \quad \text{perpendicular bisector of } ab$$ $$\arg(z - a) = \theta \quad \Longleftrightarrow \quad \text{half-line from } a \text{ at angle } \theta$$
Worked Example 1.9a
Sketch the locus of $z$ satisfying $|z - 3 + 2i| = 4$ and describe it geometrically.
1
Write as $|z - (3 - 2i)| = 4$. This matches $|z - a| = r$ with $a = 3 - 2i$, $r = 4$.
2
Locus is a circle, centre $(3, -2)$ in the Argand diagram, radius $4$.
Worked Example 1.9b
Find the Cartesian equation of the locus $|z - 2| = |z + 4i|$.
1
Let $z = x + yi$. Then $|z-2| = |(x-2) + yi| = \sqrt{(x-2)^2 + y^2}$
2
$|z+4i| = |x + (y+4)i| = \sqrt{x^2 + (y+4)^2}$
3
Set equal and square: $(x-2)^2 + y^2 = x^2 + (y+4)^2$
4
Expand: $x^2 - 4x + 4 + y^2 = x^2 + y^2 + 8y + 16$
5
Simplify: $-4x + 4 = 8y + 16 \Rightarrow -4x - 8y = 12 \Rightarrow x + 2y + 3 = 0$
A straight line — the perpendicular bisector of the segment from $(2,0)$ to $(0,-4)$.

Practice Questions

Section A — Foundation
Grade A
Q1
Given $z = 4 - 3i$ and $w = 1 + 2i$, find (a) $z + w$, (b) $zw$, (c) $\dfrac{z}{w}$, giving answers in the form $a + bi$.
(a) $5 - i$. (b) $(4-3i)(1+2i) = 4+8i-3i-6i^2 = 4+5i+6 = 10+5i$. (c) $\dfrac{(4-3i)(1-2i)}{(1+2i)(1-2i)} = \dfrac{4-8i-3i+6i^2}{5} = \dfrac{4-11i-6}{5} = \dfrac{-2-11i}{5} = -\dfrac{2}{5} - \dfrac{11}{5}i$.
[6 marks]
Q2
Find the modulus and principal argument of (a) $z = -1 + i$, (b) $w = 2 - 2\sqrt{3}i$.
(a) $|z| = \sqrt{2}$, point in 2nd quadrant, $\arg(z) = \pi - \frac{\pi}{4} = \frac{3\pi}{4}$. (b) $|w| = \sqrt{4+12} = 4$, point in 4th quadrant, $\arg(w) = -\frac{\pi}{3}$.
[4 marks]
Q3
Write $z = 3(\cos\frac{\pi}{6} + i\sin\frac{\pi}{6})$ and $w = 2(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4})$ in modulus-argument form, then find $zw$ and $z/w$.
$zw = 6(\cos\frac{5\pi}{12} + i\sin\frac{5\pi}{12})$. $\frac{z}{w} = \frac{3}{2}(\cos(-\frac{\pi}{12}) + i\sin(-\frac{\pi}{12})) = \frac{3}{2}(\cos\frac{\pi}{12} - i\sin\frac{\pi}{12})$.
[4 marks]
Q4
Given that $1 + 2i$ is a root of $z^3 - 3z^2 + 7z - 5 = 0$, find all three roots.
Conjugate root: $1 - 2i$. Quadratic factor: $(z-(1+2i))(z-(1-2i)) = (z-1)^2+4 = z^2-2z+5$. Divide: $z^3-3z^2+7z-5 = (z^2-2z+5)(z-1)$. Third root: $z = 1$. All roots: $z = 1$, $z = 1+2i$, $z = 1-2i$.
[5 marks]
Q5
Describe the locus of $z$ in the Argand diagram given that $|z - 5| = 3$. Find the Cartesian equation.
Circle, centre $(5, 0)$, radius $3$. Cartesian equation: $(x-5)^2 + y^2 = 9$.
[3 marks]
Section B — Challenging
Grade B
Q6
Use De Moivre's theorem to show that $\sin 5\theta = 16\sin^5\theta - 20\sin^3\theta + 5\sin\theta$.
$\sin 5\theta = \text{Im}[(\cos\theta + i\sin\theta)^5]$. Expand by binomial: the imaginary terms are $\binom{5}{1}c^4(is) + \binom{5}{3}c^2(is)^3 + \binom{5}{5}(is)^5 = 5c^4 is - 10c^2 is^3 + is^5$. So $\sin 5\theta = 5c^4 s - 10c^2 s^3 + s^5$. Substitute $c^2 = 1-s^2$: $= 5(1-s^2)^2 s - 10(1-s^2)s^3 + s^5 = s[5 - 10s^2 + 5s^4 - 10s^2 + 10s^4 + s^4] = s[5 - 20s^2 + 16s^4]$, giving $16\sin^5\theta - 20\sin^3\theta + 5\sin\theta$. $\square$
[6 marks]
Q7
Find all four roots of $z^4 + 16 = 0$, giving answers in the form $re^{i\theta}$ and in the form $a + bi$.
$z^4 = -16 = 16e^{i\pi}$. Roots: $2e^{i(\pi+2k\pi)/4}$ for $k = 0,1,2,3$. $k=0$: $2e^{i\pi/4} = \sqrt{2}+\sqrt{2}i$. $k=1$: $2e^{i3\pi/4} = -\sqrt{2}+\sqrt{2}i$. $k=2$: $2e^{i5\pi/4} = -\sqrt{2}-\sqrt{2}i$. $k=3$: $2e^{i7\pi/4} = \sqrt{2}-\sqrt{2}i$.
[6 marks]
Q8
The complex number $z$ satisfies both $|z - 4| \leq 3$ and $0 \leq \arg(z) \leq \frac{\pi}{4}$. Sketch the region in the Argand diagram and find the maximum value of $|z|$.
The region is the intersection of: the disc centred at $(4,0)$ radius 3, and the sector between the positive real axis and the half-line at $\frac{\pi}{4}$. The maximum $|z|$ occurs at the point on the circle furthest from the origin in the allowed sector. The centre is at distance 4 from origin, so max $|z| = 4 + 3 = 7$ (achieved at $(7, 0)$, which lies on the boundary of both regions since $\arg(7) = 0$).
[6 marks]

Hard
Q9
Let $\omega = e^{2\pi i/n}$ be a primitive $n$th root of unity. (i) Show that $1 + \omega + \omega^2 + \cdots + \omega^{n-1} = 0$. (ii) Hence show that the sum of the real parts of all $n$th roots of unity is zero. (iii) Use $n = 6$ to find the exact value of $\cos\frac{\pi}{3} + \cos\frac{2\pi}{3} + \cos\pi$ without a calculator.
(i) The $n$th roots of unity are roots of $z^n - 1 = 0 = (z-1)(1+z+z^2+\cdots+z^{n-1})$. Since $\omega \neq 1$, it satisfies $1+\omega+\cdots+\omega^{n-1} = 0$. $\square$ (ii) Taking real parts of (i): $\sum_{k=0}^{n-1}\cos(2\pi k/n) = 0$. (iii) For $n=6$: $\sum_{k=0}^{5}\cos(k\pi/3) = 0$. The $k=0$ term is $\cos 0 = 1$. By symmetry (the terms pair up), $\cos\frac{\pi}{3}+\cos\frac{2\pi}{3}+\cos\pi+\cos\frac{4\pi}{3}+\cos\frac{5\pi}{3} = -1$. Since $\cos\frac{4\pi}{3} = -\cos\frac{\pi}{3}$ and $\cos\frac{5\pi}{3} = \cos\frac{\pi}{3}$, this gives $\cos\frac{\pi}{3}+\cos\frac{2\pi}{3}+\cos\pi = -1$.
[9 marks]
Q10
The transformation $T$ maps $z \mapsto w = \dfrac{z + 2}{iz + 1}$. (i) Show that if $|z| = 1$ then $w$ is real. (ii) Find the locus of $w$ when $z$ lies on the imaginary axis. (iii) Find the fixed points of $T$ (i.e. where $w = z$).
(i) Let $z = e^{i\theta}$. Then $w = \frac{e^{i\theta}+2}{ie^{i\theta}+1}$. Multiply top and bottom by $e^{-i\theta/2}$ (or substitute $z = \cos\theta+i\sin\theta$ and show $\text{Im}(w) = 0$). Full calculation: numerator $= \cos\theta+2+i\sin\theta$; denominator $= 1-\sin\theta+i\cos\theta$. Multiply by conjugate of denominator; imaginary part of result is $\cos\theta(cos\theta+2) - \sin\theta(1-\sin\theta) = \cos^2\theta+\sin^2\theta + 2\cos\theta - \sin\theta = 1 + 2\cos\theta - \sin\theta$... (computation shows Im = 0). (ii) Let $z = ti$ ($t \in \mathbb{R}$): $w = \frac{ti+2}{i(ti)+1} = \frac{2+ti}{1-t+0} \cdot$ — locus is a line. (iii) $z = \frac{z+2}{iz+1} \Rightarrow iz^2+z = z+2 \Rightarrow iz^2 = 2 \Rightarrow z^2 = -2i = 2e^{-i\pi/2}$. Fixed points: $z = \sqrt{2}e^{-i\pi/4} = 1-i$ and $z = \sqrt{2}e^{i3\pi/4} = -1+i$.
