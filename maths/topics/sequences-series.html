Sequences & Series

Patterns in numbers — and the art of summing them. Arithmetic and geometric sequences appear everywhere from finance to physics, and the convergence of infinite series is one of the most beautiful ideas in mathematics.

10.1 Sequences

A sequence is an ordered list of numbers, each called a term. Sequences can be defined by a formula for the $n$th term, or by a recurrence relation that expresses each term in terms of previous ones.

$$\text{$n$th term (position-to-term): } u_n = f(n)$$ $$\text{Recurrence relation (term-to-term): } u_{n+1} = g(u_n)$$ $$\text{e.g. } u_{n+1} = u_n + d \text{ (arithmetic)} \qquad u_{n+1} = ru_n \text{ (geometric)}$$
Worked Example 10.1a
A sequence is defined by $u_1 = 5$, $u_{n+1} = 3u_n - 4$. Find the first four terms and determine whether the sequence is increasing, decreasing, or periodic.
1
$u_1 = 5$ $u_2 = 3(5) - 4 = 11$ $u_3 = 3(11) - 4 = 29$ $u_4 = 3(29) - 4 = 83$
2
Each term is larger than the previous, so the sequence is increasing (and diverges to infinity).
A sequence is periodic if it repeats after a fixed number of terms. This one does not.
Worked Example 10.1b — Periodic sequence
A sequence is defined by $u_1 = 2$, $u_{n+1} = \dfrac{6}{u_n + 1}$. Show that the sequence is periodic and state its period.
1
$u_1 = 2$, $\quad u_2 = \dfrac{6}{3} = 2$, $\quad u_3 = \dfrac{6}{3} = 2$, $\ldots$
Every term equals 2, so the sequence has period 1. This is a fixed point of the recurrence.
Worked Example 10.1c — Finding $n$th term
A sequence has $n$th term $u_n = 3n^2 - n + 2$. Find $(a)$ $u_5$   $(b)$ the first term greater than 100.
1
$(a)$: $u_5 = 3(25) - 5 + 2 = 75 - 5 + 2 = 72$.
2
$(b)$: Solve $3n^2 - n + 2 > 100 \implies 3n^2 - n - 98 > 0$. Using the quadratic formula: $n = \dfrac{1 \pm \sqrt{1 + 1176}}{6} = \dfrac{1 \pm \sqrt{1177}}{6} \approx \dfrac{1 \pm 34.3}{6}$. Positive root: $n \approx 5.89$. So the first term greater than 100 is $u_6$.
3
Verify: $u_6 = 3(36) - 6 + 2 = 108 - 4 = 104 > 100$ ✓. $u_5 = 72 < 100$ ✓.

10.2 Arithmetic Sequences

In an arithmetic sequence, consecutive terms differ by a constant amount called the common difference $d$. These sequences increase (or decrease) steadily.

$$u_n = a + (n-1)d$$ $$\text{where } a = u_1 \text{ (first term)}, \quad d = u_{n+1} - u_n \text{ (common difference)}$$ $$d > 0: \text{ increasing} \qquad d < 0: \text{ decreasing} \qquad d = 0: \text{ constant}$$
Worked Example 10.2a
An arithmetic sequence has first term $7$ and common difference $-3$. $(a)$ Find the 20th term. $(b)$ Which term is the first negative term?
1
$(a)$: $u_{20} = 7 + 19(-3) = 7 - 57 = -50$.
2
$(b)$: Need $u_n < 0$: $7 + (n-1)(-3) < 0 \implies 7 - 3n + 3 < 0 \implies 10 < 3n \implies n > 3.33$. So the first negative term is $u_4 = 7 + 3(-3) = -2$.
Worked Example 10.2b — Finding $a$ and $d$
The 5th term of an arithmetic sequence is 23 and the 12th term is 58. Find the first term and common difference.
1
$u_5 = a + 4d = 23 \quad\cdots(1)$ $u_{12} = a + 11d = 58 \quad\cdots(2)$
2
$(2) - (1)$: $7d = 35 \implies d = 5$.
3
From $(1)$: $a = 23 - 20 = 3$. So $a = 3$, $d = 5$.
Check: $u_{12} = 3 + 55 = 58$ ✓

10.3 Arithmetic Series

A series is the sum of the terms of a sequence. The sum of the first $n$ terms of an arithmetic sequence is denoted $S_n$. The formula has two equivalent forms — use whichever fits what you know.

$$S_n = \frac{n}{2}(2a + (n-1)d)$$ $$S_n = \frac{n}{2}(a + l) \quad \text{where } l = u_n \text{ is the last term}$$ $$\text{Sigma notation: } \sum_{r=1}^{n} u_r = S_n$$

The second formula is easy to remember: it's $n$ times the average of the first and last terms. Gauss famously used this idea as a child to sum $1 + 2 + \cdots + 100 = \frac{100}{2}(1 + 100) = 5050$ in seconds.

Worked Example 10.3a
An arithmetic series has first term $4$, last term $100$, and common difference $4$. Find the number of terms and the sum.
1
Number of terms: $u_n = 4 + (n-1)(4) = 100 \implies 4n = 100 \implies n = 25$.
2
$S_{25} = \dfrac{25}{2}(4 + 100) = \dfrac{25 \times 104}{2} = 25 \times 52 = 1300$.
Worked Example 10.3b — Sum to $n$ terms
Find the least value of $n$ for which $S_n > 2000$, where $S_n$ is the sum of the arithmetic series $5 + 9 + 13 + \cdots$
1
$a = 5$, $d = 4$. $S_n = \dfrac{n}{2}(10 + 4(n-1)) = \dfrac{n}{2}(4n + 6) = n(2n + 3) = 2n^2 + 3n$.
2
Solve $2n^2 + 3n > 2000$: using the quadratic formula, $n > \dfrac{-3 + \sqrt{9 + 16000}}{4} = \dfrac{-3 + \sqrt{16009}}{4} \approx \dfrac{-3 + 126.5}{4} \approx 30.9$.
3
So the least value is $n = 31$. Check: $S_{31} = 2(961) + 93 = 1922 + 93 = 2015 > 2000$ ✓. $S_{30} = 1800 + 90 = 1890 < 2000$ ✓.
Sigma Notation Tips $\displaystyle\sum_{r=1}^{n} c = cn$ (sum of a constant). $\displaystyle\sum_{r=1}^{n} r = \frac{n(n+1)}{2}$ (sum of first $n$ integers — this is an arithmetic series with $a = 1$, $d = 1$). When splitting a sigma, you can pull out constants: $\sum cr = c\sum r$.

10.4 Geometric Sequences

In a geometric sequence, consecutive terms are related by a constant common ratio $r$. Multiply each term by $r$ to get the next. These sequences grow (or decay) exponentially.

$$u_n = ar^{n-1}$$ $$\text{where } a = u_1 \text{ (first term)}, \quad r = \frac{u_{n+1}}{u_n} \text{ (common ratio)}$$ $$r > 1: \text{ growing} \quad 0 < r < 1: \text{ decaying} \quad r < 0: \text{ alternating}$$
Worked Example 10.4a
A geometric sequence has first term $3$ and common ratio $2$. $(a)$ Find the 8th term. $(b)$ Which is the first term exceeding $1000$?
1
$(a)$: $u_8 = 3 \times 2^7 = 3 \times 128 = 384$.
2
$(b)$: Need $3 \times 2^{n-1} > 1000 \implies 2^{n-1} > 333.3$. Take $\log$: $(n-1)\log 2 > \log 333.3 \implies n - 1 > \dfrac{\log 333.3}{\log 2} \approx 8.38$. So $n > 9.38$, meaning $n = 10$.
3
Check: $u_{10} = 3 \times 2^9 = 3 \times 512 = 1536 > 1000$ ✓. $u_9 = 3 \times 256 = 768 < 1000$ ✓.
Worked Example 10.4b — Finding $a$ and $r$
The 3rd term of a geometric sequence is $12$ and the 6th term is $96$. Find the first term and common ratio.
1
$u_3 = ar^2 = 12 \quad\cdots(1)$   $u_6 = ar^5 = 96 \quad\cdots(2)$
2
Divide $(2)$ by $(1)$: $r^3 = 8 \implies r = 2$.
3
From $(1)$: $4a = 12 \implies a = 3$.
⚠ Negative Common Ratios If $r < 0$, the sequence alternates between positive and negative terms. For example $r = -2$ gives $3, -6, 12, -24, \ldots$ — these still follow the formula $u_n = ar^{n-1}$ perfectly, but the sequence does not converge regardless of $|r|$.

10.5 Geometric Series

The sum of the first $n$ terms of a geometric sequence. The formula is derived by writing $S_n$, multiplying by $r$, and subtracting — a beautiful algebraic trick worth understanding, not just memorising.

$$S_n = \frac{a(1 - r^n)}{1 - r} \quad (r \neq 1)$$ $$\text{Equivalently: } S_n = \frac{a(r^n - 1)}{r - 1} \quad \text{(useful when } r > 1\text{)}$$ $$\text{If } r = 1: S_n = na$$
Worked Example 10.5a
Find the sum of the first 10 terms of the geometric series $2 + 6 + 18 + 54 + \cdots$
1
$a = 2$, $r = 3$ (since each term is multiplied by 3). $n = 10$.
2
$S_{10} = \dfrac{2(3^{10} - 1)}{3 - 1} = \dfrac{2(59049 - 1)}{2} = 59048$.
Worked Example 10.5b — Finding $n$
A geometric series has first term $5$ and common ratio $\frac{3}{2}$. Find the smallest $n$ for which $S_n > 500$.
1
$S_n = \dfrac{5\!\left(\left(\frac{3}{2}\right)^n - 1\right)}{\frac{3}{2} - 1} = \dfrac{5\!\left(\left(\frac{3}{2}\right)^n - 1\right)}{\frac{1}{2}} = 10\!\left(\left(\tfrac{3}{2}\right)^n - 1\right)$.
2
Set $> 500$: $10\!\left(\left(\tfrac{3}{2}\right)^n - 1\right) > 500 \implies \left(\tfrac{3}{2}\right)^n > 51$.
3
Take logs: $n\ln\!\left(\tfrac{3}{2}\right) > \ln 51 \implies n > \dfrac{\ln 51}{\ln 1.5} \approx \dfrac{3.932}{0.405} \approx 9.71$. Smallest integer: $n = 10$.

10.6 Infinite Series & Convergence

If the common ratio satisfies $|r| < 1$, each successive term gets smaller and the infinite sum converges to a finite limit. This is one of the most surprising and beautiful results in mathematics.

$$S_\infty = \frac{a}{1-r} \quad \text{valid only when } |r| < 1$$ $$\text{Derivation: as } n \to \infty,\ r^n \to 0 \text{ (when } |r|<1\text{), so } S_n = \frac{a(1-r^n)}{1-r} \to \frac{a}{1-r}$$ $$|r| \geq 1 \implies \text{series diverges (no finite sum)}$$
Worked Example 10.6a
Find the sum to infinity of the geometric series $12 + 4 + \dfrac{4}{3} + \cdots$
1
$a = 12$, $r = \dfrac{4}{12} = \dfrac{1}{3}$. Since $|r| = \dfrac{1}{3} < 1$, the series converges.
2
$S_\infty = \dfrac{12}{1 - \frac{1}{3}} = \dfrac{12}{\frac{2}{3}} = 12 \times \dfrac{3}{2} = 18$.
Worked Example 10.6b — Finding unknowns from $S_\infty$
A geometric series has first term $a$ and common ratio $r$. Given that $S_\infty = 8$ and $S_2 = 6$, find $a$ and $r$.
1
$S_\infty = \dfrac{a}{1-r} = 8 \implies a = 8(1-r) \quad\cdots(1)$
2
$S_2 = a + ar = a(1+r) = 6 \quad\cdots(2)$
3
Substitute $(1)$ into $(2)$: $8(1-r)(1+r) = 6 \implies 8(1-r^2) = 6 \implies 1 - r^2 = \dfrac{3}{4} \implies r^2 = \dfrac{1}{4} \implies r = \pm\dfrac{1}{2}$.
4
Both values give valid series (both have $|r| < 1$). If $r = \dfrac{1}{2}$: $a = 8(\frac{1}{2}) = 4$. If $r = -\dfrac{1}{2}$: $a = 8(\frac{3}{2}) = 12$.
Both solutions should be stated unless additional context rules one out.
Worked Example 10.6c — Recurring decimals
Express $0.\overline{27} = 0.272727\ldots$ as an exact fraction using the sum to infinity.
1
Write as a geometric series: $0.27 + 0.0027 + 0.000027 + \cdots$ Here $a = 0.27 = \dfrac{27}{100}$ and $r = 0.01 = \dfrac{1}{100}$.
2
$S_\infty = \dfrac{27/100}{1 - 1/100} = \dfrac{27/100}{99/100} = \dfrac{27}{99} = \dfrac{3}{11}$.
Check: $3 \div 11 = 0.272727\ldots$ ✓
Convergence Condition The series converges if and only if $|r| < 1$. Equivalently: $-1 < r < 1$. If a question gives you $S_\infty$ and asks for the range of $r$, write $|r| < 1$. If it asks for the range of $a$, substitute $a = S_\infty(1-r)$ and use the constraint on $r$ to find the range of $a$.

Practice Questions

Section A — Foundation
Grade A
Q1
An arithmetic sequence has first term $8$ and common difference $5$. $(a)$ Find the 15th term. $(b)$ Find the sum of the first 20 terms.
$(a)$: $u_{15} = 8 + 14(5) = 78$. $(b)$: $S_{20} = \frac{20}{2}(2\times8 + 19\times5) = 10(16+95) = 10(111) = 1110$.
[4 marks]
Q2
A geometric sequence has first term $81$ and common ratio $\dfrac{1}{3}$. $(a)$ Find the 5th term. $(b)$ Find the sum to infinity.
$(a)$: $u_5 = 81 \times (1/3)^4 = 81/81 = 1$. $(b)$: $S_\infty = \frac{81}{1-1/3} = \frac{81}{2/3} = \frac{243}{2} = 121.5$.
[4 marks]
Q3
The 4th term of an arithmetic sequence is $17$ and the 9th term is $37$. Find the first term and common difference, then find the sum of the first 30 terms.
$a + 3d = 17$, $a + 8d = 37$. Subtract: $5d = 20 \Rightarrow d = 4$, $a = 5$. $S_{30} = \frac{30}{2}(10 + 29\times4) = 15(10+116) = 15 \times 126 = 1890$.
[6 marks]
Q4
Express $0.\overline{36}$ as an exact fraction.
$a = 0.36$, $r = 0.01$. $S_\infty = \frac{36/100}{99/100} = \frac{36}{99} = \frac{4}{11}$.
[3 marks]
Q5
Find the sum of all multiples of 7 between 1 and 200 (inclusive of any multiples of 7).
Multiples of 7: $7, 14, \ldots, 196$. $n = 196/7 = 28$ terms. $S_{28} = \frac{28}{2}(7 + 196) = 14 \times 203 = 2842$.
[4 marks]
Section B — Challenging
Grade B
Q6
The sum of the first $n$ terms of a series is $S_n = 3n^2 + 2n$. $(a)$ Find the first three terms of the series. $(b)$ Show the series is arithmetic and find the common difference.
$(a)$: $u_1 = S_1 = 5$. $u_2 = S_2 - S_1 = 16 - 5 = 11$. $u_3 = S_3 - S_2 = 33 - 16 = 17$. $(b)$: $u_n = S_n - S_{n-1} = 3n^2+2n - 3(n-1)^2 - 2(n-1) = 3(2n-1) + 2 = 6n - 1$ for $n \geq 2$. Check $u_1 = 5 = 6(1)-1$ ✓. Differences: $u_{n+1} - u_n = 6(n+1)-1 - (6n-1) = 6$ (constant). So arithmetic with $d = 6$.
[6 marks]
Q7
The second and fifth terms of a geometric series are $6$ and $\dfrac{6}{27}$ respectively. Find all possible values of the common ratio and first term, and the sum to infinity where it exists.
$ar = 6$ and $ar^4 = 6/27$. Divide: $r^3 = 1/27 \Rightarrow r = 1/3$. Then $a = 18$. $|r| = 1/3 < 1$ so $S_\infty = \frac{18}{1-1/3} = \frac{18}{2/3} = 27$.
[6 marks]
Q8
A sequence is defined by $u_1 = k$, $u_{n+1} = \dfrac{u_n + 3}{2}$. $(a)$ Find the fixed point of the recurrence (where $u_{n+1} = u_n$). $(b)$ Show that $u_n - 3 = \left(\dfrac{1}{2}\right)^{n-1}(k - 3)$. $(c)$ Hence state the value the sequence converges to.
$(a)$: Fixed point: $L = \frac{L+3}{2} \Rightarrow 2L = L + 3 \Rightarrow L = 3$. $(b)$: Let $v_n = u_n - 3$. Then $v_{n+1} = u_{n+1} - 3 = \frac{u_n+3}{2} - 3 = \frac{u_n - 3}{2} = \frac{v_n}{2}$. So $\{v_n\}$ is geometric with ratio $\frac{1}{2}$: $v_n = v_1 \cdot (1/2)^{n-1} = (k-3)(1/2)^{n-1}$. $(c)$: As $n \to \infty$, $(1/2)^{n-1} \to 0$, so $u_n \to 3$ for all $k$.
[8 marks]
Section C — STEP-style
Hard
Q9
$(a)$ Prove that the sum of the first $n$ terms of an arithmetic series with first term $a$ and common difference $d$ is $S_n = \dfrac{n}{2}(2a + (n-1)d)$, by considering $S_n$ written forwards and backwards. $(b)$ The integers from $1$ to $N$ are split into two groups: multiples of $3$, and non-multiples of $3$. Show that the ratio of the sum of the multiples of 3 to the sum of the non-multiples of 3 approaches $\dfrac{1}{2}$ as $N \to \infty$.
$(a)$: Write $S_n = a + (a+d) + \cdots + (a+(n-1)d)$ and reverse: $S_n = (a+(n-1)d) + \cdots + a$. Adding: $2S_n = n(2a+(n-1)d)$, so $S_n = \frac{n}{2}(2a+(n-1)d)$. $\square$ $(b)$: Assume $N = 3m$ for simplicity. Sum of multiples of 3 from 1 to $3m$: $3 + 6 + \cdots + 3m = 3(1+2+\cdots+m) = \frac{3m(m+1)}{2}$. Total sum $1$ to $3m$: $\frac{3m(3m+1)}{2}$. Sum of non-multiples $= \frac{3m(3m+1)}{2} - \frac{3m(m+1)}{2} = \frac{3m(3m+1-m-1)}{2} = \frac{3m(2m)}{2} = 3m^2$. Ratio $= \dfrac{3m(m+1)/2}{3m^2} = \dfrac{m+1}{2m} \to \dfrac{1}{2}$ as $m \to \infty$. $\square$
[10 marks]
Q10
$(a)$ By considering the geometric series $1 + x + x^2 + \cdots$, show that $\displaystyle\sum_{n=0}^{\infty} x^n = \dfrac{1}{1-x}$ for $|x| < 1$. $(b)$ Differentiate both sides with respect to $x$ to show that $\displaystyle\sum_{n=1}^{\infty} nx^{n-1} = \dfrac{1}{(1-x)^2}$. $(c)$ Hence evaluate $\displaystyle\sum_{n=1}^{\infty} \frac{n}{2^n}$.
$(a)$: Geometric series with $a=1$, ratio $x$. For $|x|<1$: $S_\infty = \frac{1}{1-x}$. $\square$ $(b)$: Differentiating $\sum_{n=0}^\infty x^n = \frac{1}{1-x}$ term by term (valid for $|x|<1$): $\sum_{n=1}^\infty nx^{n-1} = \frac{1}{(1-x)^2}$. $\square$ $(c)$: Multiply both sides by $x$: $\sum_{n=1}^\infty nx^n = \frac{x}{(1-x)^2}$. Set $x = \frac{1}{2}$: $\sum_{n=1}^\infty \frac{n}{2^n} = \frac{1/2}{(1/2)^2} = \frac{1/2}{1/4} = 2$.
[10 marks]
