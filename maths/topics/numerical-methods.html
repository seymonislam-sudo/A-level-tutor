Numerical Methods

When equations resist exact algebraic solution, numerical methods close in on roots with increasing precision. Iteration, Newton–Raphson, and sign-change analysis are the core tools.

11.1 Locating Roots

A root of $f(x) = 0$ is an $x$-value where the curve crosses the $x$-axis. The key tool for locating roots is the sign-change principle: if $f$ is continuous on $[a, b]$ and $f(a)$ and $f(b)$ have opposite signs, then there is at least one root in $(a, b)$.

$$f(a) \cdot f(b) < 0 \implies \exists\, \alpha \in (a,b) \text{ such that } f(\alpha) = 0$$ $$\text{(provided } f \text{ is continuous on } [a,b]\text{)}$$

This is a consequence of the Intermediate Value Theorem. At A Level, you apply it directly: compute $f(a)$ and $f(b)$, show opposite signs, and conclude there is a root in the interval.

Worked Example 11.1a
Show that $f(x) = x^3 - x - 4$ has a root in the interval $(1, 2)$.
1
$f(1) = 1 - 1 - 4 = -4 < 0$.
2
$f(2) = 8 - 2 - 4 = 2 > 0$.
3
Since $f(1) < 0$ and $f(2) > 0$, and $f$ is continuous (it's a polynomial), by the sign-change principle there is at least one root in the interval $(1, 2)$. $\square$
Always state: the function is continuous, the signs differ, therefore a root exists. All three components are needed for full marks.
Worked Example 11.1b — Locating to a decimal place
Show that $f(x) = e^x - 3x$ has a root between $1.5$ and $1.6$, and use further sign changes to locate the root correct to 2 decimal places.
1
$f(1.5) = e^{1.5} - 4.5 \approx 4.482 - 4.5 = -0.018 < 0$. $f(1.6) = e^{1.6} - 4.8 \approx 4.953 - 4.8 = 0.153 > 0$. Sign change → root in $(1.5, 1.6)$.
2
Test the midpoint: $f(1.51) = e^{1.51} - 4.53 \approx 4.527 - 4.53 = -0.003 < 0$. Root is in $(1.51, 1.6)$.
3
$f(1.52) = e^{1.52} - 4.56 \approx 4.572 - 4.56 = 0.012 > 0$. Sign change in $(1.51, 1.52)$ → root is $\alpha \approx 1.51$ to 2 d.p.
To state a root correct to 2 d.p., you need a sign change in an interval of width 0.01 centred on the rounded value.
⚠ Continuity is Essential The sign-change method only guarantees a root if the function is continuous on the interval. A function like $f(x) = \frac{1}{x}$ has $f(-1) = -1 < 0$ and $f(1) = 1 > 0$, but has no root in $(-1, 1)$ — it has a discontinuity at $x = 0$. Always check for discontinuities (asymptotes, undefined points) before applying this method.

11.2 Iteration

An iterative formula repeatedly applies a function to produce a sequence of approximations that (hopefully) converge to the root. The equation $f(x) = 0$ is rearranged into the form $x = g(x)$, giving the iterative scheme $x_{n+1} = g(x_n)$.

$$x_{n+1} = g(x_n)$$ $$\text{Starting from an initial estimate } x_0 \text{ (or } x_1\text{)}$$ $$\text{If } x_n \to \alpha, \text{ then } \alpha = g(\alpha) \implies f(\alpha) = 0$$

The same equation $f(x) = 0$ can be rearranged into many different forms $x = g(x)$ — some will converge, some will diverge. The convergence depends on $|g'(\alpha)|$ at the root.

Rearranging to Find an Iterative Formula

You are often asked to show that an equation can be written in a given iterative form, or to rearrange it yourself. There are usually multiple valid rearrangements.

Worked Example 11.2b
Show that $x^3 + 4x - 1 = 0$ can be written as $x = \dfrac{1 - x^3}{4}$. Hence write down an iterative formula and use $x_1 = 0$ to find the root correct to 3 d.p.
1
Rearrange: $4x = 1 - x^3 \implies x = \dfrac{1 - x^3}{4}$. $\square$
2
Iterative formula: $x_{n+1} = \dfrac{1 - x_n^3}{4}$.
3
$x_1 = 0$, $x_2 = 0.25$, $x_3 = \frac{1-0.015625}{4} = 0.246$, $x_4 = 0.2465$, $x_5 = 0.2465$. Root $\approx 0.247$ (3 d.p.).

11.3 Convergence & Failure

Not all iterative formulas converge. The key result is: the iteration $x_{n+1} = g(x_n)$ converges to root $\alpha$ if $|g'(\alpha)| < 1$. If $|g'(\alpha)| > 1$, it diverges. If $|g'(\alpha)| = 1$, the test is inconclusive.

$$|g'(\alpha)| < 1 \implies \text{convergence (the closer to 0, the faster)}$$ $$|g'(\alpha)| > 1 \implies \text{divergence}$$ $$g'(\alpha) > 0 \implies \text{monotone convergence (staircase diagram)}$$ $$g'(\alpha) < 0, |g'(\alpha)| < 1 \implies \text{oscillating convergence (cobweb diagram)}$$
Worked Example 11.3a — Checking convergence
The equation $x = 1 + \dfrac{2}{x^2}$ has a root $\alpha \approx 1.7$. The iterative formula $x_{n+1} = 1 + \dfrac{2}{x_n^2}$ is proposed. Determine whether this converges near $\alpha$.
1
$g(x) = 1 + 2x^{-2}$, so $g'(x) = -4x^{-3} = -\dfrac{4}{x^3}$.
2
At $\alpha \approx 1.7$: $g'(1.7) = -\dfrac{4}{4.913} \approx -0.814$.
3
$|g'(\alpha)| \approx 0.814 < 1$, so the iteration converges. Since $g'(\alpha) < 0$, the convergence is oscillating (cobweb diagram).
Worked Example 11.3b — Staircase vs cobweb
Describe the behaviour of the iteration $x_{n+1} = g(x_n)$ near a root $\alpha$ when $(a)$ $0 < g'(\alpha) < 1$ and $(b)$ $-1 < g'(\alpha) < 0$.
1
$(a)$ $0 < g'(\alpha) < 1$: the iterates approach $\alpha$ from one side, producing a staircase diagram — successive points step up (or down) monotonically towards $\alpha$.
2
$(b)$ $-1 < g'(\alpha) < 0$: the iterates alternate above and below $\alpha$, spiralling inward to produce a cobweb diagram.
Why Iteration Can Fail An iteration can fail for several reasons: $|g'(\alpha)| \geq 1$ (divergence); the starting value $x_0$ is too far from the root; the function $g$ has a discontinuity between $x_0$ and $\alpha$; or there are multiple roots and the iteration converges to the wrong one. In exams, showing a few diverging iterates is sufficient evidence of failure.

11.4 Newton–Raphson Method

The Newton–Raphson method is a powerful root-finding algorithm that uses both the function value and its derivative to make much faster progress than simple iteration. Starting near a root, each step draws the tangent and uses its $x$-intercept as the next estimate.

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$ $$\text{Requires: } f'(x_n) \neq 0 \text{ at each step}$$ $$\text{Converges quadratically near a simple root — doubles correct digits each step}$$

Newton–Raphson is derived geometrically: the tangent to $y = f(x)$ at $(x_n, f(x_n))$ has equation $y - f(x_n) = f'(x_n)(x - x_n)$. Setting $y = 0$ gives $x_{n+1} = x_n - \dfrac{f(x_n)}{f'(x_n)}$.

Worked Example 11.4b
Apply one step of Newton–Raphson to $f(x) = \ln x - 2 + x$ starting from $x_0 = 1.5$, giving your answer to 4 decimal places.
1
$f(x) = \ln x - 2 + x$, $\quad f'(x) = \dfrac{1}{x} + 1$.
2
$f(1.5) = \ln 1.5 - 2 + 1.5 = 0.4055 - 0.5 = -0.0945$. $f'(1.5) = \dfrac{1}{1.5} + 1 = 1.6\overline{6}$.
3
$x_1 = 1.5 - \dfrac{-0.0945}{1.6667} = 1.5 + 0.0567 = 1.5567$.
⚠ Newton–Raphson Failure Cases The method can fail if: $f'(x_n) = 0$ at some step (tangent is horizontal — division by zero); the starting point is near a local extremum and the tangent overshoots to a different root; the function has a discontinuity nearby. In these cases, try a different starting value or switch to simple iteration.

11.5 Sign-Change Arguments

Once you have an approximate root, sign-change arguments let you prove it is accurate to a given number of decimal places by showing the root lies inside a very small interval.

$$\text{To prove } \alpha \approx p \text{ correct to } k \text{ d.p.:}$$ $$\text{Evaluate } f \text{ at } p - 0.5 \times 10^{-k} \text{ and } p + 0.5 \times 10^{-k}$$ $$\text{Show opposite signs} \implies \alpha \in (p - 0.5\times10^{-k},\ p + 0.5\times10^{-k})$$ $$\implies \alpha \text{ rounds to } p \text{ to } k \text{ decimal places}$$
Worked Example 11.5a
Show that the equation $x^3 - 3x + 1 = 0$ has a root $\alpha$ in the interval $(1.5, 2)$, and prove that $\alpha = 1.532$ correct to 3 decimal places.
1
$f(1.5) = 3.375 - 4.5 + 1 = -0.125 < 0$. $\quad f(2) = 8 - 6 + 1 = 3 > 0$. Sign change → root in $(1.5, 2)$. $\square$
2
To prove $\alpha = 1.532$ correct to 3 d.p., need a sign change in $(1.5315, 1.5325)$:
3
$f(1.5315) = 1.5315^3 - 3(1.5315) + 1 \approx 3.592 - 4.595 + 1 = -0.003 < 0$. $f(1.5325) = 1.5325^3 - 3(1.5325) + 1 \approx 3.599 - 4.598 + 1 = 0.001 > 0$.
4
Since $f(1.5315) < 0$ and $f(1.5325) > 0$ and $f$ is continuous, there is a root in $(1.5315, 1.5325)$. Therefore $\alpha = 1.532$ correct to 3 decimal places. $\square$
The interval $(1.5315, 1.5325)$ is exactly the set of values that round to $1.532$ — this is why we test at $\pm 0.0005$.
The Standard Argument For a root correct to $n$ decimal places: evaluate $f$ at the value with the last digit reduced and increased by 0.5 in the $n$th decimal place. Show opposite signs. State the function is continuous. Conclude the root lies in that interval and therefore rounds correctly. This four-line argument earns full marks every time.

Use Newton–Raphson to find the root of $x\ln x = 1$ near $x = 1.8$, correct to 4 decimal places. Show at least 3 iterations.

Hard
Q8
$(a)$ Prove that if $\alpha$ is a root of $f(x) = 0$ and $f$ is differentiable near $\alpha$, then the Newton–Raphson iteration $x_{n+1} = x_n - \dfrac{f(x_n)}{f'(x_n)}$ satisfies the error relation: $$e_{n+1} \approx -\frac{f''(\alpha)}{2f'(\alpha)}\,e_n^2$$ where $e_n = x_n - \alpha$. Hence show that Newton–Raphson has quadratic convergence. $(b)$ Explain why Newton–Raphson converges more slowly (linearly) at a repeated root where $f(\alpha) = f'(\alpha) = 0$.
$(a)$: Expand $f(x_n) = f(\alpha + e_n)$ by Taylor series: $f(\alpha + e_n) = f(\alpha) + e_n f'(\alpha) + \frac{e_n^2}{2}f''(\alpha) + \cdots = e_n f'(\alpha) + \frac{e_n^2}{2}f''(\alpha) + O(e_n^3)$ (since $f(\alpha) = 0$). Similarly $f'(x_n) = f'(\alpha) + e_n f''(\alpha) + \cdots$ $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} = \alpha + e_n - \frac{e_n f'(\alpha) + \frac{e_n^2}{2}f''(\alpha) + \cdots}{f'(\alpha) + e_n f''(\alpha) + \cdots}$ $= \alpha + e_n - \frac{e_n f'(\alpha)\left(1 + \frac{e_n f''(\alpha)}{2f'(\alpha)}\right)}{f'(\alpha)\left(1 + \frac{e_n f''(\alpha)}{f'(\alpha)}\right)} + O(e_n^3)$ $\approx \alpha + e_n - e_n\left(1 + \frac{e_n f''(\alpha)}{2f'(\alpha)}\right)\left(1 - \frac{e_n f''(\alpha)}{f'(\alpha)}\right) \approx \alpha - \frac{e_n^2 f''(\alpha)}{2f'(\alpha)}$ So $e_{n+1} \approx -\frac{f''(\alpha)}{2f'(\alpha)}e_n^2$. The error is proportional to $e_n^2$ — doubling the number of correct digits each step. $\square$ $(b)$: At a repeated root, $f(\alpha) = f'(\alpha) = 0$, so $f(x) = (x-\alpha)^2 h(x)$ with $h(\alpha) \neq 0$. Then $f'(x) = 2(x-\alpha)h(x) + (x-\alpha)^2 h'(x)$. The Newton step: $\frac{f(x_n)}{f'(x_n)} = \frac{(x_n-\alpha)^2 h(x_n)}{(x_n-\alpha)[2h(x_n) + (x_n-\alpha)h'(x_n)]} \approx \frac{e_n}{2}$. So $x_{n+1} \approx \alpha + \frac{e_n}{2}$, giving $e_{n+1} \approx \frac{1}{2}e_n$ — linear (not quadratic) convergence.
