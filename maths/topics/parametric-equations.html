Parametric Equations

A richer way to describe curves — both $x$ and $y$ expressed as functions of a third variable $t$. Parametric equations handle curves that fold back on themselves, making them essential for motion, engineering, and elegant geometry.

12.1 Parametric Curves

Instead of writing $y$ directly as a function of $x$, parametric equations express both coordinates in terms of a parameter $t$. As $t$ varies, the point $(x(t), y(t))$ traces a curve in the plane.

$$x = f(t), \quad y = g(t), \quad t \in [a, b]$$ $$\text{Each value of } t \text{ gives one point on the curve: } (f(t),\, g(t))$$

The parameter $t$ is often time (giving the position of a moving object) but can be any convenient variable — an angle, a length, or a pure abstract value. Common exam parametrisations use $t$, $\theta$, or $t = \tan\theta$.

Worked Example 12.1a — Sketching a parametric curve
A curve is defined by $x = t^2 - 1$, $y = 2t$ for $t \in [-3, 3]$. Find the coordinates when $t = -2, -1, 0, 1, 2$ and describe the curve.
1
Build a table:
$t = -2$: $x = 3$, $y = -4$
$t = -1$: $x = 0$, $y = -2$
$t = 0$: $x = -1$, $y = 0$
$t = 1$: $x = 0$, $y = 2$
$t = 2$: $x = 3$, $y = 4$
2
Notice: $y = 2t \Rightarrow t = y/2$. Substitute into $x$: $x = (y/2)^2 - 1 = \frac{y^2}{4} - 1$, i.e. $y^2 = 4(x+1)$. This is a parabola with vertex at $(-1, 0)$, opening right.
The parametric form neatly describes both branches of the parabola (positive and negative $y$) in a single sweep.
Why Parametric? Some curves cannot be written as $y = f(x)$ because they fail the vertical line test — they loop back or have multiple $y$-values for a single $x$. A circle, for example, is impossible to express as a single function $y = f(x)$, but is trivial parametrically: $x = r\cos\theta$, $y = r\sin\theta$.

12.2 Converting to Cartesian Form

To find the Cartesian equation, eliminate the parameter $t$ between the two equations. Common techniques: rearrange one equation for $t$ and substitute; use a trigonometric identity to eliminate $\theta$.

$$\text{Method 1 (algebraic): solve } x = f(t) \text{ for } t, \text{ substitute into } y = g(t)$$ $$\text{Method 2 (trig): use } \sin^2\theta + \cos^2\theta = 1 \text{ or } \sec^2\theta = 1 + \tan^2\theta$$
Worked Example 12.2a — Algebraic elimination
Find the Cartesian equation of the curve $x = \dfrac{2}{t}$, $y = t + 1$, $t \neq 0$. State any restrictions.
1
From $x = \dfrac{2}{t}$: $t = \dfrac{2}{x}$.
2
Substitute into $y = t + 1$: $$y = \frac{2}{x} + 1$$
3
Since $t \neq 0$, we have $x \neq 0$. The Cartesian equation is $y = \dfrac{2}{x} + 1$, $x \neq 0$.
Always check for and state any excluded values that arise from the original parameter restriction.
Worked Example 12.2b — Trigonometric elimination
Find the Cartesian equation of $x = 3\cos\theta$, $y = 2\sin\theta$, $0 \leq \theta < 2\pi$. Describe the curve.
1
$\cos\theta = \dfrac{x}{3}$ and $\sin\theta = \dfrac{y}{2}$.
2
Apply $\sin^2\theta + \cos^2\theta = 1$: $$\frac{x^2}{9} + \frac{y^2}{4} = 1$$
3
This is an ellipse centred at the origin, with semi-axes $a = 3$ (along $x$) and $b = 2$ (along $y$).
The standard parametrisation of the ellipse $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ is $x = a\cos\theta$, $y = b\sin\theta$. Worth memorising.
Worked Example 12.2c
Find the Cartesian equation of $x = \sec\theta$, $y = \tan\theta$.
1
Use $\sec^2\theta - \tan^2\theta = 1$: $$x^2 - y^2 = 1$$
This is a hyperbola. The identity $\sec^2\theta = 1 + \tan^2\theta$ is the key tool here.

12.3 Gradient & Tangents

The gradient of a parametric curve uses the chain rule: $\dfrac{dy}{dx} = \dfrac{dy/dt}{dx/dt}$. This is the key formula — differentiate $y$ with respect to $t$, differentiate $x$ with respect to $t$, then divide.

$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} \quad \left(\text{provided } \frac{dx}{dt} \neq 0\right)$$ $$\text{Tangent at parameter } t_0: \text{ gradient } m = \left.\frac{dy}{dx}\right|_{t=t_0}$$ $$\text{Equation: } y - y(t_0) = m\bigl(x - x(t_0)\bigr)$$
Worked Example 12.3a
A curve is defined by $x = t^3 - 3t$, $y = t^2 + t$. Find $\dfrac{dy}{dx}$ in terms of $t$, and find the equation of the tangent at $t = 2$.
1
$\dfrac{dx}{dt} = 3t^2 - 3$, $\quad\dfrac{dy}{dt} = 2t + 1$.
2
$\dfrac{dy}{dx} = \dfrac{2t+1}{3t^2-3}$.
3
At $t = 2$: gradient $= \dfrac{5}{9}$. Point: $x(2) = 8 - 6 = 2$, $y(2) = 4 + 2 = 6$.
4
Tangent: $y - 6 = \dfrac{5}{9}(x - 2) \implies 9y = 5x + 44$.
Worked Example 12.3b — Stationary points
Find the stationary points of $x = 2t^3$, $y = 3t^2 - 12$.
1
Stationary when $\dfrac{dy}{dx} = 0$, i.e. $\dfrac{dy}{dt} = 0$ (provided $\dfrac{dx}{dt} \neq 0$).
2
$\dfrac{dy}{dt} = 6t = 0 \implies t = 0$.
3
At $t = 0$: $x = 0$, $y = -12$. Check $\dfrac{dx}{dt} = 6t^2 = 0$ at $t = 0$ too — both derivatives are zero, so this needs the second derivative test (see 12.4).
⚠ Vertical Tangents If $\dfrac{dx}{dt} = 0$ but $\dfrac{dy}{dt} \neq 0$, the tangent is vertical (undefined gradient). These are not stationary points — they are points where the curve doubles back. Always check both derivatives before concluding.

12.4 Second Derivative

The second derivative $\dfrac{d^2y}{dx^2}$ for a parametric curve requires a careful application of the chain rule. You differentiate $\dfrac{dy}{dx}$ with respect to $t$, then divide by $\dfrac{dx}{dt}$.

$$\frac{d^2y}{dx^2} = \frac{\dfrac{d}{dt}\!\left(\dfrac{dy}{dx}\right)}{\dfrac{dx}{dt}}$$ $$\text{i.e. differentiate } \frac{dy}{dx} \text{ w.r.t. } t, \text{ then divide by } \frac{dx}{dt}$$
Worked Example 12.4a
For the curve $x = t^2$, $y = t^3 - 3t$, find $\dfrac{d^2y}{dx^2}$ and determine the nature of the stationary points.
1
$\dfrac{dy}{dx} = \dfrac{3t^2 - 3}{2t} = \dfrac{3(t^2-1)}{2t}$. Stationary when $\dfrac{dy}{dx} = 0$: $t = \pm 1$.
2
Differentiate $\dfrac{dy}{dx}$ with respect to $t$ using the quotient rule: $$\frac{d}{dt}\!\left(\frac{3t^2-3}{2t}\right) = \frac{6t \cdot 2t - (3t^2-3)\cdot 2}{4t^2} = \frac{12t^2 - 6t^2 + 6}{4t^2} = \frac{6t^2+6}{4t^2} = \frac{3(t^2+1)}{2t^2}$$
3
$\dfrac{d^2y}{dx^2} = \dfrac{3(t^2+1)/2t^2}{2t} = \dfrac{3(t^2+1)}{4t^3}$.
4
At $t = 1$: $\dfrac{d^2y}{dx^2} = \dfrac{6}{4} = \dfrac{3}{2} > 0$ → local minimum at $(1, -2)$. At $t = -1$: $\dfrac{d^2y}{dx^2} = \dfrac{6}{-4} = -\dfrac{3}{2} < 0$ → local maximum at $(1, 2)$.
Both stationary points have $x = t^2 = 1$ — the curve crosses itself at $x = 1$.

12.5 Parametric Integration

To find the area under a parametric curve, substitute the parametric expressions into the standard area formula, using the chain rule to change the variable of integration from $x$ to $t$.

$$A = \int_a^b y\,dx = \int_{t_1}^{t_2} y(t)\cdot\frac{dx}{dt}\,dt$$ $$\text{where } t_1, t_2 \text{ correspond to } x = a, x = b \text{ respectively}$$ $$\text{Take care with limits and signs — sketch the region first}$$
Worked Example 12.5a
Find the area enclosed between the curve $x = t^2$, $y = t(t-2)^2$ and the $x$-axis, for $0 \leq t \leq 2$.
1
$\dfrac{dx}{dt} = 2t$. When $t = 0$: $x = 0$. When $t = 2$: $x = 4$.
2
Check sign of $y$: $y = t(t-2)^2 \geq 0$ for $0 \leq t \leq 2$ ✓. $$A = \int_0^2 t(t-2)^2 \cdot 2t\,dt = 2\int_0^2 t^2(t-2)^2\,dt$$
3
Expand: $t^2(t^2 - 4t + 4) = t^4 - 4t^3 + 4t^2$. $$A = 2\int_0^2 t^4 - 4t^3 + 4t^2\,dt = 2\left[\frac{t^5}{5} - t^4 + \frac{4t^3}{3}\right]_0^2$$ $$= 2\left(\frac{32}{5} - 16 + \frac{32}{3}\right) = 2\left(\frac{96 - 240 + 160}{15}\right) = 2 \times \frac{16}{15} = \frac{32}{15}$$
Worked Example 12.5b — Area of an ellipse
Show that the area of the ellipse $x = a\cos\theta$, $y = b\sin\theta$ is $\pi ab$.
1
By symmetry, area $= 4 \times$ (area in first quadrant where $x, y \geq 0$, i.e. $0 \leq \theta \leq \pi/2$).
2
$\dfrac{dx}{d\theta} = -a\sin\theta$. As $\theta$ goes from $0$ to $\pi/2$, $x$ goes from $a$ to $0$. $$A = 4\int_{\pi/2}^{0} b\sin\theta \cdot (-a\sin\theta)\,d\theta = 4ab\int_0^{\pi/2}\sin^2\theta\,d\theta$$
3
$\displaystyle\int_0^{\pi/2}\sin^2\theta\,d\theta = \dfrac{\pi}{4}$ (standard result). So $A = 4ab \cdot \dfrac{\pi}{4} = \pi ab$. $\square$

Practice Questions

Section A — Foundation
Grade A
Q1
Find the Cartesian equation of: $(a)$ $x = 3t - 1$, $y = t^2 + 2$   $(b)$ $x = 4\cos\theta$, $y = 4\sin\theta$.
$(a)$: $t = \frac{x+1}{3}$, so $y = \left(\frac{x+1}{3}\right)^2 + 2 = \frac{(x+1)^2}{9} + 2$. $(b)$: $\cos\theta = x/4$, $\sin\theta = y/4$. $x^2 + y^2 = 16$ — a circle radius 4.
[4 marks]
Q2
For $x = 2t + 1$, $y = t^2 - 3t$, find $\dfrac{dy}{dx}$ in terms of $t$, and find the coordinates of the stationary point.
$\frac{dy}{dx} = \frac{2t-3}{2}$. Stationary when $2t-3 = 0 \Rightarrow t = \frac{3}{2}$. $x = 4$, $y = \frac{9}{4} - \frac{9}{2} = -\frac{9}{4}$. Point: $\left(4, -\frac{9}{4}\right)$.
[4 marks]
Q3
The curve $C$ is given by $x = t^2 + 1$, $y = 3t - t^3$. Find the equation of the tangent to $C$ at the point where $t = 1$.
At $t=1$: $x=2$, $y=2$. $\frac{dy}{dx} = \frac{3-3t^2}{2t}$. At $t=1$: $m=0$. Tangent: $y = 2$ (horizontal).
[4 marks]
Q4
A curve has $x = \cos 2\theta$, $y = \sin\theta$. Show the Cartesian equation is $x = 1 - 2y^2$.
$x = \cos 2\theta = 1 - 2\sin^2\theta = 1 - 2y^2$. $\square$
[2 marks]
Section B — Challenging
Grade B
Q5
The curve $C$ is defined by $x = 3\cos t$, $y = 2\sin t - 1$, $0 \leq t < 2\pi$. $(a)$ Find $\dfrac{dy}{dx}$ at $t = \dfrac{\pi}{4}$. $(b)$ Find all values of $t$ for which the tangent is horizontal.
$\frac{dy}{dx} = \frac{2\cos t}{-3\sin t} = -\frac{2\cos t}{3\sin t}$. $(a)$: At $t=\pi/4$: $-\frac{2/\sqrt{2}}{3/\sqrt{2}} = -\frac{2}{3}$. $(b)$: Horizontal when $\cos t = 0 \Rightarrow t = \pi/2$ (point $(0,1)$) and $t = 3\pi/2$ (point $(0,-3)$).
[6 marks]
Q6
Find the area enclosed between the curve $x = 3t^2$, $y = 6t - 2t^3$ and the $x$-axis for $0 \leq t \leq \sqrt{3}$.
$\frac{dx}{dt} = 6t$. $y = 0$ when $6t - 2t^3 = 2t(3-t^2) = 0 \Rightarrow t = 0$ or $t = \sqrt{3}$. $y \geq 0$ on this interval. $A = \int_0^{\sqrt{3}} (6t-2t^3)\cdot 6t\,dt = 6\int_0^{\sqrt{3}} 6t^2 - 2t^4\,dt = 6\left[2t^3 - \frac{2t^5}{5}\right]_0^{\sqrt{3}} = 6\left(6\sqrt{3} - \frac{18\sqrt{3}}{5}\right) = 6\cdot\frac{12\sqrt{3}}{5} = \frac{72\sqrt{3}}{5}$.
[7 marks]
Q7
For the curve $x = e^t$, $y = e^{2t} - 3e^t$, find $\dfrac{d^2y}{dx^2}$ and hence determine the nature of the stationary point.
$\frac{dy}{dx} = \frac{2e^{2t}-3e^t}{e^t} = 2e^t - 3$. Stationary when $2e^t = 3 \Rightarrow t = \ln(3/2)$. $\frac{d}{dt}(2e^t-3) = 2e^t$. $\frac{d^2y}{dx^2} = \frac{2e^t}{e^t} = 2 > 0$ for all $t$. Always a minimum. At $t=\ln(3/2)$: $x = 3/2$, $y = 9/4 - 9/2 = -9/4$. Minimum at $\left(\frac{3}{2}, -\frac{9}{4}\right)$.
[7 marks]
Section C — STEP-style
Hard
Q8
The astroid is defined by $x = a\cos^3 t$, $y = a\sin^3 t$, $0 \leq t \leq 2\pi$. $(a)$ Find the Cartesian equation. $(b)$ Show that $\dfrac{dy}{dx} = -\tan t$. $(c)$ The tangent at point $P$ (parameter $t$) meets the $x$- and $y$-axes at $A$ and $B$ respectively. Show that $|AB| = a$ regardless of $t$.
$(a)$: $x^{2/3} = a^{2/3}\cos^2 t$, $y^{2/3} = a^{2/3}\sin^2 t$. Sum: $x^{2/3} + y^{2/3} = a^{2/3}$. $\square$ $(b)$: $\frac{dx}{dt} = -3a\cos^2 t\sin t$, $\frac{dy}{dt} = 3a\sin^2 t\cos t$. $\frac{dy}{dx} = \frac{3a\sin^2 t\cos t}{-3a\cos^2 t\sin t} = -\tan t$. $\square$ $(c)$: Tangent at $(a\cos^3 t, a\sin^3 t)$ with gradient $-\tan t$: $y - a\sin^3 t = -\tan t(x - a\cos^3 t)$ $y\cos t - a\sin^3 t\cos t = -x\sin t + a\cos^3 t\sin t$ $x\sin t + y\cos t = a\sin t\cos t(\cos^2 t + \sin^2 t) = a\sin t\cos t$. Divide: $\frac{x}{a\cos t} + \frac{y}{a\sin t} = 1$. So $A = (a\cos t, 0)$ and $B = (0, a\sin t)$. $|AB| = \sqrt{a^2\cos^2 t + a^2\sin^2 t} = a$. $\square$
