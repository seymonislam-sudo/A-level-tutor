15.1 Compound Angle Formulae

These identities expand $\sin$, $\cos$, and $\tan$ of sums and differences of angles. They are the foundation of all further trigonometric work and must be known perfectly.

$$\sin(A \pm B) = \sin A\cos B \pm \cos A\sin B$$ $$\cos(A \pm B) = \cos A\cos B \mp \sin A\sin B$$ $$\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A\tan B}$$
Memory Aid For $\sin$: the signs match — $\sin(A+B)$ uses $+$ throughout. For $\cos$: the signs swap — $\cos(A+B)$ uses $-$ in the middle. For $\tan$: the sign in the denominator is always opposite to the sign in the numerator.
Worked Example 15.1a — Exact values
Find exact values of $(a)$ $\sin 75°$   $(b)$ $\cos 15°$   $(c)$ $\tan 105°$.
1
$(a)$: $\sin 75° = \sin(45°+30°) = \sin 45°\cos 30° + \cos 45°\sin 30°$ $= \dfrac{\sqrt{2}}{2}\cdot\dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2}\cdot\dfrac{1}{2} = \dfrac{\sqrt{6}+\sqrt{2}}{4}$.
2
$(b)$: $\cos 15° = \cos(45°-30°) = \cos 45°\cos 30° + \sin 45°\sin 30°$ $= \dfrac{\sqrt{2}}{2}\cdot\dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2}\cdot\dfrac{1}{2} = \dfrac{\sqrt{6}+\sqrt{2}}{4}$.
Note: $\sin 75° = \cos 15°$ since they are complementary angles ✓
3
$(c)$: $\tan 105° = \tan(60°+45°) = \dfrac{1+\sqrt{3}}{1-\sqrt{3}} = \dfrac{(1+\sqrt{3})^2}{1-3} = \dfrac{4+2\sqrt{3}}{-2} = -(2+\sqrt{3})$.
Worked Example 15.1b — Proving identities
Prove that $\sin(A+B) + \sin(A-B) = 2\sin A\cos B$.
1
LHS $= (\sin A\cos B + \cos A\sin B) + (\sin A\cos B - \cos A\sin B)$ $= 2\sin A\cos B$ = RHS. $\square$
This is the sum-to-product formula — it works in reverse too (product-to-sum).

15.2 Double Angle Formulae

Setting $B = A$ in the compound angle formulae gives the double angle formulae. There are three equivalent forms of $\cos 2A$ — choose the one that matches what you're given or want to achieve.

$$\sin 2A = 2\sin A\cos A$$ $$\cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A$$ $$\tan 2A = \frac{2\tan A}{1 - \tan^2 A}$$ $$\text{Rearranged: } \cos^2 A = \frac{1+\cos 2A}{2}, \quad \sin^2 A = \frac{1-\cos 2A}{2}$$
Worked Example 15.2a
Given $\sin\theta = \dfrac{3}{5}$, $0 < \theta < \dfrac{\pi}{2}$, find exact values of $(a)$ $\sin 2\theta$   $(b)$ $\cos 2\theta$   $(c)$ $\tan 2\theta$.
1
From $\sin\theta = 3/5$ and Pythagoras: $\cos\theta = 4/5$, $\tan\theta = 3/4$.
2
$(a)$: $\sin 2\theta = 2 \times \frac{3}{5} \times \frac{4}{5} = \dfrac{24}{25}$.
3
$(b)$: $\cos 2\theta = \cos^2\theta - \sin^2\theta = \dfrac{16}{25} - \dfrac{9}{25} = \dfrac{7}{25}$.
4
$(c)$: $\tan 2\theta = \dfrac{2 \times 3/4}{1 - 9/16} = \dfrac{3/2}{7/16} = \dfrac{3}{2} \times \dfrac{16}{7} = \dfrac{24}{7}$.
Check: $\frac{24/25}{7/25} = 24/7$ ✓
Worked Example 15.2b — Solving equations
Solve $\sin 2x = \sin x$ for $0 \leq x \leq 2\pi$.
1
$2\sin x\cos x = \sin x \implies \sin x(2\cos x - 1) = 0$.
2
$\sin x = 0$: $x = 0, \pi, 2\pi$. $\cos x = \dfrac{1}{2}$: $x = \dfrac{\pi}{3}, \dfrac{5\pi}{3}$.
3
Solutions: $x = 0, \dfrac{\pi}{3}, \pi, \dfrac{5\pi}{3}, 2\pi$.
Never divide both sides by $\sin x$ — you'd lose the solutions $\sin x = 0$. Always factorise.

15.3 The R-Formula

Any expression of the form $a\sin\theta + b\cos\theta$ can be written as a single sinusoid $R\sin(\theta + \alpha)$ or $R\cos(\theta - \alpha)$. This is essential for finding maxima, minima, and solving equations involving combined sinusoids.

$$a\sin\theta + b\cos\theta = R\sin(\theta + \alpha)$$ $$\text{where } R = \sqrt{a^2+b^2}, \quad \tan\alpha = \frac{b}{a}, \quad R > 0, \; 0 < \alpha < \frac{\pi}{2}$$ $$a\sin\theta - b\cos\theta = R\sin(\theta - \alpha) \quad \text{same } R, \text{ same } \alpha$$ $$a\cos\theta + b\sin\theta = R\cos(\theta - \alpha) \quad \tan\alpha = \frac{b}{a}$$
Worked Example 15.3a
Express $3\sin\theta + 4\cos\theta$ in the form $R\sin(\theta + \alpha)$, where $R > 0$ and $0 < \alpha < \pi/2$.
1
$R = \sqrt{9+16} = 5$. $\tan\alpha = \dfrac{4}{3}$, so $\alpha = \arctan\!\left(\dfrac{4}{3}\right) \approx 0.927$ rad.
2
$3\sin\theta + 4\cos\theta = 5\sin(\theta + 0.927)$.
3
Verify by expanding: $5(\sin\theta\cos\alpha + \cos\theta\sin\alpha) = 5\sin\theta\cdot\dfrac{3}{5} + 5\cos\theta\cdot\dfrac{4}{5} = 3\sin\theta + 4\cos\theta$ ✓.
Worked Example 15.3b — Solving equations
Solve $3\sin\theta + 4\cos\theta = 2$ for $0 \leq \theta \leq 2\pi$.
1
From 15.3a: $5\sin(\theta + 0.927) = 2 \implies \sin(\theta + 0.927) = 0.4$.
2
Let $\phi = \theta + 0.927$. Range: $0.927 \leq \phi \leq 2\pi + 0.927$. $\phi = \arcsin 0.4 \approx 0.412$ (outside range) or $\phi = \pi - 0.412 = 2.730$, or $\phi = 2\pi + 0.412 = 6.695$.
3
$\theta = 2.730 - 0.927 = 1.803$ or $\theta = 6.695 - 0.927 = 5.768$. Solutions: $\theta \approx 1.80$ and $\theta \approx 5.77$ (3 s.f.).
Worked Example 15.3c — Maximum value
Find the maximum value of $f(\theta) = \dfrac{1}{5 + 3\sin\theta + 4\cos\theta}$ and the value of $\theta$ at which it occurs.
1
From 15.3a: $3\sin\theta + 4\cos\theta = 5\sin(\theta + \alpha)$, max value $= 5$. So the denominator has maximum $5 + 5 = 10$ and minimum $5 - 5 = 0$.
2
Maximum of $f$ occurs when denominator is minimum (and positive): denominator $= 5 + 5\sin(\theta+\alpha) \geq 0$, minimum positive value as $\theta \to$ the value giving $\sin(\theta+\alpha) = -1$. Actually: min denominator $= 0$ (asymptote). To find the maximum of $f$ over a bounded domain, the denominator min above zero would need context. For the unconstrained problem, $f$ is unbounded. If the question intends $3\sin\theta + 4\cos\theta$ in denominator only: max of $f = \frac{1}{5-5} $... the question as stated makes $f$ unbounded. In context, typical exam phrasing would be: max of $\frac{1}{10 + 3\sin\theta + 4\cos\theta}$.
For $\frac{1}{10+5\sin(\theta+\alpha)}$: max when $5\sin(\theta+\alpha) = -5$ giving min denominator $= 5$. Max of $f = \frac{1}{5}$ at $\theta + \alpha = 3\pi/2$, i.e. $\theta = 3\pi/2 - \alpha$.

15.4 Reciprocal Trig Functions

Three additional trig functions — cosecant, secant, and cotangent — arise naturally as reciprocals. Their Pythagorean identities are just as important as $\sin^2 + \cos^2 = 1$.

$$\csc x = \frac{1}{\sin x} \qquad \sec x = \frac{1}{\cos x} \qquad \cot x = \frac{\cos x}{\sin x} = \frac{1}{\tan x}$$ $$1 + \tan^2 x = \sec^2 x \quad\text{(from } \sin^2+\cos^2=1, \div \cos^2\text{)}$$ $$1 + \cot^2 x = \csc^2 x \quad\text{(from } \sin^2+\cos^2=1, \div \sin^2\text{)}$$
Worked Example 15.4a — Proving identities
Prove that $\dfrac{\sec^2\theta - 1}{\sec^2\theta} = \sin^2\theta$.
1
LHS $= 1 - \dfrac{1}{\sec^2\theta} = 1 - \cos^2\theta = \sin^2\theta$ = RHS. $\square$
Worked Example 15.4b — Solving equations
Solve $2\cot^2\theta = 5\csc\theta - 4$ for $0 < \theta < 2\pi$.
1
Replace $\cot^2\theta = \csc^2\theta - 1$: $2(\csc^2\theta - 1) = 5\csc\theta - 4 \implies 2\csc^2\theta - 5\csc\theta + 2 = 0$.
2
Let $c = \csc\theta$: $2c^2 - 5c + 2 = 0 \implies (2c-1)(c-2) = 0$. $c = \frac{1}{2}$ or $c = 2$.
3
$\csc\theta = \frac{1}{2} \implies \sin\theta = 2$ — impossible. $\csc\theta = 2 \implies \sin\theta = \frac{1}{2} \implies \theta = \dfrac{\pi}{6}$ or $\theta = \dfrac{5\pi}{6}$.

15.5 Inverse Trig Functions

The inverse functions $\arcsin$, $\arccos$, $\arctan$ are defined on restricted domains to ensure they are well-defined (single-valued). Their ranges are crucial for solving equations correctly.

$$\arcsin: [-1,1] \to \left[-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right] \quad\text{(principal value range)}$$ $$\arccos: [-1,1] \to [0, \pi]$$ $$\arctan: \mathbb{R} \to \left(-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right)$$ $$\frac{d}{dx}(\arcsin x) = \frac{1}{\sqrt{1-x^2}} \quad \frac{d}{dx}(\arctan x) = \frac{1}{1+x^2}$$
Worked Example 15.5a
Without a calculator, evaluate $(a)$ $\arcsin\!\left(\dfrac{\sqrt{3}}{2}\right)$   $(b)$ $\arctan(-1)$   $(c)$ $\arccos\!\left(-\dfrac{1}{2}\right)$.
1
$(a)$: $\sin\!\left(\dfrac{\pi}{3}\right) = \dfrac{\sqrt{3}}{2}$ and $\dfrac{\pi}{3} \in \left[-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right]$. Answer: $\dfrac{\pi}{3}$.
2
$(b)$: $\tan\!\left(-\dfrac{\pi}{4}\right) = -1$ and $-\dfrac{\pi}{4} \in \left(-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right)$. Answer: $-\dfrac{\pi}{4}$.
3
$(c)$: $\cos\!\left(\dfrac{2\pi}{3}\right) = -\dfrac{1}{2}$ and $\dfrac{2\pi}{3} \in [0, \pi]$. Answer: $\dfrac{2\pi}{3}$.
Note: $\arccos(-x) = \pi - \arccos(x)$ — a useful shortcut.

15.6 Small Angle Approximations

When $\theta$ is small (measured in radians), the trig functions are approximated by simple polynomials. These approximations are the first terms of the Maclaurin series and are used throughout physics and engineering.

$$\sin\theta \approx \theta \qquad \cos\theta \approx 1 - \frac{\theta^2}{2} \qquad \tan\theta \approx \theta$$ $$\text{Valid for small } \theta \text{ in radians} \quad (\text{error} \sim \theta^3 \text{ for sin/tan, } \theta^4 \text{ for cos})$$
Worked Example 15.6a
Using small angle approximations, simplify $\dfrac{\sin 3\theta - \tan 2\theta}{\theta}$ for small $\theta$.
1
$\sin 3\theta \approx 3\theta$ and $\tan 2\theta \approx 2\theta$.
2
$\dfrac{3\theta - 2\theta}{\theta} = \dfrac{\theta}{\theta} = 1$.
Worked Example 15.6b
Show that $\dfrac{1 - \cos 2\theta}{\theta\sin\theta} \approx 2$ for small $\theta$, and find the next correction term.
1
$\cos 2\theta \approx 1 - \dfrac{(2\theta)^2}{2} = 1 - 2\theta^2$, so $1 - \cos 2\theta \approx 2\theta^2$.
2
$\sin\theta \approx \theta - \dfrac{\theta^3}{6}$ (including next term). $\theta\sin\theta \approx \theta^2 - \dfrac{\theta^4}{6}$.
3
$\dfrac{2\theta^2}{\theta^2(1 - \theta^2/6)} \approx 2\left(1 + \dfrac{\theta^2}{6}\right) = 2 + \dfrac{\theta^2}{3}$. So the leading correction is $+\dfrac{\theta^2}{3}$.
Connecting to Calculus The small angle approximations are exactly why $\frac{d}{dx}(\sin x) = \cos x$ — the derivative is defined as $\lim_{\delta\to 0}\frac{\sin(x+\delta)-\sin x}{\delta}$, and using the compound angle formula then the small angle $\sin\delta \approx \delta$ gives the result. This connection makes the approximations more than a trick — they are calculus in disguise.

Practice Questions

Section A — Foundation
Grade A
Q1
Given $\cos A = \dfrac{5}{13}$ and $\sin B = \dfrac{4}{5}$ where $A$ and $B$ are both acute, find exact values of $(a)$ $\sin(A+B)$   $(b)$ $\cos(A-B)$.
$\sin A = 12/13$, $\cos B = 3/5$. $(a)$: $\sin(A+B) = \frac{12}{13}\cdot\frac{3}{5} + \frac{5}{13}\cdot\frac{4}{5} = \frac{36}{65}+\frac{20}{65} = \frac{56}{65}$. $(b)$: $\cos(A-B) = \frac{5}{13}\cdot\frac{3}{5} + \frac{12}{13}\cdot\frac{4}{5} = \frac{3}{13}+\frac{48}{65} = \frac{15+48}{65} = \frac{63}{65}$.
[5 marks]
Q2
Solve $\cos 2x + \cos x = 0$ for $0 \leq x \leq 2\pi$.
$(2\cos^2 x - 1) + \cos x = 0 \Rightarrow 2\cos^2 x + \cos x - 1 = 0 \Rightarrow (2\cos x - 1)(\cos x + 1) = 0$. $\cos x = \frac{1}{2}$: $x = \pi/3, 5\pi/3$. $\cos x = -1$: $x = \pi$. Solutions: $x = \frac{\pi}{3}, \pi, \frac{5\pi}{3}$.
[5 marks]
Q3
Express $5\cos\theta - 12\sin\theta$ in the form $R\cos(\theta + \alpha)$ where $R > 0$ and $0 < \alpha < \pi/2$. Hence state its maximum and minimum values.
$R = \sqrt{25+144} = 13$. $\tan\alpha = 12/5 \Rightarrow \alpha = \arctan(12/5) \approx 1.176$ rad. $5\cos\theta - 12\sin\theta = 13\cos(\theta + 1.176)$. Maximum: $13$. Minimum: $-13$.
[5 marks]
Q4
Using small angle approximations, find an approximate value of $\dfrac{\cos\theta - 1 + \frac{1}{2}\sin^2\theta}{\theta^4}$ as $\theta \to 0$.
$\cos\theta \approx 1 - \theta^2/2 + \theta^4/24$. $\sin^2\theta \approx \theta^2$. Numerator: $(1 - \theta^2/2 + \theta^4/24) - 1 + \theta^2/2 = \theta^4/24$. So the limit is $\frac{1}{24}$.
[4 marks]
Section B — Challenging
Grade B
Q5
Prove the identity $\dfrac{\sin 3\theta}{\sin\theta} - \dfrac{\cos 3\theta}{\cos\theta} = 2$.
LHS $= \frac{\sin 3\theta\cos\theta - \cos 3\theta\sin\theta}{\sin\theta\cos\theta} = \frac{\sin(3\theta - \theta)}{\frac{1}{2}\sin 2\theta} = \frac{\sin 2\theta}{\frac{1}{2}\sin 2\theta} = 2$ = RHS. $\square$ (Used compound angle formula $\sin(A-B) = \sin A\cos B - \cos A\sin B$ and $\sin 2\theta = 2\sin\theta\cos\theta$.)
[4 marks]
Q6
Solve $\sec^2\theta + \tan\theta = 3$ for $0 \leq \theta < 2\pi$.
$(1 + \tan^2\theta) + \tan\theta = 3 \Rightarrow \tan^2\theta + \tan\theta - 2 = 0 \Rightarrow (\tan\theta + 2)(\tan\theta - 1) = 0$. $\tan\theta = 1$: $\theta = \pi/4, 5\pi/4$. $\tan\theta = -2$: $\theta = \pi - \arctan 2 \approx 2.034$ and $\theta = 2\pi - \arctan 2 \approx 5.176$. Four solutions.
[6 marks]
Q7
$(a)$ Express $\cos\theta + \sqrt{3}\sin\theta$ in the form $R\cos(\theta - \alpha)$. $(b)$ Hence find $\displaystyle\int_0^{\pi/2}\frac{1}{(2+\cos\theta+\sqrt{3}\sin\theta)^2}\,d\theta$.
$(a)$: $R = 2$, $\tan\alpha = \sqrt{3} \Rightarrow \alpha = \pi/3$. So $\cos\theta + \sqrt{3}\sin\theta = 2\cos(\theta - \pi/3)$. $(b)$: $\int_0^{\pi/2}\frac{d\theta}{(2+2\cos(\theta-\pi/3))^2} = \frac{1}{4}\int_0^{\pi/2}\frac{d\theta}{(1+\cos(\theta-\pi/3))^2}$. Using $1+\cos\phi = 2\cos^2(\phi/2)$: $= \frac{1}{16}\int_0^{\pi/2}\sec^4\!\left(\frac{\theta-\pi/3}{2}\right)d\theta$. Let $u = \frac{\theta-\pi/3}{2}$, limits $-\pi/6$ to $\pi/12$: $\frac{1}{8}\int_{-\pi/6}^{\pi/12}\sec^4 u\,du = \frac{1}{8}\int\,(1+\tan^2 u)\sec^2 u\,du = \frac{1}{8}[\tan u + \frac{\tan^3 u}{3}]_{-\pi/6}^{\pi/12}$.
[10 marks]
Section C — STEP-style
Hard
Q8
$(a)$ By writing $3\theta = 2\theta + \theta$ and using compound/double angle formulae, show that: $$\sin 3\theta = 3\sin\theta - 4\sin^3\theta \qquad \cos 3\theta = 4\cos^3\theta - 3\cos\theta$$ $(b)$ Hence solve $8\sin^3\theta - 6\sin\theta + 1 = 0$ for $0 \leq \theta \leq \pi$. $(c)$ Using the result for $\cos 3\theta$, find the exact value of $\cos 20°$, and explain why it is a root of $8x^3 - 6x - 1 = 0$.
$(a)$: $\sin 3\theta = \sin(2\theta+\theta) = \sin 2\theta\cos\theta + \cos 2\theta\sin\theta = 2\sin\theta\cos^2\theta + (1-2\sin^2\theta)\sin\theta = 2\sin\theta(1-\sin^2\theta) + \sin\theta - 2\sin^3\theta = 3\sin\theta - 4\sin^3\theta$. $\square$ $\cos 3\theta = \cos(2\theta+\theta) = \cos 2\theta\cos\theta - \sin 2\theta\sin\theta = (2\cos^2\theta-1)\cos\theta - 2\sin^2\theta\cos\theta = 2\cos^3\theta-\cos\theta - 2(1-\cos^2\theta)\cos\theta = 4\cos^3\theta - 3\cos\theta$. $\square$ $(b)$: $8\sin^3\theta - 6\sin\theta + 1 = 0 \Rightarrow -2(3\sin\theta - 4\sin^3\theta) + 1 = 0 \Rightarrow -2\sin 3\theta = -1 \Rightarrow \sin 3\theta = \frac{1}{2}$. $3\theta = \pi/6, 5\pi/6, 13\pi/6$ (for $0 \leq \theta \leq \pi$, $3\theta \in [0, 3\pi]$). $\theta = \pi/18, 5\pi/18, 13\pi/18$. $(c)$: $4\cos^3 20° - 3\cos 20° = \cos 60° = \frac{1}{2}$. So $8\cos^3 20° - 6\cos 20° = 1$, i.e. $8\cos^3 20° - 6\cos 20° - 1 = 0$. Therefore $\cos 20°$ is a root of $8x^3 - 6x - 1 = 0$. $\square$ (The exact value $\cos 20°$ cannot be expressed in simple surds — this cubic has no rational roots, illustrating the impossibility of trisecting an angle with compass and straightedge.)
