Further Integration

The full integration toolkit — substitution, parts, partial fractions, and trigonometric techniques. These methods together can integrate virtually any function encountered at A Level, and they lay the groundwork for differential equations.

14.1 Standard Integrals

These results follow directly from reversing standard derivatives. Knowing them fluently — without hesitation — is essential for all further integration work.

$$\int e^x\,dx = e^x + c \qquad \int e^{ax+b}\,dx = \frac{1}{a}e^{ax+b} + c$$ $$\int \frac{1}{x}\,dx = \ln|x| + c \qquad \int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + c$$ $$\int \sin x\,dx = -\cos x + c \qquad \int \cos x\,dx = \sin x + c$$ $$\int \sin(ax+b)\,dx = -\frac{1}{a}\cos(ax+b) + c \qquad \int \cos(ax+b)\,dx = \frac{1}{a}\sin(ax+b) + c$$ $$\int \sec^2 x\,dx = \tan x + c \qquad \int \frac{1}{\sqrt{1-x^2}}\,dx = \arcsin x + c \qquad \int \frac{1}{1+x^2}\,dx = \arctan x + c$$
Worked Example 14.1a
Evaluate $(a)$ $\displaystyle\int_0^1 e^{2x-1}\,dx$   $(b)$ $\displaystyle\int \dfrac{3x^2+1}{x^3+x}\,dx$   $(c)$ $\displaystyle\int \cos(3x)\,dx$
1
$(a)$: $\left[\dfrac{1}{2}e^{2x-1}\right]_0^1 = \dfrac{1}{2}e^1 - \dfrac{1}{2}e^{-1} = \dfrac{e}{2} - \dfrac{1}{2e} = \dfrac{e^2-1}{2e}$.
2
$(b)$: Numerator $3x^2 + 1$ is the derivative of $x^3 + x$. So $\displaystyle\int \dfrac{3x^2+1}{x^3+x}\,dx = \ln|x^3+x| + c$.
3
$(c)$: $\dfrac{1}{3}\sin(3x) + c$.

14.2 Integration by Substitution

Substitution reverses the chain rule. Replace a complicated expression with a single variable $u$, rewrite $dx$ in terms of $du$, integrate, then substitute back. For definite integrals, also change the limits.

$$\int f(g(x))\,g'(x)\,dx = \int f(u)\,du \quad \text{where } u = g(x),\; du = g'(x)\,dx$$ $$\text{Definite: change limits too — when } x = a,\; u = g(a); \text{ when } x = b,\; u = g(b)$$
Worked Example 14.2a
Find $\displaystyle\int x\sqrt{x^2+1}\,dx$.
1
Let $u = x^2 + 1$. Then $\dfrac{du}{dx} = 2x$, so $x\,dx = \dfrac{du}{2}$.
2
$\displaystyle\int x\sqrt{u}\cdot\frac{du}{2x} = \frac{1}{2}\int u^{1/2}\,du = \frac{1}{2}\cdot\frac{2}{3}u^{3/2} + c = \frac{1}{3}(x^2+1)^{3/2} + c$.
Worked Example 14.2b — Definite integral
Evaluate $\displaystyle\int_0^{\pi/2} \sin^3 x\cos x\,dx$.
1
Let $u = \sin x$, $du = \cos x\,dx$. Limits: $x=0 \Rightarrow u=0$; $x=\pi/2 \Rightarrow u=1$.
2
$\displaystyle\int_0^1 u^3\,du = \left[\frac{u^4}{4}\right]_0^1 = \frac{1}{4}$.
Worked Example 14.2c — Trigonometric substitution
Find $\displaystyle\int \frac{1}{\sqrt{4-x^2}}\,dx$ using $x = 2\sin\theta$.
1
$x = 2\sin\theta$, so $dx = 2\cos\theta\,d\theta$. $\sqrt{4-x^2} = \sqrt{4-4\sin^2\theta} = 2\cos\theta$.
2
$\displaystyle\int\frac{2\cos\theta\,d\theta}{2\cos\theta} = \int d\theta = \theta + c = \arcsin\!\left(\frac{x}{2}\right) + c$.

14.3 Integration by Parts

Integration by parts reverses the product rule. It converts the integral of a product into a (hopefully simpler) integral. The key skill is choosing which factor to differentiate ($u$) and which to integrate ($\frac{dv}{dx}$).

$$\int u\frac{dv}{dx}\,dx = uv - \int v\frac{du}{dx}\,dx$$ $$\text{LIATE priority for choosing } u\text{: Logarithm, Inverse trig, Algebraic, Trig, Exponential}$$ $$\text{(Choose } u \text{ as the first type that appears in LIATE)}$$
Worked Example 14.3a
Find $(a)$ $\displaystyle\int xe^x\,dx$   $(b)$ $\displaystyle\int x\cos x\,dx$   $(c)$ $\displaystyle\int \ln x\,dx$
1
$(a)$: $u = x$, $\frac{dv}{dx} = e^x$ so $v = e^x$. $\displaystyle\int xe^x\,dx = xe^x - \int e^x\,dx = xe^x - e^x + c = e^x(x-1) + c$.
2
$(b)$: $u = x$, $\frac{dv}{dx} = \cos x$ so $v = \sin x$. $\displaystyle\int x\cos x\,dx = x\sin x - \int \sin x\,dx = x\sin x + \cos x + c$.
3
$(c)$: Write as $\displaystyle\int \ln x \cdot 1\,dx$. $u = \ln x$, $\frac{dv}{dx} = 1$ so $v = x$. $\displaystyle\int \ln x\,dx = x\ln x - \int x \cdot \frac{1}{x}\,dx = x\ln x - x + c$.
Worked Example 14.3b — Applying parts twice
Find $\displaystyle\int x^2 e^x\,dx$.
1
First application: $u = x^2$, $v = e^x$. $\displaystyle\int x^2 e^x\,dx = x^2 e^x - 2\int xe^x\,dx$.
2
Second application on $\int xe^x\,dx$ (from 14.3a): $= e^x(x-1)$.
3
$\displaystyle\int x^2 e^x\,dx = x^2 e^x - 2e^x(x-1) + c = e^x(x^2 - 2x + 2) + c$.
Worked Example 14.3c — Cyclic integration
Find $I = \displaystyle\int e^x\sin x\,dx$.
1
$u = e^x$, $\frac{dv}{dx} = \sin x$, $v = -\cos x$. $I = -e^x\cos x + \displaystyle\int e^x\cos x\,dx$.
2
Apply parts again to $\int e^x\cos x\,dx$: $u = e^x$, $v = \sin x$. $\displaystyle\int e^x\cos x\,dx = e^x\sin x - \int e^x\sin x\,dx = e^x\sin x - I$.
3
Substitute back: $I = -e^x\cos x + e^x\sin x - I$. $2I = e^x(\sin x - \cos x)$. $I = \dfrac{e^x(\sin x - \cos x)}{2} + c$.
Whenever the original integral reappears, bring it to one side and solve algebraically.

14.4 Partial Fractions

Partial fractions decompose a rational function into simpler fractions that are easy to integrate. The form of the decomposition depends on the factors of the denominator.

$$\frac{px+q}{(ax+b)(cx+d)} = \frac{A}{ax+b} + \frac{B}{cx+d}$$ $$\frac{px^2+qx+r}{(ax+b)(cx+d)^2} = \frac{A}{ax+b} + \frac{B}{cx+d} + \frac{C}{(cx+d)^2}$$ $$\frac{px^2+qx+r}{(ax+b)(x^2+c^2)} = \frac{A}{ax+b} + \frac{Bx+C}{x^2+c^2}$$
Worked Example 14.4a
Find $\displaystyle\int \frac{5x+1}{(x+1)(2x-1)}\,dx$.
1
$\dfrac{5x+1}{(x+1)(2x-1)} = \dfrac{A}{x+1} + \dfrac{B}{2x-1}$. Multiply through: $5x+1 = A(2x-1) + B(x+1)$.
2
$x = \frac{1}{2}$: $\frac{7}{2} = \frac{3B}{2} \Rightarrow B = \frac{7}{3}$. $x = -1$: $-4 = -3A \Rightarrow A = \frac{4}{3}$.
3
$\displaystyle\int \frac{4/3}{x+1} + \frac{7/3}{2x-1}\,dx = \frac{4}{3}\ln|x+1| + \frac{7}{6}\ln|2x-1| + c$.
For $\frac{B}{2x-1}$: $\int\frac{B}{2x-1}dx = \frac{B}{2}\ln|2x-1|$, not $B\ln|2x-1|$.
Worked Example 14.4b — Improper fraction
Find $\displaystyle\int \frac{x^2+3x+1}{x^2-1}\,dx$.
1
Degree of numerator $\geq$ denominator — perform polynomial division first: $\dfrac{x^2+3x+1}{x^2-1} = 1 + \dfrac{3x+2}{x^2-1} = 1 + \dfrac{3x+2}{(x+1)(x-1)}$.
2
$\dfrac{3x+2}{(x+1)(x-1)} = \dfrac{A}{x+1} + \dfrac{B}{x-1}$. Cover-up: $A = \frac{-1}{-2} = \frac{1}{2}$, $B = \frac{5}{2}$.
3
$\displaystyle\int 1 + \frac{1/2}{x+1} + \frac{5/2}{x-1}\,dx = x + \frac{1}{2}\ln|x+1| + \frac{5}{2}\ln|x-1| + c$.

14.5 Trigonometric Integrals

Powers of trig functions require double-angle identities to reduce to integrable forms. This is one of the most tested skills in A Level integration.

$$\cos^2 x = \frac{1+\cos 2x}{2} \qquad \sin^2 x = \frac{1-\cos 2x}{2}$$ $$\int \sin^2 x\,dx = \frac{x}{2} - \frac{\sin 2x}{4} + c \qquad \int \cos^2 x\,dx = \frac{x}{2} + \frac{\sin 2x}{4} + c$$ $$\int \tan^2 x\,dx = \tan x - x + c \quad\text{(using } \tan^2 x = \sec^2 x - 1\text{)}$$
Worked Example 14.5a
Evaluate $\displaystyle\int_0^{\pi} \sin^2 x\,dx$.
1
Use $\sin^2 x = \dfrac{1-\cos 2x}{2}$: $$\int_0^{\pi}\frac{1-\cos 2x}{2}\,dx = \left[\frac{x}{2} - \frac{\sin 2x}{4}\right]_0^{\pi}$$
2
$= \left(\dfrac{\pi}{2} - 0\right) - (0 - 0) = \dfrac{\pi}{2}$.
Worked Example 14.5b — Higher powers
Find $\displaystyle\int \cos^4 x\,dx$.
1
$\cos^4 x = \left(\dfrac{1+\cos 2x}{2}\right)^2 = \dfrac{1 + 2\cos 2x + \cos^2 2x}{4}$.
2
Apply the identity again: $\cos^2 2x = \dfrac{1+\cos 4x}{2}$. $$\cos^4 x = \frac{1}{4}\left(1 + 2\cos 2x + \frac{1+\cos 4x}{2}\right) = \frac{3}{8} + \frac{\cos 2x}{2} + \frac{\cos 4x}{8}$$
3
$\displaystyle\int \cos^4 x\,dx = \frac{3x}{8} + \frac{\sin 2x}{4} + \frac{\sin 4x}{32} + c$.

14.6 Differential Equations

A differential equation relates a function to its derivatives. At A Level, we solve separable first-order ODEs: rearrange to get all $y$ terms on one side and all $x$ terms on the other, then integrate both sides.

$$\frac{dy}{dx} = f(x)\,g(y) \implies \int \frac{1}{g(y)}\,dy = \int f(x)\,dx$$ $$\text{General solution: includes constant } c$$ $$\text{Particular solution: use initial conditions to find } c$$
Worked Example 14.6a
Solve $\dfrac{dy}{dx} = 2xy$ with $y = 3$ when $x = 0$.
1
Separate: $\dfrac{1}{y}\,dy = 2x\,dx$.
2
Integrate: $\ln|y| = x^2 + c$.
3
Exponentiate: $y = Ae^{x^2}$ (where $A = e^c$). Apply $y(0) = 3$: $3 = A$. Particular solution: $y = 3e^{x^2}$.
Worked Example 14.6b — Population model
A population $P$ grows according to $\dfrac{dP}{dt} = kP(N-P)$ where $N$ and $k$ are positive constants (logistic growth). Solve by partial fractions given $P(0) = P_0$.
1
Separate: $\displaystyle\int\frac{dP}{P(N-P)} = \int k\,dt$.
2
Partial fractions: $\dfrac{1}{P(N-P)} = \dfrac{1}{N}\!\left(\dfrac{1}{P} + \dfrac{1}{N-P}\right)$. $\dfrac{1}{N}\bigl(\ln P - \ln(N-P)\bigr) = kt + c$. $\ln\dfrac{P}{N-P} = Nkt + C$.
3
$\dfrac{P}{N-P} = Ae^{Nkt}$. At $t=0$: $A = \dfrac{P_0}{N-P_0}$. Solving for $P$: $P = \dfrac{NP_0 e^{Nkt}}{N - P_0 + P_0 e^{Nkt}} = \dfrac{N}{1 + \left(\frac{N-P_0}{P_0}\right)e^{-Nkt}}$.
As $t\to\infty$, $P \to N$ (the carrying capacity) — the population stabilises.

Practice Questions

Section A — Foundation
Grade A
Q1
Integrate: $(a)$ $\displaystyle\int 4e^{2x}\,dx$   $(b)$ $\displaystyle\int \dfrac{2x}{x^2+5}\,dx$   $(c)$ $\displaystyle\int \sin(3x+1)\,dx$
$(a)$: $2e^{2x}+c$. $(b)$: $\ln(x^2+5)+c$. $(c)$: $-\frac{1}{3}\cos(3x+1)+c$.
[6 marks]
Q2
Use integration by parts to find $\displaystyle\int x\sin(2x)\,dx$.
$u=x$, $v = -\frac{1}{2}\cos 2x$. $\int x\sin 2x\,dx = -\frac{x}{2}\cos 2x + \frac{1}{2}\int\cos 2x\,dx = -\frac{x}{2}\cos 2x + \frac{\sin 2x}{4} + c$.
[4 marks]
Q3
Solve $\dfrac{dy}{dx} = \dfrac{x+1}{y}$ given $y = 2$ when $x = 0$. Give your answer in the form $y^2 = f(x)$.
$y\,dy = (x+1)\,dx$. $\frac{y^2}{2} = \frac{x^2}{2} + x + c$. At $(0,2)$: $2 = c$. $y^2 = x^2 + 2x + 4$.
[4 marks]
Q4
Use substitution $u = x^2 - 1$ to find $\displaystyle\int_1^2 \dfrac{x}{\sqrt{x^2-1}}\,dx$.
$du = 2x\,dx$. Limits: $x=1\Rightarrow u=0$; $x=2\Rightarrow u=3$. $\int_0^3 \frac{du}{2\sqrt{u}} = \left[\sqrt{u}\right]_0^3 = \sqrt{3}$.
[4 marks]
Section B — Challenging
Grade B
Q5
Find $\displaystyle\int \frac{3x^2 - 4}{(x-1)(x+2)^2}\,dx$ using partial fractions.
$\frac{3x^2-4}{(x-1)(x+2)^2} = \frac{A}{x-1}+\frac{B}{x+2}+\frac{C}{(x+2)^2}$. Cover-up: $A = \frac{-1}{9}$, $C = \frac{8}{-3} = -8/3$. Compare $x^2$: $A+B = 3 \Rightarrow B = 3+\frac{1}{9} = \frac{28}{9}$. $\int = -\frac{1}{9}\ln|x-1| + \frac{28}{9}\ln|x+2| + \frac{8}{3(x+2)} + c$.
[8 marks]
Q6
Evaluate $\displaystyle\int_0^{\pi/4} \tan^2 x\,dx$ and $\displaystyle\int_0^{\pi/2}\cos^2(3x)\,dx$.
$\int_0^{\pi/4}\tan^2 x\,dx = \int_0^{\pi/4}(\sec^2 x - 1)\,dx = [\tan x - x]_0^{\pi/4} = 1 - \pi/4$. For $\cos^2(3x) = \frac{1+\cos 6x}{2}$: $\int_0^{\pi/2}\frac{1+\cos 6x}{2}\,dx = \left[\frac{x}{2}+\frac{\sin 6x}{12}\right]_0^{\pi/2} = \frac{\pi}{4} + 0 = \frac{\pi}{4}$.
[6 marks]
Q7
A curve satisfies $\dfrac{dy}{dx} = \dfrac{y^2+1}{x}$ with $y=1$ when $x=1$. Find $y$ in terms of $x$.
$\frac{dy}{y^2+1} = \frac{dx}{x}$. $\arctan y = \ln x + c$. At $(1,1)$: $\pi/4 = 0 + c$. $\arctan y = \ln x + \pi/4$. $y = \tan\!\left(\ln x + \frac{\pi}{4}\right)$.
[5 marks]
Section C — STEP-style
Hard
Q8
$(a)$ Show that $\displaystyle\int_0^{\pi/2} \sin^n x\,dx = \frac{n-1}{n}\int_0^{\pi/2}\sin^{n-2}x\,dx$ using integration by parts (the Wallis reduction formula). $(b)$ Hence evaluate $\displaystyle\int_0^{\pi/2}\sin^5 x\,dx$. $(c)$ Show that $\displaystyle\int_0^{\pi/2}\sin^{2n}x\,dx = \frac{\pi}{2}\cdot\frac{(2n)!}{4^n(n!)^2}$ and verify this for $n=2$.
$(a)$: Write $\sin^n x = \sin^{n-1}x \cdot \sin x$. Parts: $u = \sin^{n-1}x$, $\frac{dv}{dx} = \sin x$, $v = -\cos x$. $I_n = [-\sin^{n-1}x\cos x]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\sin^{n-2}x\cos^2 x\,dx$. $= 0 + (n-1)\int_0^{\pi/2}\sin^{n-2}x(1-\sin^2 x)\,dx = (n-1)(I_{n-2} - I_n)$. $nI_n = (n-1)I_{n-2}$, so $I_n = \frac{n-1}{n}I_{n-2}$. $\square$ $(b)$: $I_5 = \frac{4}{5}I_3 = \frac{4}{5}\cdot\frac{2}{3}I_1 = \frac{8}{15}\int_0^{\pi/2}\sin x\,dx = \frac{8}{15}[-\cos x]_0^{\pi/2} = \frac{8}{15}$. $(c)$: For even: $I_{2n} = \frac{2n-1}{2n}\cdot\frac{2n-3}{2n-2}\cdots\frac{1}{2}I_0 = \frac{(2n)!}{4^n(n!)^2}\cdot\frac{\pi}{2}$. For $n=2$: $I_4 = \frac{3}{4}\cdot\frac{1}{2}\cdot\frac{\pi}{2} = \frac{3\pi}{16}$. Formula: $\frac{\pi}{2}\cdot\frac{4!}{16\cdot 4} = \frac{\pi}{2}\cdot\frac{24}{64} = \frac{3\pi}{16}$ ✓.
