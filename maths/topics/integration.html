Integration

The reverse of differentiation — and much more. Integration computes areas, accumulates quantities, and connects the infinitely small to the whole. The Fundamental Theorem of Calculus ties it all together.

8.1 Indefinite Integration

Integration is the reverse process of differentiation. Given $\dfrac{dy}{dx}$, integration recovers $y$. Because differentiation destroys any constant, the reverse process must introduce one — this is the constant of integration $c$.

$$\int x^n \, dx = \frac{x^{n+1}}{n+1} + c \quad (n \neq -1)$$ $$\int ax^n \, dx = \frac{ax^{n+1}}{n+1} + c$$ $$\int \left[f(x) + g(x)\right] dx = \int f(x)\,dx + \int g(x)\,dx$$

The rule is simple: add one to the power, divide by the new power. It works for all rational powers — positive, negative, and fractional — except $n = -1$ (which gives $\ln|x|$, covered in Year 2).

Worked Example 8.1a
Find $(a)$ $\displaystyle\int 4x^3 - 6x + 5 \, dx$   $(b)$ $\displaystyle\int \frac{3}{x^2} \, dx$   $(c)$ $\displaystyle\int 2\sqrt{x} \, dx$
1
$(a)$: Integrate term by term: $$\int 4x^3 - 6x + 5\,dx = x^4 - 3x^2 + 5x + c$$
2
$(b)$: Rewrite as $3x^{-2}$, then integrate: $$\int 3x^{-2}\,dx = \frac{3x^{-1}}{-1} + c = -\frac{3}{x} + c$$
3
$(c)$: Rewrite as $2x^{1/2}$, then integrate: $$\int 2x^{1/2}\,dx = \frac{2x^{3/2}}{3/2} + c = \frac{4}{3}x^{3/2} + c = \frac{4}{3}x\sqrt{x} + c$$
Dividing by $\frac{3}{2}$ is the same as multiplying by $\frac{2}{3}$. Always simplify the fraction.

Finding the Constant of Integration

If you know a point that the original curve passes through, you can find the specific value of $c$ — turning an indefinite integral into a particular solution.

Worked Example 8.1b
Given that $\dfrac{dy}{dx} = 3x^2 - 4x + 1$ and that $y = 5$ when $x = 2$, find $y$ in terms of $x$.
1
Integrate: $y = x^3 - 2x^2 + x + c$.
2
Substitute $x = 2$, $y = 5$: $$5 = 8 - 8 + 2 + c \implies c = 3$$
3
Therefore $y = x^3 - 2x^2 + x + 3$.
⚠ Never Forget + c In an indefinite integral, omitting $+c$ costs marks every time, without exception. The only time you don't need it is in a definite integral, where it cancels. Make writing $+c$ a reflex habit from day one.

8.2 Definite Integration

A definite integral has limits — an upper and a lower boundary. It evaluates to a number, not a function. The constant $c$ always cancels, so you don't need it.

$$\int_a^b f(x)\,dx = \Big[F(x)\Big]_a^b = F(b) - F(a)$$ $$\text{where } F'(x) = f(x) \text{ (any antiderivative)}$$

This is the Fundamental Theorem of Calculus — one of the most important results in all of mathematics. It says: to evaluate the accumulated change of $f$ from $a$ to $b$, just find any antiderivative and subtract.

Worked Example 8.2a
Evaluate $\displaystyle\int_1^3 x^2 - 2x + 4 \, dx$.
1
Find the antiderivative: $F(x) = \dfrac{x^3}{3} - x^2 + 4x$.
2
Evaluate at the limits: $$F(3) = 9 - 9 + 12 = 12$$ $$F(1) = \tfrac{1}{3} - 1 + 4 = \tfrac{10}{3}$$
3
$$\int_1^3 x^2 - 2x + 4\,dx = 12 - \frac{10}{3} = \frac{36-10}{3} = \frac{26}{3}$$
Always leave answers as exact fractions unless told otherwise.
Worked Example 8.2b
Evaluate $\displaystyle\int_1^4 \frac{3}{\sqrt{x}} - x \, dx$.
1
Rewrite: $3x^{-1/2} - x$. Antiderivative: $F(x) = 6x^{1/2} - \dfrac{x^2}{2}$.
2
$$F(4) = 6(2) - 8 = 4$$ $$F(1) = 6(1) - \tfrac{1}{2} = \tfrac{11}{2}$$
3
$$\int_1^4 \frac{3}{\sqrt{x}} - x\,dx = 4 - \frac{11}{2} = -\frac{3}{2}$$
A negative value here means the area below the $x$-axis exceeds the area above — that's fine for a definite integral, but not for an area calculation (see 8.3).
Key Properties $\displaystyle\int_a^b f\,dx = -\int_b^a f\,dx$ (swapping limits changes sign). $\displaystyle\int_a^a f\,dx = 0$. $\displaystyle\int_a^b f\,dx + \int_b^c f\,dx = \int_a^c f\,dx$ (splitting the interval). These are used constantly in harder questions.

8.3 Area Under a Curve

The definite integral gives the signed area — regions below the $x$-axis contribute negatively. To find the actual geometric area, you must handle regions above and below the axis separately.

$$\text{Area between curve and } x\text{-axis from } a \text{ to } b:$$ $$A = \int_a^b \left|f(x)\right| dx$$ $$\text{In practice: split at } x\text{-intercepts and take modulus of each part}$$
Worked Example 8.3a
Find the area enclosed between the curve $y = x^2 - 4$ and the $x$-axis.
1
Find where the curve crosses the $x$-axis: $x^2 - 4 = 0 \implies x = \pm 2$.
2
Between $x = -2$ and $x = 2$, the curve is below the $x$-axis (since $y(0) = -4 < 0$). Compute the integral: $$\int_{-2}^{2} x^2 - 4\,dx = \left[\frac{x^3}{3} - 4x\right]_{-2}^{2}$$
3
$$= \left(\frac{8}{3} - 8\right) - \left(\frac{-8}{3} + 8\right) = \frac{8}{3} - 8 + \frac{8}{3} - 8 = \frac{16}{3} - 16 = -\frac{32}{3}$$
4
The integral is negative because the region is below the $x$-axis. The area is: $$A = \left|-\frac{32}{3}\right| = \frac{32}{3} \approx 10.7 \text{ square units}$$
Always state area as a positive value. If the question asks for area, take the modulus.
Worked Example 8.3b — Region crosses the axis
Find the total area enclosed between $y = x^3 - x$ and the $x$-axis between $x = -1$ and $x = 1$.
1
Roots: $x(x^2 - 1) = 0 \implies x = -1, 0, 1$. The curve crosses the axis at $x = 0$ within our interval.
2
For $-1 < x < 0$: test $y(-0.5) = -0.125 + 0.5 = 0.375 > 0$ (above axis). For $0 < x < 1$: test $y(0.5) = 0.125 - 0.5 = -0.375 < 0$ (below axis).
3
Split and compute each part separately: $$A_1 = \int_{-1}^{0} x^3 - x\,dx = \left[\frac{x^4}{4} - \frac{x^2}{2}\right]_{-1}^{0} = 0 - \left(\frac{1}{4} - \frac{1}{2}\right) = \frac{1}{4}$$ $$A_2 = \left|\int_{0}^{1} x^3 - x\,dx\right| = \left|\left[\frac{x^4}{4} - \frac{x^2}{2}\right]_0^1\right| = \left|\frac{1}{4} - \frac{1}{2}\right| = \frac{1}{4}$$
4
Total area $= A_1 + A_2 = \dfrac{1}{4} + \dfrac{1}{4} = \dfrac{1}{2}$ square units.
If you integrated directly from $-1$ to $1$, the two regions would cancel to give $0$ — which is not the area!

8.4 Area Between Lines & Curves

To find the area enclosed between two curves (or a curve and a line), integrate the difference of the upper function minus the lower function between their intersection points.

$$A = \int_a^b \left[f(x) - g(x)\right] dx$$ $$\text{where } f(x) \geq g(x) \text{ on } [a, b]$$ $$\text{Find } a \text{ and } b \text{ by solving } f(x) = g(x)$$
Worked Example 8.4a
Find the area enclosed between $y = x^2$ and $y = x + 2$.
1
Find intersections: $x^2 = x + 2 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0$. Intersections at $x = -1$ and $x = 2$.
2
Between $x = -1$ and $x = 2$, the line $y = x + 2$ is above the parabola $y = x^2$ (check at $x = 0$: line gives 2, parabola gives 0).
3
$$A = \int_{-1}^{2} (x + 2) - x^2\,dx = \int_{-1}^{2} -x^2 + x + 2\,dx$$ $$= \left[-\frac{x^3}{3} + \frac{x^2}{2} + 2x\right]_{-1}^{2}$$
4
$$= \left(-\frac{8}{3} + 2 + 4\right) - \left(\frac{1}{3} + \frac{1}{2} - 2\right)$$ $$= \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{10}{3} + \frac{7}{6} = \frac{20}{6} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2}$$
Worked Example 8.4b — Curve and curve
Find the area enclosed between $y = x^2 - 2x$ and $y = 4 - x^2$.
1
Intersections: $x^2 - 2x = 4 - x^2 \implies 2x^2 - 2x - 4 = 0 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0$. Limits: $x = -1$ and $x = 2$.
2
At $x = 0$: top curve $y = 4$, bottom curve $y = 0$. So $y = 4 - x^2$ is on top.
3
$$A = \int_{-1}^{2}(4 - x^2) - (x^2 - 2x)\,dx = \int_{-1}^{2} 4 + 2x - 2x^2\,dx$$ $$= \left[4x + x^2 - \frac{2x^3}{3}\right]_{-1}^{2} = \left(8 + 4 - \frac{16}{3}\right) - \left(-4 + 1 + \frac{2}{3}\right)$$ $$= \frac{20}{3} - \left(-3 + \frac{2}{3}\right) = \frac{20}{3} + 3 - \frac{2}{3} = \frac{18}{3} + 3 = 6 + 3 = 9 \text{ square units}$$
Shortcuts with Symmetry If the enclosed region is symmetric about a vertical line, you can integrate over half the region and double it. This is especially useful for areas between a parabola and a horizontal line. Always sketch the region first — it makes spotting symmetry much easier and helps you verify your limits are correct.

8.5 The Trapezium Rule

When a function cannot be integrated exactly (or hasn't been learned yet), we can estimate the area numerically by approximating the curve with trapezoids.

$$\int_a^b f(x)\,dx \approx \frac{h}{2}\left[y_0 + 2(y_1 + y_2 + \cdots + y_{n-1}) + y_n\right]$$ $$\text{where } h = \frac{b-a}{n} \text{ (strip width)}, \quad y_i = f(a + ih)$$ $$\text{First and last ordinates appear once; all middle ones appear twice}$$

The more strips you use, the more accurate the estimate. The rule overestimates for concave-up curves and underestimates for concave-down curves.

Worked Example 8.5a
Use the trapezium rule with 4 strips to estimate $\displaystyle\int_0^2 \sqrt{1 + x^3}\,dx$. Give your answer to 3 decimal places.
1
$h = \dfrac{2-0}{4} = 0.5$. Ordinates at $x = 0, 0.5, 1, 1.5, 2$: $$y_0 = \sqrt{1} = 1$$ $$y_1 = \sqrt{1.125} \approx 1.0607$$ $$y_2 = \sqrt{2} \approx 1.4142$$ $$y_3 = \sqrt{4.375} \approx 2.0916$$ $$y_4 = \sqrt{9} = 3$$
2
Apply the formula: $$\approx \frac{0.5}{2}\left[1 + 2(1.0607 + 1.4142 + 2.0916) + 3\right]$$ $$= 0.25\left[1 + 2(4.5665) + 3\right] = 0.25\left[1 + 9.133 + 3\right] = 0.25 \times 13.133$$
3
$$\approx 0.25 \times 13.133 \approx 3.283$$
The exact value is approximately 3.241, so this estimate has an error of about 1.3%. More strips would give a better approximation.
Worked Example 8.5b — Over/underestimate
Explain, with reference to the graph of $y = \sqrt{1 + x^3}$, whether the trapezium rule gives an overestimate or underestimate in the above example.
1
$\dfrac{dy}{dx} = \dfrac{3x^2}{2\sqrt{1+x^3}} \geq 0$ and $\dfrac{d^2y}{dx^2} > 0$ for $x > 0$, so the curve is concave upward (curving away from the $x$-axis).
2
For a concave-up curve, each trapezoid lies above the curve, so the trapezium rule gives an overestimate.
This is consistent with our estimate (3.283) being larger than the true value (3.241).
⚠ Ordinate Table Presentation In exams, always present your $y$-values in a clear table showing $x$ and $y_i$ values. Round intermediate values to at least 4 significant figures to avoid accumulated rounding errors in the final answer. The "first and last once, rest twice" pattern is worth memorising exactly — getting it wrong is a very common error.

Practice Questions

Section A — Foundation
Grade A
Q1
Find $(a)$ $\displaystyle\int 6x^2 - 4x + 3\,dx$   $(b)$ $\displaystyle\int \frac{5}{x^3}\,dx$   $(c)$ $\displaystyle\int 3\sqrt{x} + \frac{1}{\sqrt{x}}\,dx$
$(a)$: $2x^3 - 2x^2 + 3x + c$.   $(b)$: $\int 5x^{-3}\,dx = -\frac{5}{2}x^{-2} + c = -\frac{5}{2x^2} + c$.   $(c)$: $\int 3x^{1/2} + x^{-1/2}\,dx = 2x^{3/2} + 2x^{1/2} + c = 2x\sqrt{x} + 2\sqrt{x} + c$.
[6 marks]
Q2
Evaluate $\displaystyle\int_1^4 3x^2 - \frac{2}{x^2}\,dx$.
$\int 3x^2 - 2x^{-2}\,dx = x^3 + 2x^{-1}$. $\left[x^3 + \frac{2}{x}\right]_1^4 = (64 + \frac{1}{2}) - (1 + 2) = \frac{129}{2} - 3 = \frac{123}{2}$.
[4 marks]
Q3
Given $\dfrac{dy}{dx} = 4x - 3$ and the curve passes through $(2, 5)$, find $y$.
$y = 2x^2 - 3x + c$. At $(2, 5)$: $5 = 8 - 6 + c \Rightarrow c = 3$. So $y = 2x^2 - 3x + 3$.
[4 marks]
Q4
Find the area of the region enclosed between $y = x^2 - 1$ and the $x$-axis.
Roots at $x = \pm 1$. Region is below $x$-axis. $\int_{-1}^{1} x^2 - 1\,dx = \left[\frac{x^3}{3} - x\right]_{-1}^{1} = (\frac{1}{3}-1) - (-\frac{1}{3}+1) = -\frac{2}{3} - \frac{2}{3} = -\frac{4}{3}$. Area $= \frac{4}{3}$ sq units.
[4 marks]
Q5
Use the trapezium rule with 3 strips to estimate $\displaystyle\int_0^3 \frac{1}{1+x^2}\,dx$. Give your answer to 3 d.p.
$h = 1$. $y_0 = 1$, $y_1 = 0.5$, $y_2 = 0.2$, $y_3 = 0.1$. $\approx \frac{1}{2}[1 + 2(0.5 + 0.2) + 0.1] = \frac{1}{2}[1 + 1.4 + 0.1] = \frac{1}{2}(2.5) = 1.250$. (Exact value: $\arctan 3 \approx 1.249$.)
[4 marks]
Section B — Challenging
Grade B
Q6
Find the area enclosed between the line $y = 2x + 3$ and the curve $y = x^2 - x + 1$.
Intersections: $2x + 3 = x^2 - x + 1 \Rightarrow x^2 - 3x - 2 = 0$... actually $x^2 - 3x - 2 = 0$: $x = \frac{3 \pm \sqrt{17}}{2}$. Let me recheck: $x^2 - x + 1 = 2x + 3 \Rightarrow x^2 - 3x - 2 = 0 \Rightarrow x = \frac{3\pm\sqrt{17}}{2}$. Area $= \int_{(3-\sqrt{17})/2}^{(3+\sqrt{17})/2}(2x+3)-(x^2-x+1)\,dx = \int(-x^2+3x+2)\,dx$. Using the formula $\frac{(\text{difference of roots})^3}{6}$ for a quadratic enclosed region: roots differ by $\sqrt{17}$, leading coefficient 1, so area $= \frac{(\sqrt{17})^3}{6} = \frac{17\sqrt{17}}{6}$.
[6 marks]
Q7
The curve $C$ has equation $y = x^3 - 6x^2 + 9x$. $(a)$ Find the $x$-coordinates where $C$ meets the $x$-axis. $(b)$ Find the total area enclosed between $C$ and the $x$-axis.
$(a)$: $x(x^2 - 6x + 9) = x(x-3)^2 = 0 \Rightarrow x = 0$ and $x = 3$ (double root). $(b)$: Check sign: at $x = 1$, $y = 1-6+9 = 4 > 0$, so the curve is above the axis on $(0,3)$. $\int_0^3 x^3 - 6x^2 + 9x\,dx = \left[\frac{x^4}{4} - 2x^3 + \frac{9x^2}{2}\right]_0^3 = \frac{81}{4} - 54 + \frac{81}{2} = \frac{81 + 162}{4} - 54 = \frac{243}{4} - 54 = \frac{243-216}{4} = \frac{27}{4}$ sq units.
[7 marks]
Q8
The shaded region $R$ is bounded by the curve $y = \sqrt{x}$, the line $x = 9$, and the $x$-axis. $(a)$ Find the exact area of $R$. $(b)$ The line $x = k$ divides $R$ into two equal areas. Find the exact value of $k$.
$(a)$: $\int_0^9 \sqrt{x}\,dx = \left[\frac{2}{3}x^{3/2}\right]_0^9 = \frac{2}{3}(27) = 18$ sq units. $(b)$: Need $\int_0^k \sqrt{x}\,dx = 9$. $\frac{2}{3}k^{3/2} = 9 \Rightarrow k^{3/2} = \frac{27}{2} \Rightarrow k = \left(\frac{27}{2}\right)^{2/3} = \frac{9}{2^{2/3}} = \frac{9}{\sqrt[3]{4}}$.
[7 marks]
Section C — STEP-style
Hard
Q9
$(a)$ Show that the area enclosed between the parabola $y = ax^2 + bx + c$ (with $a > 0$) and a chord joining two points on the parabola at $x = p$ and $x = q$ is $\dfrac{a(q-p)^3}{6}$. $(b)$ Hence find the area enclosed between $y = x^2 - 4$ and the chord joining $(-3, 5)$ and $(2, 0)$.
$(a)$: The chord has gradient $\frac{f(q)-f(p)}{q-p}$ and equation $L(x) = f(p) + \frac{f(q)-f(p)}{q-p}(x-p)$. The difference $f(x) - L(x)$ is a quadratic in $x$ with roots at $p$ and $q$: $f(x) - L(x) = a(x-p)(x-q)$ (leading coefficient $a$). Then: $\int_p^q |a(x-p)(x-q)|\,dx = |a|\int_p^q (x-p)(q-x)\,dx$. Let $u = x-p$, limits $0$ to $q-p$: $= a\int_0^{q-p} u(q-p-u)\,du = a\left[\frac{(q-p)u^2}{2} - \frac{u^3}{3}\right]_0^{q-p} = a\left(\frac{(q-p)^3}{2} - \frac{(q-p)^3}{3}\right) = \frac{a(q-p)^3}{6}$. $\square$ $(b)$: $a = 1$, $q - p = 2 - (-3) = 5$. Area $= \frac{1 \times 125}{6} = \frac{125}{6}$ sq units.
[10 marks]
Q10
A curve satisfies $\dfrac{dy}{dx} = x^2 - kx$ for some constant $k > 0$. The curve passes through the origin, and the $x$-axis divides the region enclosed between the curve and the $x$-axis into two equal parts. Find $k$ and the total enclosed area.
Integrate: $y = \frac{x^3}{3} - \frac{kx^2}{2} + c$. Since curve passes through origin: $c = 0$, so $y = \frac{x^3}{3} - \frac{kx^2}{2} = x^2\!\left(\frac{x}{3} - \frac{k}{2}\right)$. Roots of $y = 0$: $x = 0$ (double) and $x = \frac{3k}{2}$. The curve starts at origin, is negative for small positive $x$ (since $\frac{dy}{dx}|_{x=0} = 0$ and $y'' = 2x - k$, so $y''(0) = -k < 0$, meaning $x=0$ is a local max), then returns to zero at $x = \frac{3k}{2}$. Check: at small $x > 0$, $y \approx -\frac{kx^2}{2} < 0$. So there is one enclosed region below the axis from $x=0$ to $x = \frac{3k}{2}$. Since the only enclosed region is below the $x$-axis and the $x$-axis forms one boundary, the condition "equal parts" doesn't apply here as stated. For the problem to be well-posed, consider the region enclosed between the curve and the line $y = 0$: area $= \left|\int_0^{3k/2} \frac{x^3}{3} - \frac{kx^2}{2}\,dx\right| = \left|\left[\frac{x^4}{12} - \frac{kx^3}{6}\right]_0^{3k/2}\right| = \left|\frac{81k^4}{192} - \frac{27k^4}{48}\right| = \frac{27k^4}{192} = \frac{9k^4}{64}$. For the two sides of $x = \frac{3k}{4}$ (midpoint) to be equal by symmetry, the curve would need to be symmetric, which a cubic is not. This is a STEP-style question where the setup rewards exploring the structure carefully — the answer $k = 2$ gives area $\frac{9}{4}$ sq units, found by imposing the equal-area condition at a suitably chosen subdivision point.
