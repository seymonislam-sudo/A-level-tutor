7.1 Gradient from First Principles

The gradient of a curve at a point is defined as the limit of the gradient of a chord as the chord length tends to zero. This idea — the derivative — is one of the most profound in all of mathematics.

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$ $$\text{Also written: } \frac{dy}{dx} = \lim_{\delta x \to 0}\frac{\delta y}{\delta x}$$

The notation $f'(x)$, $\dfrac{dy}{dx}$, and $\dot{y}$ all mean the same thing: the derivative of $y$ with respect to $x$. At A Level you need to be able to derive simple results from first principles, and then use the faster rules for everything else.

Worked Example 7.1a
Differentiate $f(x) = x^2$ from first principles.
1
Write $f(x+h) = (x+h)^2 = x^2 + 2xh + h^2$.
2
Form the difference quotient: $$\frac{f(x+h)-f(x)}{h} = \frac{x^2 + 2xh + h^2 - x^2}{h} = \frac{2xh + h^2}{h} = 2x + h$$
3
Take the limit as $h \to 0$: $$f'(x) = \lim_{h \to 0}(2x + h) = 2x$$
So the gradient of $y = x^2$ at any point $x$ is $2x$. At $x = 3$, for instance, the gradient is 6.
Worked Example 7.1b
Differentiate $f(x) = x^3$ from first principles.
1
Expand $(x+h)^3 = x^3 + 3x^2h + 3xh^2 + h^3$.
2
$$\frac{(x+h)^3 - x^3}{h} = \frac{3x^2h + 3xh^2 + h^3}{h} = 3x^2 + 3xh + h^2$$
3
$$\lim_{h\to 0}(3x^2 + 3xh + h^2) = 3x^2$$
Therefore $\frac{d}{dx}(x^3) = 3x^2$. The pattern $nx^{n-1}$ is emerging.
What First Principles Shows You're asked to use first principles for simple polynomials (usually $x^2$, $x^3$, or $\frac{1}{x}$) in exams. The key steps are always: expand $f(x+h)$, subtract $f(x)$, divide by $h$, cancel the $h$, then take the limit. The cancellation step is crucial — without it you'd be dividing by zero.

7.2 The Power Rule

For any power of $x$, the derivative follows a simple rule. This works for all rational powers — positive, negative, and fractional.

$$\frac{d}{dx}(x^n) = nx^{n-1}$$ $$\frac{d}{dx}(ax^n) = anx^{n-1}$$ $$\frac{d}{dx}(f + g) = f' + g' \quad \text{(sum rule)}$$ $$\frac{d}{dx}(k) = 0 \quad \text{(constants vanish)}$$

Before differentiating, always rewrite the function so every term is of the form $ax^n$. This means expanding brackets and rewriting roots and fractions as powers.

Worked Example 7.2a
Differentiate $(a)$ $y = 3x^4 - 5x^2 + 7$   $(b)$ $y = \dfrac{4}{x^3}$   $(c)$ $y = 2\sqrt{x}$   $(d)$ $y = \dfrac{x^3 - 3x}{x}$
1
$(a)$: Apply the power rule term by term: $$\frac{dy}{dx} = 12x^3 - 10x$$
The constant $+7$ differentiates to $0$.
2
$(b)$: Rewrite as $y = 4x^{-3}$, then: $$\frac{dy}{dx} = 4 \times (-3)x^{-4} = -\frac{12}{x^4}$$
3
$(c)$: Rewrite as $y = 2x^{1/2}$, then: $$\frac{dy}{dx} = 2 \times \tfrac{1}{2}x^{-1/2} = \frac{1}{\sqrt{x}}$$
4
$(d)$: Simplify first — divide each term: $y = x^2 - 3$. Then: $$\frac{dy}{dx} = 2x$$
Never try to differentiate a fraction directly — simplify or use the quotient rule.
⚠ Rewrite Before Differentiating You cannot apply the power rule to $y = \sqrt{x+1}$ or $y = (2x+3)^4$ directly — those require the chain rule (Year 2). At this stage, only differentiate expressions that are already sums of $ax^n$ terms. If you see a bracket, check whether it can be expanded first.

7.3 Tangents & Normals

The derivative at a point gives the gradient of the tangent — the straight line that just touches the curve at that point. The normal is perpendicular to the tangent at the same point.

$$\text{Gradient of tangent at } x = a: \quad m_T = f'(a)$$ $$\text{Gradient of normal at } x = a: \quad m_N = -\frac{1}{f'(a)}$$ $$\text{Equation of tangent: } y - f(a) = f'(a)(x - a)$$ $$\text{Equation of normal: } y - f(a) = -\frac{1}{f'(a)}(x - a)$$
Worked Example 7.3a
Find the equations of the tangent and normal to $y = x^3 - 4x + 1$ at the point where $x = 2$.
1
Find the $y$-coordinate: $y(2) = 8 - 8 + 1 = 1$. Point is $(2, 1)$.
2
Differentiate: $\dfrac{dy}{dx} = 3x^2 - 4$. At $x = 2$: $m_T = 3(4) - 4 = 8$.
3
Tangent: $y - 1 = 8(x - 2) \implies y = 8x - 15$.
4
Normal gradient: $m_N = -\dfrac{1}{8}$. Normal: $y - 1 = -\dfrac{1}{8}(x - 2) \implies y = -\dfrac{x}{8} + \dfrac{5}{4}$.
Or: $8y = -x + 10$, i.e. $x + 8y - 10 = 0$.
Worked Example 7.3b
The tangent to $y = x^2 + bx + c$ at the point $(1, 3)$ has gradient $4$. Find $b$ and $c$.
1
Since $(1, 3)$ is on the curve: $1 + b + c = 3 \implies b + c = 2 \quad \cdots (1)$
2
$\dfrac{dy}{dx} = 2x + b$. At $x = 1$, gradient $= 4$: $2 + b = 4 \implies b = 2$.
3
From (1): $c = 2 - b = 0$. So $b = 2$, $c = 0$, and $y = x^2 + 2x$.

7.4 Stationary Points

A stationary point is where the gradient is zero — the tangent is horizontal. These are the peaks, troughs, and saddle points of a curve.

$$\text{Stationary point where } \frac{dy}{dx} = 0$$ $$\text{Local maximum: gradient changes } + \to 0 \to -$$ $$\text{Local minimum: gradient changes } - \to 0 \to +$$ $$\text{Point of inflection: gradient changes } + \to 0 \to + \text{ (or } - \to 0 \to -)$$

Nature of Stationary Points: Sign Analysis

To determine the type of a stationary point, check the sign of $\dfrac{dy}{dx}$ just before and just after it. A table of signs is the clearest way to present this in exams.

Worked Example 7.4a
Find and classify all stationary points of $y = 2x^3 - 9x^2 + 12x - 3$.
1
$\dfrac{dy}{dx} = 6x^2 - 18x + 12 = 6(x^2 - 3x + 2) = 6(x-1)(x-2)$. Set to zero: $x = 1$ or $x = 2$.
2
Find $y$-coordinates: $y(1) = 2 - 9 + 12 - 3 = 2$.   $y(2) = 16 - 36 + 24 - 3 = 1$.
3
Sign table for $6(x-1)(x-2)$:
$x < 1$: both factors negative → product positive (+). Between 1 and 2: $(x-1) > 0$, $(x-2) < 0$ → negative (−). $x > 2$: both positive → positive (+).
4
At $x = 1$: gradient changes $+ \to -$, so $(1, 2)$ is a local maximum. At $x = 2$: gradient changes $- \to +$, so $(2, 1)$ is a local minimum.
Exam Technique Always show a sign diagram or state explicitly: "gradient changes from positive to negative, therefore local maximum." Just writing "because $f''(x) < 0$" without the second derivative test introduced is not enough context — use whichever method you've been taught and show your working clearly.

7.5 Increasing & Decreasing Functions

The sign of the derivative tells us whether a function is increasing or decreasing on any interval.

$$f'(x) > 0 \implies f \text{ is increasing on that interval}$$ $$f'(x) < 0 \implies f \text{ is decreasing on that interval}$$ $$f'(x) = 0 \implies \text{stationary point (or point of inflection)}$$
Worked Example 7.5a
Find the values of $x$ for which $f(x) = x^3 - 3x^2 - 9x + 5$ is increasing.
1
$f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)$.
2
$f'(x) > 0$ when $(x-3)(x+1) > 0$, i.e. when both factors have the same sign. This occurs when $x < -1$ or $x > 3$.
3
The function is increasing for $x < -1$ and $x > 3$.
It is decreasing for $-1 < x < 3$, since $f'(x) < 0$ there.
Worked Example 7.5b — Optimisation
A farmer has 120 m of fencing to enclose a rectangular pen against a straight wall (the wall forms one side). Find the dimensions that maximise the area.
1
Let the width perpendicular to the wall be $x$ m. Two widths and one length use the fencing: $2x + l = 120 \implies l = 120 - 2x$.
2
Area: $A = xl = x(120 - 2x) = 120x - 2x^2$.
3
$\dfrac{dA}{dx} = 120 - 4x = 0 \implies x = 30$.
4
Check: $\dfrac{d^2A}{dx^2} = -4 < 0$, confirming a maximum. Dimensions: $30\text{ m} \times 60\text{ m}$, area $= 1800\text{ m}^2$.
Always check the nature of the stationary point and state the final answer with units.

7.6 The Second Derivative

Differentiating $f'(x)$ again gives the second derivative $f''(x)$, written $\dfrac{d^2y}{dx^2}$. It measures the rate of change of the gradient — in other words, the concavity of the curve.

$$f''(x) = \frac{d^2y}{dx^2} = \frac{d}{dx}\!\left(\frac{dy}{dx}\right)$$ $$f''(a) > 0 \implies \text{concave up (U-shape) at } a \implies \text{local minimum}$$ $$f''(a) < 0 \implies \text{concave down (∩-shape) at } a \implies \text{local maximum}$$ $$f''(a) = 0 \implies \text{inconclusive — use sign diagram instead}$$

The second derivative test is a quick alternative to the sign diagram for classifying stationary points. If $f'(a) = 0$ and $f''(a) \neq 0$, you can classify immediately.

Worked Example 7.6a
Use the second derivative to classify the stationary points of $y = x^4 - 8x^2 + 3$.
1
$\dfrac{dy}{dx} = 4x^3 - 16x = 4x(x^2 - 4) = 4x(x-2)(x+2)$. Stationary points at $x = 0, \pm 2$.
2
$\dfrac{d^2y}{dx^2} = 12x^2 - 16$.
3
At $x = 0$: $f''(0) = -16 < 0$ → local maximum. $y(0) = 3$. At $x = 2$: $f''(2) = 48 - 16 = 32 > 0$ → local minimum. $y(2) = 16 - 32 + 3 = -13$. At $x = -2$: $f''(-2) = 32 > 0$ → local minimum. $y(-2) = -13$.
Worked Example 7.6b — Second Derivative in Context
The displacement of a particle is $s = t^3 - 6t^2 + 9t$, where $t \geq 0$ is in seconds and $s$ in metres. Find when the particle is stationary and determine whether it is a maximum or minimum displacement at each time.
1
Velocity: $v = \dfrac{ds}{dt} = 3t^2 - 12t + 9 = 3(t-1)(t-3)$. Particle stationary when $v = 0$: $t = 1$ and $t = 3$.
2
Acceleration: $a = \dfrac{d^2s}{dt^2} = 6t - 12$. At $t = 1$: $a = -6 < 0$ → local maximum displacement. $s(1) = 1 - 6 + 9 = 4$ m. At $t = 3$: $a = 6 > 0$ → local minimum displacement. $s(3) = 27 - 54 + 27 = 0$ m.
3
The particle reaches a furthest point of 4 m at $t = 1$ s, then returns to the origin at $t = 3$ s.
In kinematics, $\frac{ds}{dt}$ is velocity and $\frac{d^2s}{dt^2}$ is acceleration — the same maths has a physical meaning.
⚠ When $f''(a) = 0$ If the second derivative is zero at a stationary point, the test is inconclusive — you cannot conclude anything. You must fall back on the sign diagram. For example, $y = x^4$ has $f'(0) = 0$ and $f''(0) = 0$, but it's still a minimum. And $y = x^3$ also has $f'(0) = f''(0) = 0$, but it's a point of inflection.

Practice Questions

Section A — Foundation
Grade A
Q1
Differentiate with respect to $x$: $(a)$ $y = 5x^3 - 4x + 2$   $(b)$ $y = \dfrac{3}{x^2}$   $(c)$ $y = 4\sqrt{x} - \dfrac{2}{x}$
$(a)$: $\frac{dy}{dx} = 15x^2 - 4$.   $(b)$: $y = 3x^{-2}$, so $\frac{dy}{dx} = -6x^{-3} = -\frac{6}{x^3}$.   $(c)$: $y = 4x^{1/2} - 2x^{-1}$, so $\frac{dy}{dx} = 2x^{-1/2} + 2x^{-2} = \frac{2}{\sqrt{x}} + \frac{2}{x^2}$.
[6 marks]
Q2
Find the gradient of the curve $y = x^3 - 5x + 3$ at the point $(2, 1)$.
$\frac{dy}{dx} = 3x^2 - 5$. At $x = 2$: gradient $= 12 - 5 = 7$.
[2 marks]
Q3
Find the equation of the tangent to $y = x^2 - 3x + 4$ at the point where $x = 3$.
$y(3) = 9 - 9 + 4 = 4$. Point: $(3, 4)$. $\frac{dy}{dx} = 2x - 3$. At $x = 3$: $m = 3$. Tangent: $y - 4 = 3(x - 3) \implies y = 3x - 5$.
[4 marks]
Q4
Differentiate $f(x) = x^2$ from first principles, showing all steps clearly.
$f'(x) = \lim_{h\to 0}\frac{(x+h)^2 - x^2}{h} = \lim_{h\to 0}\frac{2xh+h^2}{h} = \lim_{h\to 0}(2x+h) = 2x$.
[3 marks]
Q5
Find the stationary points of $y = x^3 - 3x$ and determine their nature using a sign diagram.
$\frac{dy}{dx} = 3x^2 - 3 = 3(x-1)(x+1) = 0 \Rightarrow x = \pm 1$. At $x = -1$: gradient changes $+ \to -$ → local max $(-1, 2)$. At $x = 1$: gradient changes $- \to +$ → local min $(1, -2)$.
[5 marks]
Section B — Challenging
Grade B
Q6
The curve $C$ has equation $y = x^3 + ax^2 + bx - 4$. The point $P(2, 0)$ lies on $C$, and the gradient of $C$ at $P$ is $-1$. Find $a$ and $b$, then find the coordinates of any other stationary points.
$P$ on $C$: $8 + 4a + 2b - 4 = 0 \Rightarrow 4a + 2b = -4 \Rightarrow 2a + b = -2 \cdots(1)$. Gradient: $\frac{dy}{dx} = 3x^2 + 2ax + b$. At $x=2$: $12 + 4a + b = -1 \Rightarrow 4a + b = -13 \cdots(2)$. (2)−(1): $2a = -11 \Rightarrow a = -5.5$, $b = -2 - 2(-5.5) = 9$. Then $\frac{dy}{dx} = 3x^2 - 11x + 9 - 4 = 0$... actually $\frac{dy}{dx} = 3x^2 - 11x + 9$. Set to 0: $x = \frac{11 \pm \sqrt{121-108}}{6} = \frac{11 \pm \sqrt{13}}{6}$. Two further stationary points at those $x$-values.
[8 marks]
Q7
A closed cylindrical tin has volume $V = 250\text{ cm}^3$. Show that the total surface area is $S = 2\pi r^2 + \dfrac{500}{r}$, and find the value of $r$ that minimises $S$.
$V = \pi r^2 h = 250 \Rightarrow h = \frac{250}{\pi r^2}$. $S = 2\pi r^2 + 2\pi r h = 2\pi r^2 + 2\pi r \cdot \frac{250}{\pi r^2} = 2\pi r^2 + \frac{500}{r}$. $\frac{dS}{dr} = 4\pi r - \frac{500}{r^2} = 0 \Rightarrow 4\pi r^3 = 500 \Rightarrow r = \left(\frac{125}{\pi}\right)^{1/3} \approx 3.41$ cm. $\frac{d^2S}{dr^2} = 4\pi + \frac{1000}{r^3} > 0$ ✓ minimum.
[7 marks]
Q8
Find the equation of the normal to $y = \dfrac{8}{x^2} + x$ at the point where $x = 2$. Find also where this normal meets the curve again.
$y(2) = 2 + 2 = 4$. $\frac{dy}{dx} = -16x^{-3} + 1$. At $x=2$: $m_T = -2 + 1 = -1$. Normal gradient $= 1$. Normal: $y - 4 = x - 2 \Rightarrow y = x + 2$. Intersect with curve: $\frac{8}{x^2} + x = x + 2 \Rightarrow \frac{8}{x^2} = 2 \Rightarrow x^2 = 4 \Rightarrow x = \pm 2$. The other intersection is at $x = -2$: $y = 0$. Point $(-2, 0)$.
[7 marks]
Section C — STEP-style
Hard
Q9
Prove from first principles that $\dfrac{d}{dx}\!\left(\dfrac{1}{x}\right) = -\dfrac{1}{x^2}$. Hence, by writing $\sqrt{x} = \dfrac{x}{x^{1/2}}$ or otherwise, find $\dfrac{d}{dx}(\sqrt{x})$ from first principles.
$\frac{d}{dx}\left(\frac{1}{x}\right) = \lim_{h\to 0}\frac{\frac{1}{x+h}-\frac{1}{x}}{h} = \lim_{h\to 0}\frac{\frac{x-(x+h)}{x(x+h)}}{h} = \lim_{h\to 0}\frac{-1}{x(x+h)} = -\frac{1}{x^2}$. $\square$ For $\sqrt{x}$: $\frac{d}{dx}(\sqrt{x}) = \lim_{h\to 0}\frac{\sqrt{x+h}-\sqrt{x}}{h}$. Rationalise: multiply by $\frac{\sqrt{x+h}+\sqrt{x}}{\sqrt{x+h}+\sqrt{x}}$: $= \lim_{h\to 0}\frac{(x+h)-x}{h(\sqrt{x+h}+\sqrt{x})} = \lim_{h\to 0}\frac{1}{\sqrt{x+h}+\sqrt{x}} = \frac{1}{2\sqrt{x}}$. $\square$
[7 marks]
Q10
A curve has equation $y = x^3 - kx$ for constant $k > 0$. Show that the tangent at the point $(a, a^3 - ka)$ meets the curve again at the point $(-2a, -8a^3 + 2ka)$. Hence find the condition on $k$ such that the tangent at $(a, a^3-ka)$ is also the tangent to the curve at the second intersection point.
Tangent at $(a, a^3-ka)$: gradient $= 3a^2 - k$. Equation: $y - (a^3-ka) = (3a^2-k)(x-a) \Rightarrow y = (3a^2-k)x - 2a^3$. Substitute into $y = x^3 - kx$: $x^3 - kx = (3a^2-k)x - 2a^3 \Rightarrow x^3 - 3a^2 x + 2a^3 = 0$. Factor out $(x-a)$: $(x-a)^2(x+2a) = 0$. So the other intersection is $x = -2a$, giving $y = -8a^3 + 2ka$ ✓. For the tangent to also be tangent at $x = -2a$, we need $x = -2a$ to also be a double root, i.e. $(x+2a)^2$ divides $x^3 - 3a^2x + 2a^3$. But $(x-a)^2(x+2a)$ already accounts for all three roots. This can only happen if $a = -2a$, i.e. $a = 0$, which is trivial. Instead, the condition is that the gradient of the curve at $x = -2a$ equals the gradient of the tangent: $3(-2a)^2 - k = 3a^2 - k \Rightarrow 12a^2 = 3a^2 \Rightarrow 9a^2 = 0 \Rightarrow a = 0$. So for $k > 0$ there is no non-trivial point where a single tangent line is tangent at two distinct points on this curve.
