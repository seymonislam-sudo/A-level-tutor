Variables & Data Types

A variable is a named memory location that stores a value which can change during program execution. A constant is similar but its value cannot change once set.

```

AQA requires you to know the following primitive types:

TypeDescriptionExample

BOOLEANTrue or False onlyTRUE, FALSE
CHARACTERSingle character'A', '3', '!'
STRINGSequence of characters"Hello", "AQA"

Casting (Type Conversion)

You can convert between types using casting functions. AQA pseudocode uses:

``` int(“42”) # String to integer → 42 str(42) # Integer to string → “42” float(“3.14”) # String to real → 3.14 bool(0) # Integer to boolean → False
```
Key Point String concatenation uses the + operator: "Hello" + " " + "World" gives "Hello World". You cannot concatenate a string with an integer directly — you must cast first.
Worked Example

1
Get input — user input is always a string, so cast to integer.
age = int(input("Enter your age: "))
2
Compare using a selection statement.
if age >= 18:
3
Output appropriate message in each branch.
``` age = int(input(“Enter your age: “)) if age >= 18: print(“You are old enough to vote.”) else: print(“You are not old enough to vote.”)

Sequence, Selection & Iteration

```

Sequence

Statements execute one after another in order. The simplest form of program flow.

Selection

Selection uses conditions to choose which code to execute. AQA uses IF...THEN...ELSE...ENDIF in pseudocode.

``` # Simple IF if score >= 70: grade = “A” elif score >= 60: grade = “B” elif score >= 50: grade = “C” else: grade = “U”
```

Iteration

Iteration repeats a block of code. AQA distinguishes two types:

Count-controlled Use a FOR loop when you know exactly how many times to repeat. FOR i ← 1 TO 10 in pseudocode.
Condition-controlled Use WHILE or REPEAT...UNTIL when the number of iterations depends on a condition. WHILE checks before each iteration; REPEAT...UNTIL checks after (always runs at least once).
``` # Count-controlled for i in range(1, 11): # 1 to 10 inclusive print(i) # Condition-controlled (WHILE) total = 0 while total < 100: total += int(input(“Enter a number: “))
```
Worked Example
Write a program that keeps asking the user for a password until they enter the correct one ("letmein"). Count and output the number of attempts.
1
Use a REPEAT...UNTIL style loop — we need it to run at least once. In Python, use a while True with a break.
2
Increment a counter each attempt.
3
Break when the correct password is entered.
``` attempts = 0 while True: password = input(“Enter password: “) attempts += 1 if password == “letmein”: break print(“Access granted after”, attempts, “attempt(s).”)

Subroutines

A subroutine is a named block of code that performs a specific task. AQA distinguishes between:

TypeDescriptionReturns a value?
ProcedurePerforms a task, no return valueNo
FunctionPerforms a task and returns a valueYes
```
``` # Procedure — no return value def greet(name): print(“Hello, “ + name) # Function — returns a value def square(n): return n * n result = square(5) # result = 25
```
Why use subroutines? Subroutines make code modular, reusable, and easier to test. Each subroutine should perform one clearly defined task. This is a key principle of structured programming.
Worked Example
Write a function that takes two integers and returns the larger of the two.
``` def larger(a, b): if a > b: return a else: return b print(larger(7, 12)) # Outputs: 12

Scope & Parameters

```

Local vs Global Variables

Local variables are declared inside a subroutine and can only be used within it. Global variables are declared outside all subroutines and can be accessed anywhere.

Common Mistake In Python, you can read a global variable inside a function, but to assign to it you must declare global x first. AQA exam questions often test this distinction.
``` total = 0 # Global variable def add_to_total(amount): global total # Declare intent to modify global total += amount def calculate(): result = 42 # Local variable — only exists here return result
```

Passing Parameters

Parameters are values passed into a subroutine. AQA requires you to understand two methods:

MethodDescriptionEffect on original
By valueA copy of the value is passedOriginal unchanged
By referenceThe memory address is passedOriginal can be changed
AQA Exam Note Python passes immutable types (integers, strings) by value and mutable types (lists) by reference. You may be asked to trace through code and identify whether a variable changes after a subroutine call.
```

Object-Oriented Programming

OOP organises code around objects — instances of classes. Each object has attributes (data) and methods (behaviours).

```

Key OOP Concepts

ConceptDefinition
ClassA blueprint/template for creating objects
ObjectAn instance of a class
AttributeA variable belonging to an object
MethodA function belonging to a class
ConstructorSpecial method called when an object is created (__init__)
EncapsulationHiding internal data; access via public methods only
InheritanceA subclass inherits attributes and methods from a parent class
PolymorphismDifferent classes can share the same method name with different behaviour
``` class Animal: def **init**(self, name, sound): self.name = name # Attribute self.sound = sound ``` def speak(self): # Method print(self.name + " says " + self.sound) ``` class Dog(Animal): # Inheritance def **init**(self, name): super().**init**(name, “Woof”) ``` def fetch(self): print(self.name + " fetches the ball!") ``` rex = Dog(“Rex”) rex.speak() # Rex says Woof rex.fetch() # Rex fetches the ball!
```
Worked Example
Design a class called BankAccount with attributes for account number and balance. Include methods to deposit, withdraw (with a check for sufficient funds), and display the balance.
``` class BankAccount: def **init**(self, account_number, balance=0): self.account_number = account_number self.__balance = balance # Private (encapsulation) ``` def deposit(self, amount): self.__balance += amount def withdraw(self, amount): if amount <= self.__balance: self.__balance -= amount else: print("Insufficient funds.") def get_balance(self): return self.__balance ``` acc = BankAccount(“12345678”, 100) acc.deposit(50) acc.withdraw(30) print(acc.get_balance()) # 120
