1.1 Laws of Indices

Indices (or powers/exponents) describe repeated multiplication. The rules for manipulating them are essential throughout all of A Level Maths.

$$a^m \times a^n = a^{m+n}$$ $$a^m \div a^n = a^{m-n}$$ $$(a^m)^n = a^{mn}$$ $$a^0 = 1 \quad (a \neq 0)$$ $$a^{-n} = \dfrac{1}{a^n}$$ $$a^{\frac{1}{n}} = \sqrt[n]{a}$$ $$a^{\frac{m}{n}} = \left(\sqrt[n]{a}\right)^m = \sqrt[n]{a^m}$$
Key Point The rules above only apply when the base is the same. You cannot simplify $a^3 \times b^2$ using these rules.

Worked Examples

Worked Example 1.1a
Simplify: $\dfrac{x^5 \times x^{-2}}{x^3}$
1
Multiply terms in the numerator: $x^5 \times x^{-2} = x^{5+(-2)} = x^3$
Add the powers when multiplying same base.
2
Divide: $\dfrac{x^3}{x^3} = x^{3-3} = x^0 = 1$
Subtract powers when dividing. Anything to the power 0 is 1.
Worked Example 1.1b
Evaluate: $27^{-\frac{2}{3}}$
1
$27^{\frac{1}{3}} = \sqrt[3]{27} = 3$
The denominator of the fractional power is the root.
2
Apply the numerator: $3^2 = 9$
The numerator is the power.
3
Apply the negative: $27^{-\frac{2}{3}} = \dfrac{1}{9}$
Negative power means take the reciprocal.

1.2 Surds

A surd is an irrational root that cannot be simplified to a rational number. We leave answers in surd form to keep them exact.

$$\sqrt{ab} = \sqrt{a}\cdot\sqrt{b}$$ $$\sqrt{\dfrac{a}{b}} = \dfrac{\sqrt{a}}{\sqrt{b}}$$ $$(a + \sqrt{b})(a - \sqrt{b}) = a^2 - b$$

Simplifying Surds

Find the largest square factor of the number under the root.

Worked Example 1.2a
Simplify $\sqrt{72}$
1
Find the largest square factor of 72: $72 = 36 \times 2$
2
$\sqrt{72} = \sqrt{36 \times 2} = \sqrt{36}\cdot\sqrt{2} = 6\sqrt{2}$

Rationalising the Denominator

We never leave a surd in the denominator. Multiply top and bottom by the conjugate.

Worked Example 1.2b
Rationalise the denominator of $\dfrac{5}{3+\sqrt{2}}$
1
Multiply by the conjugate $(3 - \sqrt{2})$: $\dfrac{5}{3+\sqrt{2}} \times \dfrac{3-\sqrt{2}}{3-\sqrt{2}}$
2
Denominator by difference of two squares: $(3+\sqrt{2})(3-\sqrt{2}) = 9 - 2 = 7$
3
$$\dfrac{5(3-\sqrt{2})}{7} = \dfrac{15 - 5\sqrt{2}}{7}$$
⚠ Common Mistake $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$. Example: $\sqrt{9+16} = 5$, but $\sqrt{9} + \sqrt{16} = 7$. Never split a sum under a root.

1.3 Quadratics

A quadratic is any expression $ax^2 + bx + c$ where $a \neq 0$. You need to solve, factorise, and complete the square.

The Quadratic Formula

$$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Completing the Square

$$ax^2 + bx + c = a\left(x + \dfrac{b}{2a}\right)^2 - \dfrac{b^2}{4a} + c$$
Worked Example 1.3a
Complete the square for $2x^2 - 8x + 3$
1
Factor out 2: $2(x^2 - 4x) + 3$
2
Half of $-4$ is $-2$, $(-2)^2 = 4$: $2\left[(x-2)^2 - 4\right] + 3$
3
$2(x-2)^2 - 8 + 3 = 2(x-2)^2 - 5$
Vertex at $(2, -5)$.

The Discriminant

$$\Delta = b^2 - 4ac$$ $$\Delta > 0 \implies \text{two distinct real roots}$$ $$\Delta = 0 \implies \text{one repeated real root}$$ $$\Delta < 0 \implies \text{no real roots}$$

1.4 Simultaneous Equations

At A Level you solve simultaneous equations where one or both are quadratic, using substitution.

Worked Example 1.4a
Solve: $y = x + 3$ and $x^2 + y^2 = 29$
1
Substitute $y = x + 3$: $x^2 + (x+3)^2 = 29$
2
Expand: $2x^2 + 6x - 20 = 0$
3
Factorise: $(x+5)(x-2) = 0$
4
$x = -5 \Rightarrow y = -2$    $x = 2 \Rightarrow y = 5$
Exam Technique Always find both coordinates and write them as pairs. Leaving just $x$ values loses marks.

1.5 Inequalities

Linear inequalities work like equations except: multiplying or dividing by a negative flips the sign. For quadratic inequalities, sketch the curve first.

Worked Example 1.5a
Solve $x^2 - 5x + 4 < 0$
1
Factorise: $(x-1)(x-4) = 0$, roots $x = 1$ and $x = 4$
2
Parabola opens upward, so below zero between the roots.
3
Solution: $1 < x < 4$
⚠ Common Mistake Never write $4 > x > 1$. Always write $x$ increasing left to right: $1 < x < 4$.

1.6 Graph Transformations

$$y = f(x) + a \;\longrightarrow\; \text{translation } \begin{pmatrix}0\\a\end{pmatrix}$$ $$y = f(x + a) \;\longrightarrow\; \text{translation } \begin{pmatrix}-a\\0\end{pmatrix}$$ $$y = af(x) \;\longrightarrow\; \text{vertical stretch, s.f. } a$$ $$y = f(ax) \;\longrightarrow\; \text{horizontal stretch, s.f. } \tfrac{1}{a}$$ $$y = -f(x) \;\longrightarrow\; \text{reflection in } x\text{-axis}$$ $$y = f(-x) \;\longrightarrow\; \text{reflection in } y\text{-axis}$$
Memory Trick Changes inside the brackets affect $x$ and do the opposite to what you'd expect: $f(x + 3)$ shifts left by 3.
Worked Example 1.6a
$y = f(x)$ has a maximum at $(2, 5)$. Find the maximum after $y = 3f(x - 1)$.
1
$f(x-1)$: translate $+1$ in $x$. Point: $(3, 5)$.
2
$3f(x-1)$: vertical stretch $\times 3$. Point: $(3, 15)$.
3
New maximum: $\mathbf{(3,\, 15)}$

Practice Questions

Section A — Foundation
Grade A
Q1
Simplify: $\dfrac{(2x^3)^2 \times x^{-4}}{4x}$
$\dfrac{4x^6 \times x^{-4}}{4x} = \dfrac{4x^2}{4x} = x$
[3 marks]
Q2
Rationalise the denominator: $\dfrac{6}{2 - \sqrt{3}}$
Multiply by $\dfrac{2+\sqrt{3}}{2+\sqrt{3}}$: $\dfrac{6(2+\sqrt{3})}{4-3} = 12 + 6\sqrt{3}$
[3 marks]
Q3
Complete the square for $x^2 + 6x - 7$ and find the minimum value.
$(x+3)^2 - 16$. Minimum $-16$ at $x = -3$.
[4 marks]
Q4
Solve simultaneously: $y = 2x - 1$ and $y = x^2 - 3x + 5$.
$(x-2)(x-3) = 0$. Solutions: $(2, 3)$ and $(3, 5)$.
[5 marks]
Q5
Find the set of values of $x$ satisfying $x^2 - x - 12 \leq 0$.
$(x-4)(x+3) \leq 0$. Solution: $-3 \leq x \leq 4$.
[4 marks]
Section B — Challenging
Grade B
Q6
Show that $\dfrac{(9x)^{\frac{1}{2}} \cdot 27x^2}{3^4 \cdot x^{-\frac{1}{2}}}$ simplifies to $x^k$ and find $k$.
Numerator: $3^4 x^{\frac{5}{2}}$. Divided by $3^4 x^{-\frac{1}{2}}$. Result: $x^3$. So $k = 3$.
[5 marks]
Q7
Find the values of $k$ for which $kx^2 + 4x + k = 0$ has two distinct real solutions.
$16 - 4k^2 > 0 \Rightarrow k^2 < 4$, so $-2 < k < 2$, $k \neq 0$.
[5 marks]
Q8
$y = f(x)$ passes through $(4, 7)$. Find the corresponding point on $y = \tfrac{1}{2}f(2x + 6)$.
$2x + 6 = 4 \Rightarrow x = -1$. $y = \tfrac{1}{2} \times 7 = 3.5$. Point: $(-1,\, 3.5)$.
[4 marks]
Section C — STEP-style
Hard
Q9
Prove that $x^2 + 4x + 7 > 0$ for all real $x$. Hence prove $\dfrac{x^2 + 4x + 7}{2x^2 + 3} > 0$ for all real $x$.
$(x+2)^2 + 3 \geq 3 > 0$. Also $2x^2 + 3 \geq 3 > 0$. Both always positive, so quotient always positive. $\square$
[6 marks]
Q10
Given $\sqrt{a} + \sqrt{b} = \sqrt{c}$ for positive integers $a, b, c$, show that at least one of $a, b, c$ is divisible by 4, or two are equal.
Square both sides: $a + b + 2\sqrt{ab} = c$. So $2\sqrt{ab}$ is an integer, meaning $ab$ is a perfect square. Writing $a = m^2 s$, $b = n^2 s$ (square-free $s$) gives $c = (m+n)^2 s$. Considering cases modulo 4 establishes the result.
[8 marks]
