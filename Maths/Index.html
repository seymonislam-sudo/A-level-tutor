1.1 Laws of Indices

```

Indices (or powers/exponents) describe repeated multiplication. The rules for manipulating them are essential throughout all of A Level Maths.

$$a^m \times a^n = a^{m+n}$$ $$a^m \div a^n = a^{m-n}$$ $$(a^m)^n = a^{mn}$$ $$a^0 = 1 \quad (a \neq 0)$$ $$a^{-n} = \dfrac{1}{a^n}$$ $$a^{\frac{1}{n}} = \sqrt[n]{a}$$ $$a^{\frac{m}{n}} = \left(\sqrt[n]{a}\right)^m = \sqrt[n]{a^m}$$
Key Point The rules above only apply when the base is the same. You cannot simplify $a^3 \times b^2$ using these rules — the bases $a$ and $b$ are different.

Worked Examples

Worked Example 1.1a
Simplify: $\dfrac{x^5 \times x^{-2}}{x^3}$
1
Multiply terms in the numerator: $x^5 \times x^{-2} = x^{5+(-2)} = x^3$
Add the powers when multiplying same base.
2
Divide: $\dfrac{x^3}{x^3} = x^{3-3} = x^0 = 1$
Subtract powers when dividing. Anything to the power 0 is 1.
Worked Example 1.1b
Evaluate: $27^{-\frac{2}{3}}$
1
Deal with the denominator of the fraction first: $27^{\frac{1}{3}} = \sqrt[3]{27} = 3$
The denominator of the fractional power is the root.
2
Apply the numerator: $3^2 = 9$
The numerator is the power.
3
Apply the negative: $27^{-\frac{2}{3}} = \dfrac{1}{9}$
Negative power means take the reciprocal.
```

1.2 Surds

```

A surd is an irrational root that cannot be simplified to a rational number — for example $\sqrt{2}$, $\sqrt{5}$, $\sqrt{12}$. We leave answers in surd form to keep them exact.

$$\sqrt{ab} = \sqrt{a}\cdot\sqrt{b}$$ $$\sqrt{\dfrac{a}{b}} = \dfrac{\sqrt{a}}{\sqrt{b}}$$ $$(a + \sqrt{b})(a - \sqrt{b}) = a^2 - b$$

Simplifying Surds

To simplify a surd, find the largest square factor of the number under the root.

Worked Example 1.2a
Simplify $\sqrt{72}$
1
Find the largest square factor of 72: $72 = 36 \times 2$
2
$\sqrt{72} = \sqrt{36 \times 2} = \sqrt{36}\cdot\sqrt{2} = 6\sqrt{2}$

Rationalising the Denominator

We never leave a surd in the denominator of a fraction. We rationalise it by multiplying top and bottom by a suitable expression.

Worked Example 1.2b
Rationalise the denominator of $\dfrac{5}{3+\sqrt{2}}$
1
Multiply numerator and denominator by the conjugate: $(3 - \sqrt{2})$ $$\dfrac{5}{3+\sqrt{2}} \times \dfrac{3-\sqrt{2}}{3-\sqrt{2}}$$
2
Expand the denominator using the difference of two squares: $(3+\sqrt{2})(3-\sqrt{2}) = 9 - 2 = 7$
3
$$\dfrac{5(3-\sqrt{2})}{7} = \dfrac{15 - 5\sqrt{2}}{7}$$
⚠ Common Mistake $\sqrt{a+b} \neq \sqrt{a} + \sqrt{b}$. For example, $\sqrt{9+16} = \sqrt{25} = 5$, but $\sqrt{9} + \sqrt{16} = 3 + 4 = 7$. Never split a sum under a square root.
```

1.3 Quadratics

```

A quadratic is any expression of the form $ax^2 + bx + c$ where $a \neq 0$. You need to be able to solve, factorise, and complete the square.

Solving by Factorisation

If you can write the quadratic as a product of two brackets, set each to zero.

The Quadratic Formula

Always works — use it when factorisation is not obvious.

$$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Completing the Square

$$ax^2 + bx + c = a\left(x + \dfrac{b}{2a}\right)^2 - \dfrac{b^2}{4a} + c$$

Completing the square is vital for finding the vertex of a parabola and for transformations.

Worked Example 1.3a
Complete the square for $2x^2 - 8x + 3$
1
Factor out the coefficient of $x^2$: $2(x^2 - 4x) + 3$
2
Complete the square inside the bracket. Take half of $-4$, square it: $(-2)^2 = 4$ $$2\left[(x-2)^2 - 4\right] + 3$$
3
Expand and simplify: $$2(x-2)^2 - 8 + 3 = 2(x-2)^2 - 5$$
Vertex is at $(2, -5)$.

The Discriminant

The expression $b^2 - 4ac$ tells us about the roots of a quadratic:

$$\Delta = b^2 - 4ac$$ $$\Delta > 0 \implies \text{two distinct real roots}$$ $$\Delta = 0 \implies \text{one repeated real root}$$ $$\Delta < 0 \implies \text{no real roots}$$
```

1.4 Simultaneous Equations

```

At A Level you need to solve simultaneous equations where one or both are quadratic — this involves substitution.

Worked Example 1.4a
Solve simultaneously: $y = x + 3$ and $x^2 + y^2 = 29$
1
Substitute $y = x + 3$ into the second equation: $$x^2 + (x+3)^2 = 29$$
2
Expand: $x^2 + x^2 + 6x + 9 = 29 \implies 2x^2 + 6x - 20 = 0$
3
Simplify and factorise: $x^2 + 3x - 10 = 0 \implies (x+5)(x-2) = 0$
4
So $x = -5$ or $x = 2$. Find $y$ using $y = x + 3$: $$x = -5 \implies y = -2 \qquad x = 2 \implies y = 5$$
Exam Technique Always find both coordinates of each solution and write them as coordinate pairs. Leaving answers as just $x$ values will lose marks.
```

1.5 Inequalities

```

Linear inequalities work like equations, except: when you multiply or divide by a negative number, you must flip the inequality sign.

Quadratic Inequalities

For quadratic inequalities, always sketch the curve first to see which regions satisfy the inequality.

Worked Example 1.5a
Solve $x^2 - 5x + 4 < 0$
1
Factorise: $(x-1)(x-4) = 0$, so roots are $x = 1$ and $x = 4$
2
The parabola opens upward ($a = 1 > 0$), so it is below zero between the roots.
3
Solution: $1 < x < 4$
For $ax^2 + bx + c > 0$ with two roots: the answer is the two outer regions. For $< 0$: the middle region.
⚠ Common Mistake Never write $4 > x > 1$ for the solution above — always write with $x$ increasing from left to right: $1 < x < 4$.
```

1.6 Graph Transformations

```

Given the graph of $y = f(x)$, you need to know how the following transformations change it:

$$y = f(x) + a \quad \longrightarrow \quad \text{translation by } \begin{pmatrix}0\\a\end{pmatrix}$$ $$y = f(x + a) \quad \longrightarrow \quad \text{translation by } \begin{pmatrix}-a\\0\end{pmatrix}$$ $$y = af(x) \quad \longrightarrow \quad \text{vertical stretch, scale factor } a$$ $$y = f(ax) \quad \longrightarrow \quad \text{horizontal stretch, scale factor } \dfrac{1}{a}$$ $$y = -f(x) \quad \longrightarrow \quad \text{reflection in the } x\text{-axis}$$ $$y = f(-x) \quad \longrightarrow \quad \text{reflection in the } y\text{-axis}$$
Memory Trick Changes inside the function brackets affect the $x$-axis and do the opposite to what you'd expect. $f(x + 3)$ shifts the graph left by 3, not right.
Worked Example 1.6a
The graph of $y = f(x)$ has a maximum at $(2, 5)$. State the coordinates of the maximum after the transformation $y = 3f(x - 1)$.
1
$f(x - 1)$: translation by $+1$ in $x$. Point moves to $(3, 5)$.
2
$3f(x-1)$: vertical stretch by factor 3. $y$-coordinate multiplied by 3: $(3, 15)$.
3
New maximum: $\mathbf{(3, 15)}$
```

Practice Questions

Section A — Foundation
Grade A
```
Q1
Simplify: $\dfrac{(2x^3)^2 \times x^{-4}}{4x}$
$\dfrac{4x^6 \times x^{-4}}{4x} = \dfrac{4x^2}{4x} = x$
[3 marks]
Q2
Rationalise the denominator: $\dfrac{6}{2 - \sqrt{3}}$
Multiply by $\dfrac{2+\sqrt{3}}{2+\sqrt{3}}$: $\quad \dfrac{6(2+\sqrt{3})}{4-3} = 12 + 6\sqrt{3}$
[3 marks]
Q3
Complete the square for $x^2 + 6x - 7$ and hence find the minimum value and the value of $x$ at which it occurs.
$(x+3)^2 - 9 - 7 = (x+3)^2 - 16$. Minimum value is $-16$ at $x = -3$.
[4 marks]
Q4
Solve the simultaneous equations $y = 2x - 1$ and $y = x^2 - 3x + 5$.
$2x - 1 = x^2 - 3x + 5 \implies x^2 - 5x + 6 = 0 \implies (x-2)(x-3) = 0$. So $x = 2, y = 3$ or $x = 3, y = 5$.
[5 marks]
Q5
Find the set of values of $x$ satisfying $x^2 - x - 12 \leq 0$.
$(x-4)(x+3) \leq 0$. Roots at $x = 4$ and $x = -3$. Solution: $-3 \leq x \leq 4$.
[4 marks]
```
Section B — Challenging
Grade B
```
Q6
Show that $\dfrac{(9x)^{\frac{1}{2}} \cdot 27x^2}{3^4 \cdot x^{-\frac{1}{2}}}$ simplifies to $x^k$ for some constant $k$, and find $k$.
Numerator: $3x^{\frac{1}{2}} \cdot 3^3 x^2 = 3^4 x^{\frac{5}{2}}$. Divided by $3^4 \cdot x^{-\frac{1}{2}} = 3^4 x^{-\frac{1}{2}}$. Result: $x^{\frac{5}{2}+\frac{1}{2}} = x^3$. So $k = 3$.
[5 marks]
Q7
Find the values of $k$ for which $kx^2 + 4x + k = 0$ has two distinct real solutions.
Discriminant $> 0$: $16 - 4k^2 > 0 \implies k^2 < 4 \implies -2 < k < 2$. Also $k \neq 0$ (otherwise not quadratic). So $-2 < k < 0$ or $0 < k < 2$.
[5 marks]
Q8
The graph of $y = f(x)$ passes through $(4, 7)$. Write down the coordinates of the corresponding point on the graph of $y = \frac{1}{2}f(2x + 6)$.
$f(2x+6) = f(4)$ when $2x + 6 = 4 \implies x = -1$. The $y$ value is $\frac{1}{2} \times 7 = 3.5$. Point: $(-1, 3.5)$.
[4 marks]
```
Section C — STEP-style
Hard
```
Q9
Prove that for all real $x$, the expression $x^2 + 4x + 7$ is always positive. Hence, or otherwise, prove that $\dfrac{x^2 + 4x + 7}{2x^2 + 3} > 0$ for all real $x$.
Complete the square: $x^2 + 4x + 7 = (x+2)^2 + 3 \geq 3 > 0$ for all real $x$. For the denominator: $2x^2 + 3 = 2(x^2 + \frac{3}{2}) \geq 3 > 0$. Since numerator and denominator are both always positive, their quotient is always positive. $\square$
[6 marks]
Q10
Given that $\sqrt{a} + \sqrt{b} = \sqrt{c}$ where $a$, $b$, $c$ are positive integers, show that at least one of $a$, $b$, $c$ must be divisible by 4, or two of them must be equal.
Squaring: $a + b + 2\sqrt{ab} = c$. Since $c$ is an integer, $2\sqrt{ab}$ must be an integer, so $\sqrt{ab}$ is rational. Since $a, b$ are integers, $ab$ must be a perfect square. Write $a = m^2 s$, $b = n^2 s$ where $s$ is square-free. Then $ab = m^2 n^2 s^2$ is a perfect square always. Squaring the original: $c = a + b + 2mn\cdot s$. This is a valid starting point for the STEP solution — continue by considering cases modulo 4.
[8 marks]
```
