2.1 Straight Lines

```

Every straight line can be described by an equation. There are three main forms you need to be comfortable with.

$$\text{Gradient: } m = \frac{y_2 - y_1}{x_2 - x_1}$$ $$\text{Slope-intercept form: } y = mx + c$$ $$\text{Point-slope form: } y - y_1 = m(x - x_1)$$ $$\text{General form: } ax + by + c = 0$$

The point-slope form is the most useful at A Level — if you know a point on the line and its gradient, you can write the equation immediately.

Distance and Midpoint

$$\text{Distance} = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$ $$\text{Midpoint} = \left(\frac{x_1+x_2}{2},\ \frac{y_1+y_2}{2}\right)$$
Worked Example 2.1a
Find the equation of the line passing through $A(1, 3)$ and $B(4, 9)$. Give your answer in the form $ax + by + c = 0$.
1
Find the gradient: $m = \dfrac{9-3}{4-1} = \dfrac{6}{3} = 2$
2
Use point-slope form with point $A(1,3)$: $$y - 3 = 2(x - 1)$$
3
Rearrange to general form: $$y - 3 = 2x - 2 \implies 2x - y - 1 = 0$$
Worked Example 2.1b
The points $P(2, -1)$ and $Q(8, 5)$ are the endpoints of a diameter of a circle. Find the midpoint of $PQ$ and the length of the diameter.
1
Midpoint: $\left(\dfrac{2+8}{2}, \dfrac{-1+5}{2}\right) = (5, 2)$
2
Length: $\sqrt{(8-2)^2 + (5-(-1))^2} = \sqrt{36 + 36} = \sqrt{72} = 6\sqrt{2}$
```

2.2 Parallel & Perpendicular Lines

```
$$\text{Parallel lines: same gradient} \implies m_1 = m_2$$ $$\text{Perpendicular lines: } m_1 \times m_2 = -1 \implies m_2 = -\frac{1}{m_1}$$
Memory Trick To get the perpendicular gradient, flip the fraction and change the sign. So if $m = \frac{2}{3}$, the perpendicular gradient is $-\frac{3}{2}$.
Worked Example 2.2a
Find the equation of the line perpendicular to $y = 3x - 2$ that passes through $(6, 1)$.
1
Gradient of given line: $m_1 = 3$
2
Perpendicular gradient: $m_2 = -\dfrac{1}{3}$
3
Point-slope with $(6, 1)$: $$y - 1 = -\frac{1}{3}(x - 6) \implies y = -\frac{1}{3}x + 3$$

The Perpendicular Bisector

The perpendicular bisector of a line segment passes through its midpoint and is perpendicular to it. This comes up constantly in circle questions.

Worked Example 2.2b
Find the perpendicular bisector of the line segment joining $A(2, 5)$ and $B(8, 3)$.
1
Midpoint of $AB$: $\left(\dfrac{2+8}{2}, \dfrac{5+3}{2}\right) = (5, 4)$
2
Gradient of $AB$: $m = \dfrac{3-5}{8-2} = \dfrac{-2}{6} = -\dfrac{1}{3}$
3
Perpendicular gradient: $m_\perp = 3$
4
Equation through $(5, 4)$: $y - 4 = 3(x - 5) \implies y = 3x - 11$
```

2.3 Equation of a Circle

```

A circle is the set of all points equidistant from a fixed centre. That distance is the radius $r$.

$$\text{Centre }(a, b)\text{, radius }r\text{:}$$ $$(x - a)^2 + (y - b)^2 = r^2$$ $$\text{Expanded (general) form:}$$ $$x^2 + y^2 + 2fx + 2gy + c = 0$$ $$\text{where centre} = (-f, -g),\quad r = \sqrt{f^2 + g^2 - c}$$
⚠ Common Mistake In $(x-a)^2 + (y-b)^2 = r^2$, the right-hand side is $r^2$, not $r$. Students often write $r$ instead of $r^2$ when reading off the radius from the equation. Always square-root the RHS to find $r$.
Worked Example 2.3a
Find the centre and radius of the circle $x^2 + y^2 - 6x + 4y - 3 = 0$.
1
Complete the square in $x$: $x^2 - 6x = (x-3)^2 - 9$
2
Complete the square in $y$: $y^2 + 4y = (y+2)^2 - 4$
3
Substitute back: $$(x-3)^2 - 9 + (y+2)^2 - 4 - 3 = 0$$ $$(x-3)^2 + (y+2)^2 = 16$$
4
Centre: $(3, -2)$, Radius: $\sqrt{16} = 4$
```

2.4 Circle Properties & Theorems

```

There are three key circle theorems from GCSE that are used constantly at A Level:

Theorem 1 — Tangent-Radius The tangent to a circle at any point is perpendicular to the radius at that point. This means if you know the point of tangency, you can immediately find the gradient of the tangent.
Theorem 2 — Angle in Semicircle The angle subtended by a diameter at any point on the circle is always 90°. So if $AB$ is a diameter and $P$ is on the circle, then $\angle APB = 90°$.
Theorem 3 — Perpendicular from Centre The perpendicular from the centre of a circle to a chord bisects the chord.
Worked Example 2.4a
The circle $C$ has equation $(x-3)^2 + (y+1)^2 = 25$. Find the equation of the tangent to $C$ at the point $P(7, 2)$.
1
Centre of circle: $(3, -1)$. Verify $P(7,2)$ is on the circle: $(7-3)^2 + (2+1)^2 = 16 + 9 = 25$ ✓
2
Gradient of radius from centre to $P$: $$m_r = \frac{2-(-1)}{7-3} = \frac{3}{4}$$
3
Tangent is perpendicular to radius, so gradient of tangent: $m_t = -\dfrac{4}{3}$
4
Equation of tangent through $P(7, 2)$: $$y - 2 = -\frac{4}{3}(x - 7) \implies 3y - 6 = -4x + 28 \implies 4x + 3y - 34 = 0$$
```

2.5 Intersections of Lines and Circles

```

To find where a line meets a circle, substitute the equation of the line into the equation of the circle. This gives a quadratic — the discriminant then tells you how many intersections there are.

$$\Delta > 0 \implies \text{line intersects circle at two points}$$ $$\Delta = 0 \implies \text{line is a tangent (one point)}$$ $$\Delta < 0 \implies \text{line does not meet the circle}$$
Worked Example 2.5a
Show that the line $y = x + 3$ is a tangent to the circle $x^2 + y^2 = \dfrac{9}{2}$, and find the point of tangency.
1
Substitute $y = x + 3$ into $x^2 + y^2 = \dfrac{9}{2}$: $$x^2 + (x+3)^2 = \frac{9}{2}$$
2
Expand: $x^2 + x^2 + 6x + 9 = \dfrac{9}{2} \implies 2x^2 + 6x + 9 - \dfrac{9}{2} = 0 \implies 2x^2 + 6x + \dfrac{9}{2} = 0$
3
Multiply by 2: $4x^2 + 12x + 9 = 0$
4
Discriminant: $\Delta = 144 - 4(4)(9) = 144 - 144 = 0$
$\Delta = 0$ confirms the line is a tangent.
5
Solve $4x^2 + 12x + 9 = (2x+3)^2 = 0 \implies x = -\dfrac{3}{2}$, $y = -\dfrac{3}{2} + 3 = \dfrac{3}{2}$
Point of tangency: $\left(-\dfrac{3}{2}, \dfrac{3}{2}\right)$
Exam Technique In "show that the line is a tangent" questions, always calculate the discriminant explicitly and state that $\Delta = 0$. Don't just factorise and say "repeated root" — examiners want to see the discriminant.
```

Practice Questions

Section A — Foundation
Grade A
```
Q1
Find the equation of the straight line through $(3, -2)$ and $(7, 6)$. Give your answer in the form $y = mx + c$.
$m = \dfrac{6-(-2)}{7-3} = 2$. Using $(3,-2)$: $y + 2 = 2(x-3) \implies y = 2x - 8$.
[3 marks]
Q2
Find the equation of the perpendicular bisector of $AB$ where $A = (-1, 4)$ and $B = (5, 0)$.
Midpoint: $(2, 2)$. Gradient of $AB$: $\dfrac{0-4}{5-(-1)} = -\dfrac{2}{3}$. Perpendicular gradient: $\dfrac{3}{2}$. Equation: $y - 2 = \dfrac{3}{2}(x-2) \implies y = \dfrac{3}{2}x - 1$.
[4 marks]
Q3
A circle has equation $x^2 + y^2 - 8x + 2y + 8 = 0$. Find the centre and radius.
$(x-4)^2 - 16 + (y+1)^2 - 1 + 8 = 0 \implies (x-4)^2 + (y+1)^2 = 9$. Centre: $(4, -1)$, Radius: $3$.
[4 marks]
Q4
Find the equation of the tangent to the circle $(x+1)^2 + (y-3)^2 = 20$ at the point $(3, 7)$.
Centre $(-1, 3)$. Gradient of radius to $(3,7)$: $\dfrac{7-3}{3-(-1)} = 1$. Tangent gradient: $-1$. Equation: $y - 7 = -(x-3) \implies y = -x + 10$.
[4 marks]
```
Section B — Challenging
Grade B
```
Q5
The points $A(0, 2)$, $B(6, 0)$ and $C(8, 4)$ lie on a circle. Find the equation of the circle by finding the perpendicular bisectors of $AB$ and $BC$ and solving simultaneously.
Perp bisector of $AB$: midpoint $(3,1)$, gradient of $AB = -\frac{1}{3}$, perp gradient $= 3$, line: $y = 3x - 8$. Perp bisector of $BC$: midpoint $(7,2)$, gradient of $BC = 2$, perp gradient $= -\frac{1}{2}$, line: $y = -\frac{1}{2}x + \frac{11}{2}$. Solve: $3x - 8 = -\frac{1}{2}x + \frac{11}{2} \implies x = 3, y = 1$. Centre $(3,1)$, $r = \sqrt{9+1} = \sqrt{10}$. Circle: $(x-3)^2 + (y-1)^2 = 10$.
[7 marks]
Q6
Find the values of $k$ for which the line $y = kx + 1$ is a tangent to the circle $x^2 + y^2 = 4$.
Substitute: $x^2 + (kx+1)^2 = 4 \implies (1+k^2)x^2 + 2kx - 3 = 0$. For tangent, $\Delta = 0$: $4k^2 + 12(1+k^2) = 0$? No — $\Delta = 4k^2 - 4(1+k^2)(-3) = 4k^2 + 12 + 12k^2 = 16k^2 + 12$. Set $= 0$: no real solutions... actually set $\Delta = 0$: $4k^2 + 12(1+k^2) = 0$ gives $16k^2 = -12$. So checking: $\Delta = (2k)^2 - 4(1+k^2)(1-4) = 4k^2 + 12(1+k^2) = 16k^2 + 12 = 0 \implies k^2 = -\frac{3}{4}$. No real solutions — revisit. Correct working: $x^2 + k^2x^2 + 2kx + 1 = 4 \implies (1+k^2)x^2 + 2kx - 3 = 0$. $\Delta = 4k^2 + 12(1+k^2) = 4k^2 + 12 + 12k^2 = 16k^2 + 12$. This is always positive, so the line always intersects the circle for all real $k$. The line $y = kx + 1$ always passes through $(0,1)$ which is inside the circle $x^2+y^2=4$ (since $0+1 < 4$), so it always cuts the circle in two points for any gradient.
Note: This is an interesting result! A line through an interior point always intersects the circle twice.
[5 marks]
```
Section C — STEP-style
Hard
```
Q7
Two circles $C_1: x^2 + y^2 = r^2$ and $C_2: (x-d)^2 + y^2 = r^2$ have the same radius $r$ and centres $O$ and $A$ respectively, where $d > 0$. They intersect at points $P$ and $Q$. Prove that $OA$ is the perpendicular bisector of $PQ$, and find the length $PQ$ in terms of $r$ and $d$, stating any conditions on $d$ for the circles to intersect.
Subtract equations: $x^2 - (x-d)^2 = 0 \implies 2dx - d^2 = 0 \implies x = \dfrac{d}{2}$. So both intersection points have $x = \dfrac{d}{2}$, the midpoint of $OA$. Since $OA$ lies along the $x$-axis and $PQ$ is vertical ($x = \frac{d}{2}$), they are perpendicular — proving $OA$ perp-bisects $PQ$. For $PQ$ length: at $x = \frac{d}{2}$, $y^2 = r^2 - \frac{d^2}{4}$, so $y = \pm\sqrt{r^2 - \frac{d^2}{4}}$. Length $PQ = 2\sqrt{r^2 - \frac{d^2}{4}} = \sqrt{4r^2 - d^2}$. Condition for intersection: $r^2 > \frac{d^2}{4} \implies d < 2r$.
[8 marks]
```
